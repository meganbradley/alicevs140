# FOPEN_MAX, _SYS_OPEN
## Syntax  
  
```  
  
#include <stdio.h>  
  
```  
  
## Remarks  
 This is the maximum number of files that can be opened simultaneously. `FOPEN_MAX` is the ANSI-compatible name. `_SYS_OPEN` is provided for compatibility with existing code.
