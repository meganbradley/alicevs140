# CAtlServiceModuleT::IsInstalled
Confirms that the service has been installed.  
  
## Syntax  
  
```  
  
BOOL IsInstalled( ) throw( );  
  
```  
  
## Return Value  
 Returns TRUE if the service is installed, FALSE otherwise.  
  
## Requirements  
 **Header:** atlbase.h
