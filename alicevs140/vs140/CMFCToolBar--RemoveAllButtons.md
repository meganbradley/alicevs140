# CMFCToolBar::RemoveAllButtons
Removes all buttons and separators from the toolbar.  
  
## Syntax  
  
```  
virtual void RemoveAllButtons();  
```  
  
## Remarks  
 The framework calls this method when it recreates or destroys a toolbar.  
  
## Requirements  
 **Header:** afxtoolbar.h
