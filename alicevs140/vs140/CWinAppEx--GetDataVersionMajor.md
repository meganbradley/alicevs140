# CWinAppEx::GetDataVersionMajor
Returns the major version of the application that is saved in the Windows registry when you call [CWinAppEx::SaveState](../vs140/CWinAppEx--SaveState.md).  
  
## Syntax  
  
```  
int GetDataVersionMajor() const;  
```  
  
## Return Value  
 An integer value that contains the major version number.  
  
## Requirements  
 **Header:** afxwinappex.h
