# CMFCCaptionBar::SetFlatBorder
Sets the border style of the caption bar.  
  
## Syntax  
  
```  
void SetFlatBorder(  
   BOOL bFlat=TRUE   
);  
```  
  
#### Parameters  
 [in] `bFlat`  
 `TRUE` if the border of a caption bar is flat. `FALSE` if the border is 3D.  
  
## Requirements  
 **Header:** afxcaptionbar.h
