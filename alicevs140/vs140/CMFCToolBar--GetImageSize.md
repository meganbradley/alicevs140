# CMFCToolBar::GetImageSize
Returns the current size of toolbar button images.  
  
## Syntax  
  
```  
CSize GetImageSize() const;  
```  
  
## Return Value  
 A [CSize](../vs140/CSize-Class.md) object that represents the current size of toolbar button images.  
  
## Requirements  
 **Header:** afxtoolbar.h
