# CMFCRibbonSeparator::IsTabStop
Indicates whether this is a tab stop.  
  
## Syntax  
  
```  
virtual BOOL IsTabStop() const;  
```  
  
## Return Value  
 Always `FALSE` for this class.  
  
## Remarks  
 A ribbon separator is not a tab stop.  
  
## Requirements  
 **Header:** afxbaseribbonelement.h
