# CD2DGeometrySink::AddArc
Adds a single arc to the path geometry  
  
## Syntax  
  
```  
void AddArc(  
   const D2D1_ARC_SEGMENT& arc  
);  
```  
  
#### Parameters  
 `arc`  
 The arc segment to add to the figure  
  
## Requirements  
 **Header:** afxrendertarget.h
