# CMFCBaseTabCtrl::GetActiveTab
Retrieves the index of the currently active tab.  
  
## Syntax  
  
```  
virtual int GetActiveTab() const;  
```  
  
## Return Value  
 The zero-based index of the active tab; -1 if there is no active tab.  
  
## Requirements  
 **Header:** afxbasetabctrl.h
