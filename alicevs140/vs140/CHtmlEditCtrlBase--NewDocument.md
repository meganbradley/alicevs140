# CHtmlEditCtrlBase::NewDocument
Creates a new document.  
  
## Syntax  
  
```  
  
HRESULT NewDocument( ) const;  
  
```  
  
## Return Value  
 Returns S_OK on success, or an error HRESULT on failure.

 **Header:** afxhtml.h
