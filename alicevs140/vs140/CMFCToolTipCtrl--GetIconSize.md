# CMFCToolTipCtrl::GetIconSize
Returns the size of an icon in a tooltip.  
  
## Syntax  
  
```  
virtual CSize GetIconSize();  
```  
  
## Return Value  
 The size of the icon, in pixels.  
  
## Requirements  
 **Header:** afxToolTipCtrl.h
