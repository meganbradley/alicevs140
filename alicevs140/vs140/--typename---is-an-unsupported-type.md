# &#39;&lt;typename&gt;&#39; is an unsupported type
The type specified cannot be represented by [!INCLUDE[vsprvs](../vs140/includes/vsprvs_md.md)].  
  
 **Error ID:** BC30649  
  
### To correct this error  
  
-   Use a supported type.  
  
-   Develop a new type.
