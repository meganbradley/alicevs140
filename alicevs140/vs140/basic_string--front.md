# basic_string::front
Returns a reference to the first element in a string.  
  
## Syntax  
  
```  
const_reference front() const;  
reference front();  
```  
  
## Return Value  
 A reference to the first element of the string, which must be non-empty.  
  
## Remarks  
  
## Requirements  
 **Header:** <string\>  
  
 **Namespace:** std
