# CMFCRibbonButtonsGroup::AddButton
Adds a button to a group.  
  
## Syntax  
  
```  
void AddButton(  
   CMFCRibbonBaseElement* pButton   
);  
```  
  
#### Parameters  
 [in] `pButton`  
 A pointer to a button to add.  
  
## Requirements  
 **Header:** afxribbonbuttonsgroup.h
