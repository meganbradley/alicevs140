# .BREAK
Generates code to terminate a [.WHILE](../vs140/.WHILE.md) or [.REPEAT](../vs140/.REPEAT.md) block if *condition* is true.  
  
## Syntax  
  
```  
  
.BREAK [[.IF condition]]   
```
