# forward_list::end
Returns an iterator that addresses the location succeeding the last element in a forward list.  
  
## Syntax  
  
```  
const_iterator end() const;  
iterator end();  
```  
  
## Return Value  
 A forward iterator that points just beyond the end of the sequence.  
  
## Requirements  
 **Header:** <forward_list>  
  
 **Namespace:** std
