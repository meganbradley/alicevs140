# time_put::iter_type
A type that describes an output iterator.  
  
## Syntax  
  
```  
  
typedef OutputIterator iter  
_  
type;  
  
```  
  
## Remarks  
 The type is a synonym for the template parameter **OutputIterator**.

 **Header:** <locale\>  
  
 **Namespace:** std
