# Core.int&lt;&#39;Measure&gt; Type (F#)
The type of 32-bit signed integer numbers, annotated with a unit of measure. The unit of measure is erased in compiled code and when values of this type are analyzed using reflection. The compiled form of this type is equivalent to <xref:System.Int32?qualifyHint=False>.  
  
 **Namespace/Module Path:** Microsoft.FSharp.Core  
  
 **Assembly:** FSharp.Core (in FSharp.Core.dll)  
  
## Syntax  
  
```  
[<MeasureAnnotatedAbbreviationAttribute>]  
type int<'Measure> = int  
```
