# CMFCColorMenuButton::SetColorByCmdID
Sets the color of the specified color menu button.  
  
## Syntax  
  
```  
static void SetColorByCmdID(  
   UINT uiCmdID,  
   COLORREF color  
);  
```  
  
#### Parameters  
 [in] `uiCmdID`  
 The resource ID of a color menu button.  
  
 [in] `color`  
 An RGB color value.  
  
## Requirements  
 **Header:** afxcolormenubutton.h
