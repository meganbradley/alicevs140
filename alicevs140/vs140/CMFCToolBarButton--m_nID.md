# CMFCToolBarButton::m_nID
The command ID of the button.  
  
## Syntax  
  
```  
UINT m_nID;  
```  
  
## Remarks  
 A command ID of -1 indicates that the button is a separator. All button separators have the `TBBS_SEPARATOR` style. See [CMFCToolBarButton::m_nStyle](../vs140/CMFCToolBarButton--m_nStyle.md) for more information about button styles.  
  
## Requirements  
 **Header:** afxtoolbarbutton.h
