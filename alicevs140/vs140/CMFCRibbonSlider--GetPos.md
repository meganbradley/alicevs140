# CMFCRibbonSlider::GetPos
Returns the current position of the slider control.  
  
## Syntax  
  
```  
int GetPos() const;  
```  
  
## Return Value  
 The current position of the slider control, which is a position relative to the beginning of the slider.  
  
## Requirements  
 **Header:** afxribbonslider.h
