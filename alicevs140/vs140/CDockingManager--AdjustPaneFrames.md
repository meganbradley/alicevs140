# CDockingManager::AdjustPaneFrames
Causes the `WM_NCCALCSIZE` message to be sent to all panes and `CPaneFrameWnd` windows.  
  
## Syntax  
  
```  
virtual void AdjustPaneFrames();  
```

## Requirements  
 **Header:** afxDockingManager.h
