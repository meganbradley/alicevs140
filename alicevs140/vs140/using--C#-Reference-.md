# using (C# Reference)
The `using` keyword has two major uses:  
  
-   As a directive, when it is used to create an alias for a namespace or to import types defined in other namespaces. See [using Directive](../Topic/using%20Directive%20\(C%23%20Reference\).md).  
  
-   As a statement, when it defines a scope at the end of which an object will be disposed. See [using Statement](../Topic/using%20Statement%20\(C%23%20Reference\).md).
