# LanguagePrimitives.Int16WithMeasure&lt;&#39;Measure&gt; Function

## Syntax  
  
```  
// Signature:  
Int16WithMeasure : int16 -> int16<'u>  
  
// Usage:  
Int16WithMeasure value  
```  
  
#### Parameters  
 `value`  
 Type: [int16](../vs140/Core.int16-Type-Abbreviation--F#-.md)

## Return Value  
 The 16-bit integer with the specified units-of-measure.
