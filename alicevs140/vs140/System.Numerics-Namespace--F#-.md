This topic describes the F# extensions to the System.Numerics namespace. For information about the .NET Framework System.Numerics namespace, see <xref:System.Numerics?qualifyHint=False>.

## Remarks  
 The APIs documented in this topic are provided for use only with the version of the F# Core Library Versions that targets .NET Framework 2.0. If you are using .NET Framework 4, use the .NET Framework 4 version of these APIs, which are documented in <xref:System.Numerics?qualifyHint=False>.
