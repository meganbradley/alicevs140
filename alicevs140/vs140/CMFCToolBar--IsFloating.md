# CMFCToolBar::IsFloating
Determines whether the toolbar is floating.  
  
## Syntax  
  
```  
virtual BOOL IsFloating() const;  
```  
  
## Return Value  
 `TRUE` if the toolbar is floating; otherwise, `FALSE`.  
  
## Requirements  
 **Header:** afxtoolbar.h
