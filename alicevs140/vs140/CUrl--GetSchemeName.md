# CUrl::GetSchemeName
Call this method to get the URL scheme name.  
  
## Syntax  
  
```  
  
inline LPCTSTR GetSchemeName( ) const throw( );  
  
```  
  
## Return Value  
 Returns the URL scheme name (such as "http" or "ftp").  
  
## Requirements  
 **Header:** atlutil.h
