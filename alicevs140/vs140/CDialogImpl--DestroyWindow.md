# CDialogImpl::DestroyWindow
Destroys a modeless dialog box.  
  
## Syntax  
  
```  
  
BOOL DestroyWindow( );  
  
```  
  
## Return Value  
 **TRUE** if the dialog box was successfully destroyed; otherwise **FALSE**.  
  
## Remarks  
 Returns **TRUE** if the dialog box was successfully destroyed; otherwise **FALSE**.  
  
## Requirements  
 **Header:** atlwin.h
