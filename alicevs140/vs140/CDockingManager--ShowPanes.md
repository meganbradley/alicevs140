# CDockingManager::ShowPanes
Shows or hides the panes of the control and autohide bars.  
  
## Syntax  
  
```  
virtual BOOL ShowPanes(  
   BOOL bShow  
);  
```  
  
#### Parameters  
 [in] `bShow`  
 `TRUE` to show the panes; `FALSE to` hide the panes.  
  
## Return Value  
 Always `FALSE`.  
  
## Requirements  
 **Header:** afxdockingmanager.h
