# CPathT::XCHAR
A character type.  
  
## Syntax  
  
```  
  
typedef StringType::XCHAR XCHAR;  
  
```  
  
## Remarks  
 `StringType` is the template parameter to `CPathT`.  
  
## Requirements  
 **Header:** atlpath.h
