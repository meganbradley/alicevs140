# CDocument::SetChunkValue
Sets a chunk value.  
  
## Syntax  
  
```  
virtual BOOL SetChunkValue (  
   IFilterChunkValue* pValue  
);  
```  
  
#### Parameters  
 `pValue`  
 Specifies a chunk value to set.  
  
## Return Value  
 Nonzero if successful; otherwise 0.
