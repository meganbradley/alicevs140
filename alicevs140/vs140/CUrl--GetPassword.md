# CUrl::GetPassword
Call this method to get the password from the URL.  
  
## Syntax  
  
```  
  
inline LPCTSTR GetPassword( ) const throw( );  
  
```  
  
## Return Value  
 Returns the password.  
  
## Requirements  
 **Header:** atlutil.h
