# CComFakeCriticalSection::Unlock
Does nothing since there is no critical section.  
  
## Syntax  
  
```  
  
HRESULT Unlock( ) throw( );  
  
```  
  
## Return Value  
 Returns S_OK.  
  
## Requirements  
 **Header:** atlcore.h
