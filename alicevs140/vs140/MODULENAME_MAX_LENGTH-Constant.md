# MODULENAME_MAX_LENGTH Constant
Stores the maximum length of the module name. This value must be the same on the compiler and runtime.  
  
## Syntax  
  
```  
static const UINT MODULENAME_MAX_LENGTH = 1024;  
```  
  
## Requirements  
 **Header:** amprt.h  
  
 **Namespace:** Concurrency
