# CD2DTextFormat::Get
Returns IDWriteTextFormat interface  
  
## Syntax  
  
```  
IDWriteTextFormat* Get();  
```  
  
## Return Value  
 Pointer to an IDWriteTextFormat interface or NULL if object is not initialized yet.

 **Header:** afxrendertarget.h
