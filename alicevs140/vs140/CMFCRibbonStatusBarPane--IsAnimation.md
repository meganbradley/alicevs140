# CMFCRibbonStatusBarPane::IsAnimation
Determines whether the animation is in progress.  
  
## Syntax  
  
```  
BOOL IsAnimation() const;  
```  
  
## Return Value  
 `TRUE` if animation is in progress; `FALSE` otherwise.  
  
## Requirements  
 **Header:** afxribbonstatusbarpane.h
