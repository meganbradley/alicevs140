# CMenu::m_hMenu
Specifies the `HMENU` handle of the Windows menu attached to the `CMenu` object.  
  
## Syntax  
  
```  
  
HMENU m_hMenu;  
  
```  
  
## Example  
 See the example for [CMenu::LoadMenu](../vs140/CMenu--LoadMenu.md).  
  
## Requirements  
 **Header:** afxwin.h
