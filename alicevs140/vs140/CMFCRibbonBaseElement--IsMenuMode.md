# CMFCRibbonBaseElement::IsMenuMode
Indicates whether the ribbon element is contained in a menu.  
  
## Syntax  
  
```  
BOOL IsMenuMode() const;  
```  
  
## Return Value  
 `TRUE` if the ribbon element is contained in a menu; otherwise, `FALSE`.
