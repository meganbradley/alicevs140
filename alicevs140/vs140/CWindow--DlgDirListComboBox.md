# CWindow::DlgDirListComboBox
Fills a combo box with the names of all files matching a specified path or file name.  
  
## Syntax  
  
```  
  
      int DlgDirListComboBox(  
   LPTSTR lpPathSpec,  
   int nIDComboBox,  
   int nIDStaticPath,  
   UINT nFileType   
) throw();  
```
