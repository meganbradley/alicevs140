# tanpif Function
Calculates the tangent value of pi * _X  
  
## Syntax  
  
```  
inline float tanpif(  
   float _X  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
## Return Value  
 Returns the tangent value of pi * _X  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::precise_math
