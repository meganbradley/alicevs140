# CAtlMap::RemoveAll
Call this method to remove all elements from the `CAtlMap` object.  
  
## Syntax  
  
```  
  
void RemoveAll( ) throw( );  
  
```  
  
## Remarks  
 Clears out the `CAtlMap` object, freeing the memory used to store the elements.  
  
## Requirements  
 **Header:** atlcoll.h
