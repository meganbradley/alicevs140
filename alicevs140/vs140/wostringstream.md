# wostringstream
Creates a type `basic_ostringstream` specialized on a `wchar_t` template parameter.  
  
## Syntax  
  
```  
  
typedef basic  
_  
ostringstream<wchar  
_  
t> wostringstream;  
  
```  
  
## Remarks  
 The type is a synonym for template class [basic_ostringstream](../vs140/basic_ostringstream-Class.md), specialized for elements of type `wchar_t`.  
  
## Requirements  
 **Header:** <sstream\>  
  
 **Namespace:** std
