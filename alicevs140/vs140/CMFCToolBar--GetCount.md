# CMFCToolBar::GetCount
Returns the number of buttons and separators on the toolbar.  
  
## Syntax  
  
```  
int GetCount() const;  
```  
  
## Return Value  
 The number of buttons and separators on the toolbar.  
  
## Requirements  
 **Header:** afxtoolbar.h
