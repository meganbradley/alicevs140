# Compiler Error CS0584
Internal Compiler Error: stage 'stage' symbol 'symbol'  
  
 Try to determine whether the compiler is failing because of its inability to parse unexpected syntax. If that is not the case, try [Getting Help from Microsoft Product Support Services (Visual Studio)](../vs140/Talk-to-Us.md).
