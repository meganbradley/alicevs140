# CAsyncSocket::m_hSocket
Contains the **SOCKET** handle for the socket encapsulated by this `CAsyncSocket` object.  
  
## Syntax  
  
```  
  
SOCKET m_hSocket;  
  
```  
  
## Requirements  
 **Header:** afxsock.h
