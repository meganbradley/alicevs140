# concurrent_unordered_multiset::clear Method
Erases all the elements in the concurrent container. This function is not concurrency safe.  
  
## Syntax  
  
```  
void clear();  
```  
  
## Requirements  
 **Header:** internal_concurrent_hash.h  
  
 **Namespace:** concurrency
