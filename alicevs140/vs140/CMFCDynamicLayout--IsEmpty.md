# CMFCDynamicLayout::IsEmpty
Checks if a dynamic layout has no child windows added.  
  
## Syntax  
  
```  
  
BOOL IsEmpty();  
```  
  
## Return Value  
 TRUE if layout has no items; otherwise FALSE.  
  
## Remarks  
  
## Requirements  
 **Header:** afxlayout.h
