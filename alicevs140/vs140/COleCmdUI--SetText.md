# COleCmdUI::SetText
Call this function to return a text name or status string for a command.  
  
## Syntax  
  
```  
  
      virtual void SetText(  
   LPCTSTR lpszText   
);  
```  
  
#### Parameters  
 `lpszText`  
 A pointer to the text to be used with the command.  
  
## Requirements  
 **Header:** afxdocob.h
