# CComCriticalSection::m_sec
Contains a critical section object that is used by all `CComCriticalSection` methods.  
  
## Syntax  
  
```  
  
CRITICAL_SECTION m_sec;  
  
```  
  
## Requirements  
 **Header:** atlcore.h
