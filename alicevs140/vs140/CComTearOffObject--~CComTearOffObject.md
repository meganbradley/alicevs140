# CComTearOffObject::~CComTearOffObject
The destructor.  
  
## Syntax  
  
```  
  
~CComTearOffObject( );  
  
```  
  
## Remarks  
 Frees all allocated resources, calls FinalRelease, and decrements the module lock count.  
  
## Requirements  
 **Header:** atlcom.h
