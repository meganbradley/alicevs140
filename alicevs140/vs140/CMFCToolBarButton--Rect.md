# CMFCToolBarButton::Rect
Retrieves the bounding rectangle of the button.  
  
## Syntax  
  
```  
const CRect& Rect() const;  
```  
  
## Return Value  
 A `CRect` object that contains the bounding rectangle of a button.  
  
## Requirements  
 **Header:** afxtoolbarbutton.h
