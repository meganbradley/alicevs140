# CWindow::SetScrollPos
Changes the position of the scroll box.  
  
## Syntax  
  
```  
  
      int SetScrollPos(  
   int nBar,  
   int nPos,  
   BOOL bRedraw = TRUE   
) throw();  
```  
  
## Remarks  
 See [SetScrollPos](http://msdn.microsoft.com/library/windows/desktop/bb787597) in the [!INCLUDE[winSDK](../vs140/includes/winSDK_md.md)].
