# END_PROPERTY_SET
Marks the end of a property set.  
  
## Syntax  
  
```  
  
END_PROPERTY_SET(  
guid   
)  
  
```  
  
#### Parameters  
 `guid`  
 [in] The property GUID.

 See [BEGIN_PROPSET_MAP](../vs140/BEGIN_PROPSET_MAP.md).
