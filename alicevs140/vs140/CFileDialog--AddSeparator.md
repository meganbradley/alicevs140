# CFileDialog::AddSeparator
Adds a separator to the dialog.  
  
## Syntax  
  
```  
HRESULT AddSeparator(  
   DWORD dwIDCtl  
);  
```  
  
#### Parameters  
 `dwIDCtl`  
 The ID of the separator add.
