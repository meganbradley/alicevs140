# wistringstream
Creates a type `basic_istringstream` specialized on a `wchar_t` template parameter.  
  
## Syntax  
  
```  
  
typedef basic  
_  
istringstream<wchar  
_  
t> wistringstream;  
  
```  
  
## Remarks  
 The type is a synonym for template class [basic_istringstream](../vs140/basic_istringstream-Class.md), specialized for elements of type `wchar_t`.  
  
## Requirements  
 **Header:** <sstream\>  
  
 **Namespace:** std
