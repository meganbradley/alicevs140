# CSacl::~CSacl
The destructor.  
  
## Syntax  
  
```  
  
~CSacl( ) throw( );  
  
```  
  
## Remarks  
 The destructor frees any resources acquired by the object, including all access-control entries (ACEs).  
  
## Requirements  
 **Header:** atlsecurity.h
