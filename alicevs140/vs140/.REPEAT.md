# .REPEAT
Generates code that repeats execution of the block of *statements* until `condition` becomes true. [.UNTILCXZ](../vs140/.UNTILCXZ.md), which becomes true when CX is zero, may be substituted for [.UNTIL](../vs140/.UNTIL.md). The `condition` is optional with **.UNTILCXZ**.  
  
## Syntax  
  
```  
  
   .REPEAT  
statements  
.UNTIL condition  
```
