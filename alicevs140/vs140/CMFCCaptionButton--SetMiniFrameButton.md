# CMFCCaptionButton::SetMiniFrameButton
Sets the mini size of the title bar.  
  
## Syntax  
  
```  
void SetMiniFramebutton(  
   BOOL bSet = TRUE  
);  
```  
  
#### Parameters  
 [in] `bSet`  
 `TRUE` for mini title bar height; `FALSE` for default title bar height.  
  
## Requirements  
 **Header:** afxcaptionbutton.h
