# CMFCRibbonBar::GetCategoryCount
Retrieves the number of ribbon categories in the ribbon bar.  
  
## Syntax  
  
```  
int GetCategoryCount() const;  
```  
  
## Return Value  
 The number of the ribbon categories in the ribbon bar.  
  
## Requirements  
 **Header:** afxribbonbar.h
