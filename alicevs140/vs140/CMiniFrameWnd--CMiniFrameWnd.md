# CMiniFrameWnd::CMiniFrameWnd
Constructs a `CMiniFrameWnd` object, but does not create the window.  
  
## Syntax  
  
```  
  
CMiniFrameWnd( );  
```  
  
## Remarks  
 To create the window, call [CMiniFrameWnd::Create](../vs140/CMiniFrameWnd--Create.md).  
  
## Requirements  
 **Header:** afxwin.h
