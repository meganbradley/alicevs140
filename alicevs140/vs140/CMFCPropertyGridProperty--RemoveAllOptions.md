# CMFCPropertyGridProperty::RemoveAllOptions
Removes all options (items) from a property.  
  
## Syntax  
  
```  
void RemoveAllOptions();  
```  
  
## Remarks  
 Options are also known as the list items of a property list control.  
  
## Requirements  
 **Header:** afxpropertygridctrl.h
