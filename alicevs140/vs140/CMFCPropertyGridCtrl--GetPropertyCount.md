# CMFCPropertyGridCtrl::GetPropertyCount
Retrieves the number of properties in a property grid control.  
  
## Syntax  
  
```  
int GetPropertyCount() const;  
```  
  
## Return Value  
 The number of properties.

## Requirements  
 **Header:** afxpropertygridctrl.h
