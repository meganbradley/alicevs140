# CToolBarCtrl::GetMaxTextRows
Retrieves the maximum number of text rows displayed on a toolbar button.  
  
## Syntax  
  
```  
  
int GetMaxTextRows( ) const;  
  
```  
  
## Return Value  
 The maximum number of text rows displayed on a toolbar button.  
  
## Requirements  
 **Header:** afxcmn.h
