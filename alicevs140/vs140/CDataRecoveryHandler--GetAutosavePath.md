# CDataRecoveryHandler::GetAutosavePath
Returns the path of the autosaved files.  
  
## Syntax  
  
```  
virtual CString GetAutosavePath() const;  
```  
  
## Return Value  
 The location where the autosaved documents are stored.  
  
## Requirements  
 **Header:** afxdatarecovery.h
