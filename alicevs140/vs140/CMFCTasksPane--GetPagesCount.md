# CMFCTasksPane::GetPagesCount
Returns the number of pages.  
  
## Syntax  
  
```  
int GetPagesCount() const;  
```  
  
## Return Value  
 The number of pages in the task pane.  
  
## Requirements  
 **Header:** afxTasksPane.h
