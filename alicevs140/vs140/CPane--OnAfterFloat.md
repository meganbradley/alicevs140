# CPane::OnAfterFloat
Called by the framework after a pane floats.  
  
## Syntax  
  
```  
virtual void OnAfterFloat();  
```  
  
## Remarks  
 You can override this method in a derived class if you want to perform any processing after a pane floats.  
  
## Requirements  
 **Header:** afxPane.h
