# END_DHTML_EVENT_MAP
Marks the end of the DHTML event map.  
  
## Syntax  
  
```  
  
END_DHTML_EVENT_MAP( )  
  
```  
  
## Remarks  
 Must be used in conjunction with [BEGIN_DHTML_EVENT_MAP](../vs140/BEGIN_DHTML_EVENT_MAP.md).  
  
## Requirements  
 **Header:** afxdhtml.h
