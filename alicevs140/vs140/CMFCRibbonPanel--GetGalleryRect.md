# CMFCRibbonPanel::GetGalleryRect
Returns a bounding rectangle of a Gallery element.  
  
## Syntax  
  
```  
CRect GetGalleryRect();  
```  
  
## Return Value  
 Size and position of the Gallery element within this panel.
