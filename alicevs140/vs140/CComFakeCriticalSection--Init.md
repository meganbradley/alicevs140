# CComFakeCriticalSection::Init
Does nothing since there is no critical section.  
  
## Syntax  
  
```  
  
HRESULT Init( ) throw( );  
  
```  
  
## Return Value  
 Returns S_OK.  
  
## Requirements  
 **Header:** atlcore.h
