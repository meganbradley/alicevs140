# CPropExchange::IsLoading
Call this function to determine whether properties are being loaded to the control or saved from it.  
  
## Syntax  
  
```  
  
BOOL IsLoading( );  
```  
  
## Return Value  
 Nonzero if properties are being loaded; otherwise 0.  
  
## Requirements  
 **Header:** afxctl.h
