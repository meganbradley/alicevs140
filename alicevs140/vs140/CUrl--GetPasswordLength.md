# CUrl::GetPasswordLength
Call this method to get the length of the password.  
  
## Syntax  
  
```  
  
inline DWORD GetPasswordLength( ) const throw( );  
  
```  
  
## Return Value  
 Returns the password length.  
  
## Requirements  
 **Header:** atlutil.h
