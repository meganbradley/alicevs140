# CD2DBitmap::GetPixelFormat
Retrieves the pixel format and alpha mode of the bitmap  
  
## Syntax  
  
```  
D2D1_PIXEL_FORMAT GetPixelFormat() const;  
```  
  
## Return Value  
 The pixel format and alpha mode of the bitmap.  
  
## Requirements  
 **Header:** afxrendertarget.h
