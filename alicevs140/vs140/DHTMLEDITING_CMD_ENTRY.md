# DHTMLEDITING_CMD_ENTRY
Maps a command ID to an HTML editing command.  
  
## Syntax  
  
```  
  
DHTMLEDITING_CMD_ENTRY(  
cmdID  
,   
dhtmlcmdID  
)  
  
```  
  
#### Parameters  
 `cmdID`  
 The command ID (such as **ID_EDIT_COPY**).  
  
 `dhtmlcmdID`  
 The HTML editing command to which `cmdID` maps (such as **IDM_COPY**).
