# CTimeSpan::GetTimeSpan
Returns the value of the `CTimeSpan` object.  
  
## Syntax  
  
```  
__time64_t GetTimeSpan( ) const throw( );  
```  
  
## Return Value  
 Returns the current value of the `CTimeSpan` object.  
  
## Requirements  
 **Header:** atltime.h
