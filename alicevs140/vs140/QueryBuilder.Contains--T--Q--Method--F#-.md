# QueryBuilder.Contains&lt;&#39;T,&#39;Q&gt; Method (F#)

```  
// Signature:  
member this.Contains : QuerySource<'T,'Q> * 'T -> bool  
  
// Usage:  
queryBuilder.Contains (source, key)  
```

 Type: [QuerySource](../vs140/Linq.QuerySource--T--Q--Class--F#-.md)<'T,'Q>
