# CMFCRibbonEdit::GetIntermediateSize
Retrieves the intermediate size of the [CMFCRibbonEdit Class](../vs140/CMFCRibbonEdit-Class.md) object.  
  
## Syntax  
  
```  
virtual CSize GetIntermediateSize(  
   CDC* pDC  
);  
```  
  
#### Parameters  
 [in] `pDC`  
 Pointer to a device context for the [CMFCRibbonEdit Class](../vs140/CMFCRibbonEdit-Class.md) object.  
  
## Return Value  
 The intermediate size of the [CMFCRibbonEdit Class](../vs140/CMFCRibbonEdit-Class.md) object.

## Requirements  
 **Header:** afxribbonedit.h
