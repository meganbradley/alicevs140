# Module::objectCount_ Data Member
Keeps track of how many classes have been created with the [Make](../vs140/Make-Function.md) function.  
  
## Syntax  
  
```  
volatile long objectCount_;  
```  
  
## Requirements  
 **Header:** module.h  
  
 **Namespace:** Microsoft::WRL
