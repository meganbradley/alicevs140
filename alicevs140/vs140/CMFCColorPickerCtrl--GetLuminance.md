# CMFCColorPickerCtrl::GetLuminance
Retrieves the luminance component of the color that the user selects.  
  
## Syntax  
  
```  
double GetLuminance() const;  
```  
  
## Return Value  
 The luminance component of the selected color.

## Requirements  
 **Header:** afxcolorpickerctrl.h
