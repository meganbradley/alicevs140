# condition_variable_any::notify_all Method
Unblocks all threads that are waiting for the `condition_variable_any` object.  
  
## Syntax  
  
```  
void notify_all() _NOEXCEPT;  
```  
  
## Requirements  
 **Header:** condition_variable  
  
 **Namespace:** std
