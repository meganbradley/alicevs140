# phif Function
Returns the cumulative distribution function of the argument  
  
## Syntax  
  
```  
inline float phif(  
   float _X  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
## Return Value  
 Returns the cumulative distribution function of the argument  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::precise_math
