# CMDIFrameWndEx::SetPrintPreviewFrame
Sets the print preview frame window.  
  
## Syntax  
  
```  
void SetPrintPreviewFrame(  
   CFrameWnd* pWnd  
);  
```  
  
#### Parameters  
 [in] `pWnd`  
 Pointer to a print preview frame window.
