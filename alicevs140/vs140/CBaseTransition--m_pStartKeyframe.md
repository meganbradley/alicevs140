# CBaseTransition::m_pStartKeyframe
Stores a pointer to the keyframe that specifies the beginning of the transition.  
  
## Syntax  
  
```  
CBaseKeyFrame* m_pStartKeyframe;  
```  
  
## Requirements  
 **Header:** afxanimationcontroller.h
