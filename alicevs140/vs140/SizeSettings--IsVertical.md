# SizeSettings::IsVertical
Checks if the resize data specifies a nonzero vertical resizing.  
  
## Syntax  
  
```vb  
BOOL IsVertical() const  
```  
  
## Return Value  
 TRUE if the `SizeSettings` object specifies a nonzero vertical resizing.

## Requirements
