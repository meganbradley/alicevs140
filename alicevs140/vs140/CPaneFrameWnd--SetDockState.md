# CPaneFrameWnd::SetDockState
Sets the docking state.  
  
## Syntax  
  
```  
virtual void SetDockState(  
   CDockingManager* pDockManager  
);  
```  
  
#### Parameters  
 [in] `pDockManager`  
 A pointer to a docking manager.  
  
## Requirements  
 **Header:** afxpaneframewnd.h
