# CFrameWndEx::OnToolbarContextMenu
Called by the framework to build a toolbar pop-up menu.  
  
## Syntax  
  
```  
afx_msg LRESULT OnToolbarContextMenu(  
   WPARAM wp,  
   LPARAM lp  
);  
```  
  
#### Parameters  
 [in] `wp`  
 This parameter is not used.  
  
 [in] `lp`  
 This parameter is not used.  
  
## Return Value  
 Always returns 1.

## Requirements  
 **Header:** afxframewndex.h
