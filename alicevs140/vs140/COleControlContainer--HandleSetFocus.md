# COleControlContainer::HandleSetFocus
Determines if the container handles `WM_SETFOCUS` messages.  
  
## Syntax  
  
```  
  
virtual BOOL HandleSetFocus( );  
  
```  
  
## Return Value  
 Nonzero if the container handles `WM_SETFOCUS` messages; otherwise zero.  
  
## Requirements  
 **Header:** afxocc.h
