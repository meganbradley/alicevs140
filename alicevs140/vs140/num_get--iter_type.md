# num_get::iter_type
A type that describes an input iterator.  
  
## Syntax  
  
```  
typedef InputIterator iter_type;  
```  
  
## Remarks  
 The type is a synonym for the template parameter **InputIterator**.

 **Header:** <locale\>  
  
 **Namespace:** std
