# CDialog::SetDefID
Changes the default pushbutton control for a dialog box.  
  
## Syntax  
  
```  
  
      void SetDefID(  
   UINT nID   
);  
```  
  
#### Parameters  
 `nID`  
 Specifies the ID of the pushbutton control that will become the default.  
  
## Requirements  
 **Header:** afxwin.h
