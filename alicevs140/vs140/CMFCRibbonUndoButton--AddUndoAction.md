# CMFCRibbonUndoButton::AddUndoAction
Adds a new action to the list of actions.  
  
## Syntax  
  
```  
void AddUndoAction(  
   LPCTSTR lpszLabel   
);  
```  
  
#### Parameters  
 [in] `lpszLabel`  
 The action label that will be displayed in the drop-down list.  
  
## Requirements  
 **Header:** afxribbonundobutton.h
