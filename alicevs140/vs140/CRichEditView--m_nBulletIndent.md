# CRichEditView::m_nBulletIndent
The indentation for bullet items in a list; by default, 720 units, which is 1/2 inch.  
  
## Syntax  
  
```  
  
int m_nBulletIndent;  
  
```  
  
## Requirements  
 **Header:** afxrich.h
