# CMFCCaptionBar::SetButtonToolTip
Sets the tooltip for the button.  
  
## Syntax  
  
```  
void SetButtonToolTip(  
   LPCTSTR lpszToolTip,  
   LPCTSTR lpszDescription=NULL   
);  
```  
  
#### Parameters  
 [in] `lpszToolTip`  
 The tooltip caption.  
  
 [in] `lpszDescription`  
 The tooltip description.  
  
## Requirements  
 **Header:** afxcaptionbar.h
