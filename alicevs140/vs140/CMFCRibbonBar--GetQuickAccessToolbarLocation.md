# CMFCRibbonBar::GetQuickAccessToolbarLocation
Retrieves the display rectangle for the quick access toolbar.  
  
## Syntax  
  
```  
CRect GetQuickAccessToolbarLocation() const;  
```  
  
## Return Value  
 The display rectangle for the quick access toolbar.
