# CMFCRibbonCheckBox::OnDraw
Called by the framework to draw the checkbox using a specified device context.  
  
## Syntax  
  
```  
virtual void OnDraw(  
   CDC* pDC  
);  
```  
  
#### Parameters  
 [in] `pDC`  
 Pointer to the `CDC` in which to draw the checkbox.  
  
## Remarks  
  
## Requirements  
 **Header:** afxribboncheckbox.h
