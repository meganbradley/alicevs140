# IView::OnUpdate
Called by MFC after the view's document has been modified.  
  
## Syntax  
  
```  
void OnUpdate();  
```  
  
## Remarks  
 This function allows the view to update its display to reflect modifications.  
  
## Requirements  
 **Header:** afxwinforms.h
