# CFileDialog::AddText
Adds text to the dialog.  
  
## Syntax  
  
```  
HRESULT AddText(  
   DWORD dwIDCtl,  
   const CString& strText  
);  
```  
  
#### Parameters  
 `dwIDCtl`  
 The ID of the text to add.  
  
 `strText`  
 The text name.
