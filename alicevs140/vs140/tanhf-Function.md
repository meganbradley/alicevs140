# tanhf Function
Calculates the hyperbolic tangent value of the argument  
  
## Syntax  
  
```  
inline float tanhf(  
   float _X  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
## Return Value  
 Returns the hyperbolic tangent value of the argument  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::precise_math
