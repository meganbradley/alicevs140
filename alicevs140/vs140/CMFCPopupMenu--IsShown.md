# CMFCPopupMenu::IsShown
Indicates whether the pop-up menu is currently visible.  
  
## Syntax  
  
```  
BOOL IsShown() const;  
```  
  
## Return Value  
 `TRUE` if a pop-up menu is visible; otherwise `FALSE`.  
  
## Requirements  
 **Header:** afxpopupmenu.h
