# CFrameWndEx::IsMenuBarAvailable
Determines whether the pointer to the menu bar object is valid.  
  
## Syntax  
  
```  
BOOL IsMenuBarAvailable() const;  
```  
  
## Return Value  
 `TRUE` if the frame window has a menu bar; otherwise `FALSE`.  
  
## Requirements  
 **Header:** afxframewndex.h
