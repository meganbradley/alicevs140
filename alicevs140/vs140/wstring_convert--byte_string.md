# wstring_convert::byte_string
A type that represents a byte string.  
  
## Syntax  
  
```  
typedef std::basic_string<char> byte_string;  
```  
  
## Remarks  
 The type is a synonym for `std::basic_string<char>`.  
  
## Requirements  
 **Header:** <locale\>  
  
 **Namespace:** std
