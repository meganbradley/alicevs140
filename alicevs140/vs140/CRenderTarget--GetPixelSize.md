# CRenderTarget::GetPixelSize
Returns the size of the render target in device pixels  
  
## Syntax  
  
```  
CD2DSizeU GetPixelSize() const;  
```  
  
## Return Value  
 The size of the render target in device pixels  
  
## Requirements  
 **Header:** afxrendertarget.h
