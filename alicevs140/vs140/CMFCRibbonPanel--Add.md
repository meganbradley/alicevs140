# CMFCRibbonPanel::Add
Appends the specified ribbon element to the array of ribbon elements that is contained in the ribbon panel.  
  
## Syntax  
  
```  
virtual void Add(  
    CMFCRibbonBaseElement* pElem   
);  
```  
  
#### Parameters  
 [in, out] `pElem`  
 Pointer to a ribbon element.
