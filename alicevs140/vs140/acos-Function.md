# acos Function
Calculates the arccosine of the argument  
  
## Syntax  
  
```  
inline float acos(  
   float _X  
) restrict(amp);  
inline double acos(  
   double _X  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
## Return Value  
 Returns the arccosine value of the argument  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::precise_math
