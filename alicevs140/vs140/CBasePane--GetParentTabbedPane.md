# CBasePane::GetParentTabbedPane
Returns a pointer to the parent tabbed pane.  
  
## Syntax  
  
```  
CBaseTabbedPane* GetParentTabbedPane() const;  
```  
  
## Return Value  
 A pointer to the parent tabbed pane if it exists; otherwise `NULL`.  
  
## Requirements  
 **Header:** afxbasepane.h
