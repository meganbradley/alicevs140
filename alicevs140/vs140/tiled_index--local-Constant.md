# tiled_index::local Constant
Stores an [index](../vs140/index-Class.md) object of rank 1, 2, or 3 that represents the relative index in the current tile of a [tiled_extent](../vs140/tiled_extent-Class.md) object.  
  
## Syntax  
  
```  
const index<rank> local;  
```  
  
## Requirements  
 **Header:** amp.h  
  
 **Namespace:** Concurrency
