# radians Function
Converts _X from degrees to radians  
  
## Syntax  
  
```  
inline float radians(  
   float _X  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
## Return Value  
 Returns _X converted from degrees to radians  
  
## Requirements  
 **Header:** amp.h  
  
 **Namespace:** Concurrency::direct3d
