# CMFCRibbonBaseElement::GetSize
Returns the current size of the ribbon element.  
  
## Syntax  
  
```  
virtual CSize GetSize(  
   CDC* pDC   
);  
```  
  
#### Parameters  
 [in] `pDC`  
 A pointer to a device context.  
  
## Return Value  
 The current size of the ribbon element.  
  
## Requirements  
 **Header:** afxbaseribbonelement.h
