# CMFCPopupMenuBar::LoadFromHash
Loads an archived menu.  
  
## Syntax  
  
```  
BOOL LoadFromHash(  
   HMENU hMenu  
);  
```  
  
#### Parameters  
 [in] `hMenu`  
 A handle to the archived menu to load.  
  
## Return Value  
 Returns `TRUE` if the menu is loaded successfully, or `FALSE` if not.

## Requirements  
 **Header:** afxpopupmenubar.h
