# expf Function (fast_math)
Calculates the base-e exponential of the argument  
  
## Syntax  
  
```  
inline float expf(  
   float _X  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
## Return Value  
 Returns the base-e exponential of the argument  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::fast_math
