# BEGIN_PROPERTY_SET
Marks the beginning of a property set in a property set map.  
  
## Syntax  
  
```  
  
BEGIN_PROPERTY_SET(  
guid   
)  
  
```  
  
#### Parameters  
 `guid`  
 [in] The property GUID.  
  
## Example  
 See [BEGIN_PROPSET_MAP](../vs140/BEGIN_PROPSET_MAP.md).  
  
## Requirements  
 **Header:** atldb.h
