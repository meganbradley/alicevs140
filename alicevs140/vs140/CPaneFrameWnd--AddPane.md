# CPaneFrameWnd::AddPane
Adds a pane.  
  
## Syntax  
  
```  
virtual void AddPane(  
   CBasePane* pWnd  
);  
```  
  
#### Parameters  
 [in] `pWnd`  
 The pane to add.
