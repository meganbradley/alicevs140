# Function Specifiers
You can use the [inline](../vs140/inline--__inline--__forceinline.md) and [virtual](../vs140/virtual-Specifier.md) keywords as specifiers in function declarations. This use of **virtual** differs from its use in the base-class specifier of a class definition.
