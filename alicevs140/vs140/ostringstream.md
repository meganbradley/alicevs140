# ostringstream
Creates a type `basic_ostringstream` specialized on a `char` template parameter.  
  
## Syntax  
  
```  
  
typedef basic  
_  
ostringstream<char> ostringstream;  
  
```  
  
## Remarks  
 The type is a synonym for template class [basic_ostringstream](../vs140/basic_ostringstream-Class.md), specialized for elements of type `char`*.*  
  
## Requirements  
 **Header:** <sstream\>  
  
 **Namespace:** std
