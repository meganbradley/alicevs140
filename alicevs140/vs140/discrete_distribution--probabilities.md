# discrete_distribution::probabilities
Returns the stored probabilities for the distribution.  
  
## Syntax  
  
```  
std::vector<double> probabilities() const;  
```  
  
## Remarks  
 This member function returns the stored probability values.  
  
## Requirements  
 **Header:** <random\>  
  
 **Namespace:** std
