# CMFCRibbonPanel::GetCaptionHeight
Retrieves the height of a caption for the ribbon panel.  
  
## Syntax  
  
```  
int GetCaptionHeight() const;  
```  
  
## Return Value  
 The height, in pixels, of a caption for the ribbon panel.

## Requirements  
 **Header:** afxribbonpanel.h
