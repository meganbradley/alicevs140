# COleSafeArray::AllocData
Allocates memory for a safe array.  
  
## Syntax  
  
```  
  
void AllocData( );  
```  
  
## Remarks  
 On error, the function throws a [CMemoryException](../vs140/CMemoryException-Class.md) or [COleException](../vs140/COleException-Class.md).  
  
## Requirements  
 **Header:** afxdisp.h
