# Resource Compiler Fatal Error RC1203
invalid hexadecimal default language ID specified.  
  
 The Specify Default Language (/l) option was followed by an invalid hexadecimal language ID. See [Language and Country/Region Strings](../vs140/Locale-Names--Languages--and-Country-Region-Strings.md) in the *Run-Time Library Reference* for a list of valid language IDs.
