# ENDS (MASM)
Marks the end of segment, structure, or union *name* previously begun with [SEGMENT](../vs140/SEGMENT.md), [STRUCT](../vs140/STRUCT--MASM-.md), [UNION](../vs140/UNION.md), or a simplified segment directive.  
  
## Syntax  
  
```  
  
name ENDS  
```
