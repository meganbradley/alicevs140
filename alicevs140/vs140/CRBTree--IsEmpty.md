# CRBTree::IsEmpty
Call this method to test for an empty tree object.  
  
## Syntax  
  
```  
  
bool IsEmpty( ) const throw( );  
  
```  
  
## Return Value  
 Returns **true** if the tree is empty, **false** otherwise.  
  
## Requirements  
 **Header:** atlcoll.h
