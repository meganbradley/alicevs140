# CUserTool::m_strInitialDirectory
Specifies the initial directory for the user tool.  
  
## Syntax  
  
```  
CString m_strInitialDirectory;  
```  
  
## Remarks  
 This variable specifies the initial directory that the tool executes in when you call [CUserTool::Invoke](../vs140/CUserTool--Invoke.md) or when a user clicks the command associated with this tool.  
  
## Requirements  
 **Header:** afxusertool.h
