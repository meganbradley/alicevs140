# COleObjectFactory::IsLicenseValid
Determines if the license of the control is valid.  
  
## Syntax  
  
```  
  
BOOL IsLicenseValid( );  
  
```  
  
## Return Value  
 TRUE if successul; otherwise false.

 **Header:** afxdisp.h
