# CCmdUI::m_nID
The ID of the menu item, toolbar button, or other user-interface object represented by the `CCmdUI` object.  
  
## Syntax  
  
```  
  
UINT m_nID;  
  
```  
  
## Requirements  
 **Header:** afxwin.h
