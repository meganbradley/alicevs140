# CMFCRibbonBaseElement::IsChecked
Specifies whether the ribbon element is checked.  
  
## Syntax  
  
```  
virtual BOOL IsChecked() const;  
```  
  
## Return Value  
 `TRUE` if the ribbon element is checked; otherwise `FALSE`.  
  
## Requirements  
 **Header:** afxbaseribbonelement.h
