# CDocTemplate::SaveAllModified
Saves all documents that have been modified.  
  
## Syntax  
  
```  
  
virtual BOOL SaveAllModified( );  
  
```  
  
## Return Value  
 Non-zero if successful; otherwise 0.  
  
## Requirements  
 **Header:** afxwin.h
