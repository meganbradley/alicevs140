# END_MESSAGE_MAP
Ends the definition of your message map.  
  
## Syntax  
  
```  
  
END_MESSAGE_MAP( )  
```  
  
## Remarks  
 For more information on message maps and the `END_MESSAGE_MAP` macro, see [Message Handling and Mapping Topics](../vs140/Message-Handling-and-Mapping.md).  
  
## Requirements  
 **Header:** afxwin.h
