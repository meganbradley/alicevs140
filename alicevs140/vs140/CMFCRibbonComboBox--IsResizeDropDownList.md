# CMFCRibbonComboBox::IsResizeDropDownList
Indicates whether or not the list box can be resized.  
  
## Syntax  
  
```  
BOOL IsResizeDropDownList() const;  
```  
  
## Return Value  
 `TRUE` if the list box can be resized; otherwise `FALSE`.[CMFCRibbonComboBox::EnableDropDownListResize](../vs140/CMFCRibbonComboBox--EnableDropDownListResize.md)  
  
## Remarks  
 You can enable list box resizing by using the [EnableDropDownListResize](../vs140/CMFCRibbonComboBox--EnableDropDownListResize.md) method.

 **Header:** afxribboncombobox.h
