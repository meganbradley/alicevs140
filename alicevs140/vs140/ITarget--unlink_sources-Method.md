# ITarget::unlink_sources Method
When overridden in a derived class, unlinks all source blocks from this `ITarget` block.  
  
## Syntax  
  
```  
virtual void unlink_sources() = 0;  
```  
  
## Requirements  
 **Header:** agents.h  
  
 **Namespace:** concurrency
