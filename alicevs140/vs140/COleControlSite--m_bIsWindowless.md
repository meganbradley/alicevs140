# COleControlSite::m_bIsWindowless
Determines if the object is a windowless control.  
  
## Syntax  
  
```  
  
BOOL m_bIsWindowless;  
  
```  
  
## Remarks  
 Nonzero if the control has no window, otherwise zero.  
  
## Requirements  
 **Header:** afxocc.h
