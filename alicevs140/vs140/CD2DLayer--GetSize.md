# CD2DLayer::GetSize
Returns the size of the render target in device-independent pixels  
  
## Syntax  
  
```  
CD2DSizeF GetSize() const;  
```  
  
## Return Value  
 The current size of the render target in device-independent pixels  
  
## Requirements  
 **Header:** afxrendertarget.h
