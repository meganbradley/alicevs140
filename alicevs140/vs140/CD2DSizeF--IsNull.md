# CD2DSizeF::IsNull
Returns a Boolean value that indicates whether an expression contains no valid data (Null).  
  
## Syntax  
  
```  
BOOL IsNull() const;  
```  
  
## Return Value  
 TRUE if width and height are empty; otherwise FALSE.  
  
## Requirements  
 **Header:** afxrendertarget.h
