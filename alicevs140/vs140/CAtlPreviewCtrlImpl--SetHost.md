# CAtlPreviewCtrlImpl::SetHost
Sets a new parent for this control.  
  
## Syntax  
  
```  
virtual void SetHost(  
   HWND hWndParent  
);  
```  
  
#### Parameters  
 `hWndParent`  
 A handle to the new parent window.
