# critical_section::scoped_lock::~scoped_lock Destructor
Destroys a `scoped_lock` object and releases the critical section supplied in its constructor.  
  
## Syntax  
  
```  
~scoped_lock();  
```  
  
## Requirements  
 **Header:** concrt.h  
  
 **Namespace:** concurrency
