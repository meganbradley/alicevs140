# CComSafeArrayBound::GetLowerBound
Call this method to return the lower bound.  
  
## Syntax  
  
```  
  
LONG GetLowerBound( ) const throw( );  
  
```  
  
## Return Value  
 Returns the lower bound of the `CComSafeArrayBound` object.  
  
## Requirements  
 **Header:** atlsafe.h
