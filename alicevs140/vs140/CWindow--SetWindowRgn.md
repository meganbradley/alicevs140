# CWindow::SetWindowRgn
Sets the window region of a window.  
  
## Syntax  
  
```  
  
      int SetWindowRgn(  
   HRGN hRgn,  
   BOOL bRedraw = FALSE   
) throw();  
```  
  
## Remarks  
 See [SetWindowRgn](http://msdn.microsoft.com/library/windows/desktop/dd145102) in the [!INCLUDE[winSDK](../vs140/includes/winSDK_md.md)].
