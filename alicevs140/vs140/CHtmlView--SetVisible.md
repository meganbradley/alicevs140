# CHtmlView::SetVisible
Call this member function to set the visibility state of the WebBrowser control.  
  
## Syntax  
  
```  
  
      void SetVisible(  
   BOOL bNewValue   
);  
```  
  
#### Parameters  
 `bNewValue`  
 Nonzero if the control is visible; otherwise zero.  
  
## Remarks  
 Applies to Internet Explorer and WebBrowser.  
  
## Requirements  
 **Header:** afxhtml.h
