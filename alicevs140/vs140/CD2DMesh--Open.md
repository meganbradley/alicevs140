# CD2DMesh::Open
Opens the mesh for population.  
  
## Syntax  
  
```  
ID2D1TessellationSink* Open();  
```  
  
## Return Value  
 A pointer to an ID2D1TessellationSink that is used to populate the mesh.  
  
## Requirements  
 **Header:** afxrendertarget.h
