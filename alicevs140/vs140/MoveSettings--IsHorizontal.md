# MoveSettings::IsHorizontal
Check if the move data specifies a nonzero horizontal move.  
  
## Syntax  
  
```vb  
BOOL IsHorizontal() const  
```  
  
## Return Value  
 TRUE if the `MoveSettings` object specifies a nonzero horizontal move.
