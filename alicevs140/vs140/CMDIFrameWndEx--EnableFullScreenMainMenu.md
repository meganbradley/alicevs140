# CMDIFrameWndEx::EnableFullScreenMainMenu
Shows or hides the main menu in full-screen mode.  
  
## Syntax  
  
```  
void EnableFullScreenMainMenu(  
   BOOL bEnableMenu  
);  
```  
  
#### Parameters  
 [in] `bEnableMenu`  
 `TRUE` to show the main menu in full-screen mode, or `FALSE` to hide it.
