# CMFCRibbonBaseElement::SetTextAlwaysOnRight
Sets the text for the ribbon element to display on the right.  
  
## Syntax  
  
```  
virtual void SetTextAlwaysOnRight(  
   BOOL bSet = TRUE  
);  
```  
  
#### Parameters  
 [in] `bSet`  
 `TRUE` to display the text on the right; otherwise `FALSE`.
