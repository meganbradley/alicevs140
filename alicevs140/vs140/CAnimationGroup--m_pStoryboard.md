# CAnimationGroup::m_pStoryboard
Points to animation storyboard. This pointer is valid only after call on Animate.  
  
## Syntax  
  
```  
ATL::CComPtr<IUIAnimationStoryboard> m_pStoryboard;  
```  
  
## Requirements  
 **Header:** afxanimationcontroller.h
