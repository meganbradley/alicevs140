# CWnd::IsTouchWindow
Specifies whether `CWnd` has touch support.  
  
## Syntax  
  
```  
BOOL IsTouchWindow() const;  
```  
  
## Return Value  
 `TRUE` if `CWnd` has touch support; otherwise `FALSE`.  
  
## Remarks  
  
## Requirements  
 **Header:** afxwin.h
