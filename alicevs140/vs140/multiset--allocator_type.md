# multiset::allocator_type
A type that represents the allocator class for the multiset object  
  
## Syntax  
  
```  
typedef Allocator allocator_type;  
```  
  
## Remarks  
 `allocator_type` is a synonym for the template parameter `Allocator`.  
  
 For more information on `Allocator`, see the Remarks section of the [multiset Class](../vs140/multiset-Class.md) topic.  
  
## Example  
 See the example for [get_allocator](../vs140/multiset--get_allocator.md) for an example using `allocator_type`  
  
## Requirements  
 **Header:** <set\>  
  
 **Namespace:** std
