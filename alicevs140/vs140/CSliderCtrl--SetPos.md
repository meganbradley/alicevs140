# CSliderCtrl::SetPos
Sets the current position of the slider in a slider control.  
  
## Syntax  
  
```  
  
      void SetPos(  
   int nPos   
);  
```  
  
#### Parameters  
 `nPos`  
 Specifies the new slider position.  
  
## Requirements  
 **Header:** afxcmn.h
