# All Rules rule set
The All Rules rule set contains all of the rules for both native and managed code. The rule set includes all the rules that are described  in the following topics:  
  
1.  [Code Analysis for C/C++ Warnings](../vs140/Code-Analysis-for-C-C---Warnings.md)  
  
2.  [Code Analysis for Managed Code Warnings](../vs140/Code-Analysis-for-Managed-Code-Warnings.md)
