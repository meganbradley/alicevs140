# CFrameWnd::InModalState
Call this member function to check if a frame window is modal or modeless.  
  
## Syntax  
  
```  
  
BOOL InModalState( ) const;  
  
```  
  
## Return Value  
 Nonzero if yes; otherwise 0.  
  
## Requirements  
 **Header:** afxwin.h
