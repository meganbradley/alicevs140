# CMFCCaptionBar::EnableButton
Enables or disables the button on the caption bar.  
  
## Syntax  
  
```  
void EnableButton(  
   BOOL bEnable=TRUE   
);  
```  
  
#### Parameters  
 [in] `bEnable`  
 `TRUE` to enable the button, `FALSE` to disable the button.  
  
## Requirements  
 **Header:** afxcaptionbar.h
