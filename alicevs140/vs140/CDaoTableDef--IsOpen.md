# CDaoTableDef::IsOpen
Call this member function to determine whether the `CDaoTableDef` object is currently open.  
  
## Syntax  
  
```  
  
BOOL IsOpen( ) const;  
  
```  
  
## Return Value  
 Nonzero if the `CDaoTableDef` object is open; otherwise 0.
