# CMFCTasksPaneTaskGroup::m_lstTasks
Contains the internal list of tasks.  
  
## Syntax  
  
```  
CObList m_lstTasks;  
```  
  
## Remarks  
 To fill this list, call [CMFCTasksPane::AddTask](../vs140/CMFCTasksPane--AddTask.md).  
  
## Requirements  
 **Header:** afxTasksPane.h
