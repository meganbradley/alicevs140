# Navigating the Class Structure (Visual C++)
You can navigate a project class structure, including functions, inheritance and interfaces using either [Class View](assetId:///8d7430a9-3e33-454c-a9e1-a85e3d2db925) or the [Object Browser](assetId:///f89acfc5-1152-413d-9f56-3dc16e3f0470).  
  
 You can go to a function declaration by double-clicking on the function name in Class View. You can go to a class definition by double-clicking on a class name in Class View.
