# CMFCPopupMenuBar::GetDefaultMenuId
Returns the identifier of the default menu item.  
  
## Syntax  
  
```  
UINT GetDefaultMenuId( ) const;  
```  
  
## Return Value  
 Returns the identifier of the default menu item in the popup menu bar.

## Requirements
