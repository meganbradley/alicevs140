# CPane::OnAfterDock
Called by the framework when a pane has been docked.  
  
## Syntax  
  
```  
virtual void OnAfterDock(  
    CBasePane* pBar,  
    LPCRECT lpRect,  
    AFX_DOCK_METHOD dockMethod  
);  
```  
  
#### Parameters  
 [in] `pBar`  
 This parameter is not used.  
  
 [in] `lpRect`  
 This parameter is not used.  
  
 [in] `dockMethod`  
 This parameter is not used.  
  
## Requirements  
 **Header:** afxPane.h
