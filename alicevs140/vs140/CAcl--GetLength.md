# CAcl::GetLength
Returns the length of the access-control list (ACL).  
  
## Syntax  
  
```  
  
UINT GetLength( ) const throw( );  
  
```  
  
## Return Value  
 Returns the required length in bytes necessary to hold the **ACL** structure.  
  
## Requirements  
 **Header:** atlsecurity.h
