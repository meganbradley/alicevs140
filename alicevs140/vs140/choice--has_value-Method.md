# choice::has_value Method
Checks whether this `choice` messaging block has been initialized with a value yet.  
  
## Syntax  
  
```  
bool has_value() const;  
```  
  
## Return Value  
 `true` if the block has received a value, `false` otherwise.  
  
## Requirements  
 **Header:** agents.h  
  
 **Namespace:** concurrency
