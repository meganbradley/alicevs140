# CMFCOutlookBarPane::SetBackImage
Sets the background image.  
  
## Syntax  
  
```  
void SetBackImage(  
   UINT uiImageID   
);  
```  
  
#### Parameters  
 [in] `uiImageID`  
 Specifies the image resource ID.  
  
## Remarks  
 Call this method to set the Outlook bar's background image. The list of background images is managed by the embedded [CMFCToolBarImages Class](../vs140/CMFCToolBarImages-Class.md) object.  
  
## Requirements  
 **Header:** afxoutlookbarpane.h
