# CAtlModuleT::InitLibId
Initializes the data member containing the GUID of the current module.  
  
## Syntax  
  
```  
  
static void InitLibId( ) throw( );  
  
```  
  
## Remarks  
 Called by the constructor [CAtlModuleT::CAtlModuleT](../vs140/CAtlModuleT--CAtlModuleT.md).  
  
## Requirements  
 **Header:** atlbase.h
