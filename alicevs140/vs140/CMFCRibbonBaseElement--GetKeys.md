# CMFCRibbonBaseElement::GetKeys
Returns the keytip associated with the ribbon element.  
  
## Syntax  
  
```  
LPCTSTR GetKeys() const;  
```  
  
## Return Value  
 A keytip associated with the ribbon element.

 **Header:** afxbaseribbonelement.h
