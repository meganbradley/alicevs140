# CFileDialog::OnItemSelected
Called when the container item is selected.  
  
## Syntax  
  
```  
virtual void OnItemSelected(  
   DWORD dwIDCtl,  
   DWORD dwIDItem  
);  
```  
  
#### Parameters  
 `dwIDCtl`  
 The ID of the container control.  
  
 `dwIDItem`  
 The ID of the item.
