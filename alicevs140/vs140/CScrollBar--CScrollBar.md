# CScrollBar::CScrollBar
Constructs a `CScrollBar` object.  
  
## Syntax  
  
```  
  
CScrollBar( );  
```  
  
## Remarks  
 After constructing the object, call the **Create** member function to create and initialize the Windows scroll bar.
