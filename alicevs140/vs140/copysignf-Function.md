# copysignf Function
Produces a value with the magnitude of _X and the sign of _Y  
  
## Syntax  
  
```  
inline float copysignf(  
   float _X,  
   float _Y  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
 `_Y`  
 Floating-point value  
  
## Return Value  
 Returns a value with the magnitude of _X and the sign of _Y  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::precise_math
