# ceil Function
Calculates the ceiling of the argument  
  
## Syntax  
  
```  
inline float ceil(  
   float _X  
) restrict(amp);  
inline double ceil(  
   double _X  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
## Return Value  
 Returns the ceiling of the argument  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::precise_math
