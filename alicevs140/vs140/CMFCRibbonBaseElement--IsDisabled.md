# CMFCRibbonBaseElement::IsDisabled
Specifies whether the ribbon element is disabled.  
  
## Syntax  
  
```  
virtual BOOL IsDisabled() const;  
```  
  
## Return Value  
 `TRUE` if the ribbon element is disabled; otherwise `FALSE`.  
  
## Requirements  
 **Header:** afxbaseribbonelement.h
