# tan Function
Calculates the tangent value of the argument  
  
## Syntax  
  
```  
inline float tan(  
   float _X  
) restrict(amp);  
inline double tan(  
   double _X  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
## Return Value  
 Returns the tangent value of the argument  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::precise_math
