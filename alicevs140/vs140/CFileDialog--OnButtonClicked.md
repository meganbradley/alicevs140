# CFileDialog::OnButtonClicked
Called when the button is clicked.  
  
## Syntax  
  
```  
virtual void OnButtonClicked(  
   DWORD dwIDCtl  
);  
```  
  
#### Parameters  
 `dwIDCtl`  
 The ID of the button.
