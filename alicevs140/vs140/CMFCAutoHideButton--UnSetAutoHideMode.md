# CMFCAutoHideButton::UnSetAutoHideMode
Disable auto-hide mode.  
  
## Syntax  
  
```  
virtual void UnSetAutoHideMode(  
   CDockablePane* pFirstBarInGroup  
);  
```  
  
#### Parameters  
 [in] `pFirstBarInGroup`  
 A pointer to the first bar in the group.

## Requirements  
 **Header:** afxautohidebutton.h
