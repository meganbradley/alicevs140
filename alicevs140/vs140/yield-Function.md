# yield Function
Signals the operating system to run other threads, even if the current thread would ordinarily continue to run.  
  
## Syntax  
  
```  
inline void yield() _NOEXCEPT;  
```  
  
## Requirements  
 **Header:** thread  
  
 **Namespace:** std::this_thread
