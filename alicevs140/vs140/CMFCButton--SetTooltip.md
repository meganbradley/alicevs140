# CMFCButton::SetTooltip
Associates a tooltip with a button.  
  
## Syntax  
  
```  
void SetTooltip(  
   LPCTSTR lpszToolTipText   
);  
```  
  
#### Parameters  
 [in] `lpszToolTipText`  
 Pointer to the text for the tooltip. Specify NULL to disable the tooltip.
