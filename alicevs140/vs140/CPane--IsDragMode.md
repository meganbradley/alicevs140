# CPane::IsDragMode
Specifies whether the pane is being moved.  
  
## Syntax  
  
```  
virtual BOOL IsDragMode() const;  
```  
  
## Return Value  
 `TRUE` if the pane is being moved; otherwise, `FALSE`.

## Requirements  
 **Header:** afxpane.h
