# CMFCRibbonGallery::RedrawIcons
Redraws the gallery.  
  
## Syntax  
  
```  
void RedrawIcons();  
```  
  
## Remarks  
 Call this function to redraw the gallery. You must call this method if you have changed the contents of the gallery at runtime.  
  
## Requirements  
 **Header:** afxRibbonPaletteGallery.h
