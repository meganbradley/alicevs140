# shared_future::valid Method
Specifies whether the object has an *associated asynchronous state*.  
  
## Syntax  
  
```  
bool valid() _NOEXCEPT;  
```  
  
## Return Value  
 `true` if the object has an associated asynchronous state; otherwise, `false`.  
  
## Requirements  
 **Header:** future  
  
 **Namespace:** std
