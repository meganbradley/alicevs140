# -SECTION (DUMPBIN)
```  
/SECTION:section  
```  
  
## Remarks  
 This option restricts the output to information on the specified *section*. Use the [/HEADERS](../vs140/-HEADERS.md) option to get a list of sections in the file.  
  
 Only the [/HEADERS](../vs140/-HEADERS.md) DUMPBIN option is available for use on files produced with the [/GL](../Topic/-GL%20\(Whole%20Program%20Optimization\).md) compiler option.
