# CMFCRibbonBaseElement::SetParentMenu
Sets the parent menu container for the ribbon element.  
  
## Syntax  
  
```  
virtual void SetParentMenu(  
   CMFCRibbonPanelMenuBar* pMenuBar  
);  
```  
  
#### Parameters  
 [in] `pMenuBar`  
 The parent menu.
