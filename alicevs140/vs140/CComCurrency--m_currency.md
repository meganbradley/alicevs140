# CComCurrency::m_currency
The **CURRENCY** data member.  
  
## Syntax  
  
```  
  
CURRENCY m_currency;  
  
```  
  
## Remarks  
 This member holds the currency accessed and manipulated by the methods of this class.  
  
## Requirements  
 **Header:** atlcur.h
