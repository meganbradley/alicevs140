# basic_iostream::swap
Exchanges the contents of the provided `basic_iostream` object for the contents of this object.  
  
## Syntax  
  
```  
void swap(basic_iostream& _Right);  
```  
  
#### Parameters  
 `_Right`  
 The `basic_iostream` object to swap.  
  
## Remarks  
 The member function calls `(``_Right``)`
