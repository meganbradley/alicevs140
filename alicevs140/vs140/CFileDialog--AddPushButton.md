# CFileDialog::AddPushButton
Adds a button to the dialog.  
  
## Syntax  
  
```  
HRESULT AddPushButton(  
   DWORD dwIDCtl,  
   const CString& strLabel  
);  
```  
  
#### Parameters  
 `dwIDCtl`  
 The ID of the button to add.  
  
 `strLabel`  
 The button name.
