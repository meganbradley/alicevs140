# CMFCToolBarComboBoxButton::RemoveAllItems
Deletes all items from the list and edit boxes.  
  
## Syntax  
  
```  
void RemoveAllItems();  
```  
  
## Remarks  
 Removes all items from the list box and edit control of a combo box.  
  
## Requirements  
 **Header:** afxtoolbarcomboboxbutton.h
