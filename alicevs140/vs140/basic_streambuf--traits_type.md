# basic_streambuf::traits_type
Associates a type name with the **Tr** template parameter.  
  
## Syntax  
  
```  
  
typedef Tr traits  
_  
type;  
  
```  
  
## Requirements  
 **Header:** <streambuf\>  
  
 **Namespace:** std
