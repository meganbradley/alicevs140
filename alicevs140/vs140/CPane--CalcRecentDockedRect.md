# CPane::CalcRecentDockedRect
Calculates the recently docked rectangle.  
  
## Syntax  
  
```  
void CalcRecentDockedRect();  
```  
  
## Remarks  
 This method updates [CPane::m_recentDockInfo](../vs140/CPane--m_recentDockInfo.md).  
  
## Requirements  
 **Header:** afxpane.h
