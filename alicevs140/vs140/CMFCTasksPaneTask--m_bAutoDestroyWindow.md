# CMFCTasksPaneTask::m_bAutoDestroyWindow
Determines whether the task window is automatically destroyed.  
  
## Syntax  
  
```  
BOOL m_bAutoDestroyWindow;  
```  
  
## Remarks  
 Set to `TRUE` to specify that the task window ([CMFCTasksPaneTask::m_hwndTask](../vs140/CMFCTasksPaneTask--m_hwndTask.md)) should be destroyed automatically; otherwise, `FALSE`.  
  
## Requirements  
 **Header:** afxTasksPane.h
