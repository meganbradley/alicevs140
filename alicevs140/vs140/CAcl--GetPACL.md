# CAcl::GetPACL
Returns a pointer to an access-control list (ACL).  
  
## Syntax  
  
```  
  
const ACL * GetPACL( ) const throw(...);  
  
```  
  
## Return Value  
 Returns a pointer to the **ACL** structure.  
  
## Requirements  
 **Header:** atlsecurity.h
