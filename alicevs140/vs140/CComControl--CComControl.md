# CComControl::CComControl
The constructor.  
  
## Syntax  
  
```  
  
CComControl( );  
  
```  
  
## Remarks  
 Calls the [CComControlBase](../vs140/CComControlBase--CComControlBase.md) constructor, passing the `m_hWnd` data member inherited through [CWindowImpl](../vs140/CWindowImpl-Class.md).
