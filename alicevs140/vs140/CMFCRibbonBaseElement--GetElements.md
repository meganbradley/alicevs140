# CMFCRibbonBaseElement::GetElements
Adds the current ribbon element to the specified array.  
  
## Syntax  
  
```  
virtual void GetElements(  
   CArray<CMFCRibbonBaseElement*, CMFCRibbonBaseElement*>& arElements  
);  
```  
  
#### Parameters  
 [in, out] `arElements`  
 An array of ribbon elements.
