# CMFCTasksPane::RemoveAllTasks
Removes all tasks from the specified group.  
  
## Syntax  
  
```  
void RemoveAllTasks(  
    int nGroup  
);  
```  
  
#### Parameters  
 [in] `nGroup`  
 Specifies the zero-based index of the group.  
  
## Requirements  
 **Header:** afxTasksPane.h
