# CAtlBaseModule::m_bInitFailed
A variable that indicates if the module initialization has failed.  
  
## Syntax  
  
```  
  
static bool m_bInitFailed;  
  
```  
  
## Remarks  
 True if the module initialized, false if it failed to initialize.  
  
## Requirements  
 **Header:** atlcore.h
