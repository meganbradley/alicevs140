# CMFCTasksPane::IsWrapLabelsEnabled
Indicates whether the task pane wraps words in labels.  
  
## Syntax  
  
```  
BOOL IsWrapLabelsEnabled() const;  
```  
  
## Return Value  
 `TRUE` if words in labels are wrapped; otherwise, `FALSE`.  
  
## Requirements  
 **Header:** afxTasksPane.h
