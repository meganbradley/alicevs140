# CMFCRibbonPanel::RemoveAll
Deletes all ribbon elements from the ribbon panel.  
  
## Syntax  
  
```  
void RemoveAll();  
```  
  
## Remarks  
 All ribbon elements are deleted from the ribbon panel and destroyed.  
  
## Requirements  
 **Header:** afxRibbonPanel.h
