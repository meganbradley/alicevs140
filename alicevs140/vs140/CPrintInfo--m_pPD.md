# CPrintInfo::m_pPD
Contains a pointer to the `CPrintDialog` object used to display the Print dialog box for the print job.  
  
## Remarks  
 The `m_pPD` member is a public variable declared as a pointer to `CPrintDialog`.  
  
## Requirements  
 **Header:** afxext.h
