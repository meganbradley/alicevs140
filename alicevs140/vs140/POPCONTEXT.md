# POPCONTEXT
Restores part or all of the current `context` (saved by the [PUSHCONTEXT](../vs140/PUSHCONTEXT.md) directive). The `context` can be **ASSUMES**, `RADIX`, **LISTING**, **CPU**, or **ALL**.  
  
## Syntax  
  
```  
  
POPCONTEXT context  
```
