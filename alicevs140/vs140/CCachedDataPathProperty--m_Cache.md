# CCachedDataPathProperty::m_Cache
Contains the class name of the memory file into which data is cached.  
  
## Syntax  
  
```  
  
CMemFile m_Cache;  
```  
  
## Remarks  
 A memory file is stored in RAM rather than on disk.  
  
## Requirements  
 **Header:** afxctl.h
