# CMFCRibbonComboBox::GetDropDownHeight
Gets the height of the list box when the list box is dropped down.  
  
## Syntax  
  
```  
int GetDropDownHeight();  
```  
  
## Return Value  
 The height, in pixels, of the list box.

## Requirements  
 **Header:** afxribboncombobox.h
