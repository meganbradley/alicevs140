# basic_streambuf::epptr
A protected function that returns a pointer just past the end of the output buffer.  
  
## Syntax  
  
```  
  
char  
_  
type *epptr( ) const;  
  
```  
  
## Return Value  
 A pointer just past the end of the output buffer.  
  
## Requirements  
 **Header:** <streambuf\>  
  
 **Namespace:** std
