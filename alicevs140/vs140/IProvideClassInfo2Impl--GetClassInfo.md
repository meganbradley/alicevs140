# IProvideClassInfo2Impl::GetClassInfo
Retrieves an `ITypeInfo` pointer to the coclass' type information.  
  
## Syntax  
  
```  
  
      STDMETHOD(GetClassInfo)(  
   ITypeInfo** pptinfo   
);  
```
