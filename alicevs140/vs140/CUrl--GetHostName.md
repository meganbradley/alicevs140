# CUrl::GetHostName
Call this method to get the host name from the URL.  
  
## Syntax  
  
```  
  
inline LPCTSTR GetHostName( ) const throw( );  
  
```  
  
## Return Value  
 Returns the host name.  
  
## Requirements  
 **Header:** atlutil.h
