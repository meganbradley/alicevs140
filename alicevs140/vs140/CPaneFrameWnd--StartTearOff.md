# CPaneFrameWnd::StartTearOff
Tears off a menu.  
  
## Syntax  
  
```  
BOOL StartTearOff(  
   CMFCPopu* pMenu  
);  
```  
  
#### Parameters  
 [in] `pMenu`  
 A pointer to a menu.  
  
## Return Value  
 `TRUE` if the method was successful; otherwise, `FALSE`.  
  
## Requirements  
 **Header:** afxpaneframewnd.h
