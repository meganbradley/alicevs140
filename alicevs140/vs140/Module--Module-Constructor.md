# Module::Module Constructor
Initializes a new instance of the Module class.  
  
## Syntax  
  
```  
Module();  
```  
  
## Remarks  
 This constructor is protected and cannot be called with the `new` keyword. Instead, call either [Module::GetModule](../vs140/Module--GetModule-Method.md) or [Module::Create](../vs140/Module--Create-Method.md).  
  
## Requirements  
 **Header:** module.h  
  
 **Namespace:** Microsoft::WRL
