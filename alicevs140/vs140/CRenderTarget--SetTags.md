# CRenderTarget::SetTags
Specifies a label for subsequent drawing operations.  
  
## Syntax  
  
```  
void SetTags(  
   D2D1_TAG tag1,  
   D2D1_TAG tag2  
);  
```  
  
#### Parameters  
 `tag1`  
 A label to apply to subsequent drawing operations.  
  
 `tag2`  
 A label to apply to subsequent drawing operations.  
  
## Requirements  
 **Header:** afxrendertarget.h
