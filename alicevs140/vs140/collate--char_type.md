# collate::char_type
A type that describes a character of type **CharType**.  
  
## Syntax  
  
```  
  
typedef CharType char  
_  
type;  
  
```  
  
## Remarks  
 The type is a synonym for the template parameter **CharType**.  
  
## Requirements  
 **Header:** <locale\>  
  
 **Namespace:** std
