# CMDIFrameWndEx::GetMenuBar
Returns a pointer to a menu bar object attached to the frame window.  
  
## Syntax  
  
```  
const CMFCMenuBar* GetMenuBar() const;  
```  
  
## Return Value  
 A pointer to a menu bar object.  
  
## Requirements  
 **Header:** afxMDIFrameWndEx.h
