# basic_ofstream::close
Closes a file.  
  
## Syntax  
  
```  
  
void close( );  
  
```  
  
## Remarks  
 The member function calls [rdbuf](../vs140/basic_ifstream--rdbuf.md) **->** [close](../vs140/basic_filebuf--close.md).  
  
## Example  
 See [basic_filebuf::close](../vs140/basic_filebuf--close.md) for an example that uses **close**.  
  
## Requirements  
 **Header:** <fstream\>  
  
 **Namespace:** std
