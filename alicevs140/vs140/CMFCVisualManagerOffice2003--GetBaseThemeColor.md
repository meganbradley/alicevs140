# CMFCVisualManagerOffice2003::GetBaseThemeColor
Gets the base theme color.  
  
## Syntax  
  
```  
virtual COLORREF GetBaseThemeColor();  
```  
  
## Return Value  
 Returns the theme color of the base theme, if one is set, or the color-bar face color.
