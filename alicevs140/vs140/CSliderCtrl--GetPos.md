# CSliderCtrl::GetPos
Retrieves the current position of the slider in a slider control.  
  
## Syntax  
  
```  
  
int GetPos( ) const;  
```  
  
## Return Value  
 The current position.  
  
## Requirements  
 **Header:** afxcmn.h
