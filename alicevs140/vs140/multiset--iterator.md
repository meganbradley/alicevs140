# multiset::iterator
A type that provides a constant [bidirectional iterator](../vs140/bidirectional_iterator_tag-Struct.md) that can read any element in a multiset.  
  
## Syntax  
  
```  
typedef implementation-defined iterator;  
```  
  
## Example  
 See the example for [begin](../vs140/multiset--begin.md) for an example of how to declare and use an **iterator**.  
  
## Requirements  
 **Header:** <set\>  
  
 **Namespace:** std
