# element_type
The type is a synonym for the template parameter `Type`.  
  
## Syntax  
  
```  
typedef Type element_type;  
```  
  
## Remarks  
 The type is a synonym for the template parameter `Ty`.

 **Header:** <memory\>  
  
 **Namespace:** std
