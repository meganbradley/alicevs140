# &lt;example&gt; (C# Programming Guide)
## Syntax  
  
```  
<example>description</example>  
```  
  
#### Parameters  
 `description`  
 A description of the code sample.  
  
## Remarks  
 The <example\> tag lets you specify an example of how to use a method or other library member. This commonly involves using the [<code\>](../vs140/-code---C#-Programming-Guide-.md) tag.  
  
 Compile with [/doc](../Topic/-doc%20\(C%23%20Compiler%20Options\).md) to process documentation comments to a file.
