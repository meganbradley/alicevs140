# CComObjectGlobal::m_hResFinalConstruct
Contains the `HRESULT` from calling `FinalConstruct` during construction of the `CComObjectGlobal` object.  
  
## Syntax  
  
```  
  
HRESULT m_hResFinalConstruct;  
```  
  
## Requirements  
 **Header:** atlcom.h
