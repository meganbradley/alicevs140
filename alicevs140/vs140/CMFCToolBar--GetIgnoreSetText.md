# CMFCToolBar::GetIgnoreSetText
Specifies whether calls to set button labels are ignored.  
  
## Syntax  
  
```  
BOOL GetIgnoreSetText() const;  
```  
  
## Return Value  
 `TRUE` if calls to set button labels are ignored; otherwise, `FALSE`.

## Requirements  
 **Header:** afxtoolbar.h
