# operator OVERFLOW?
Status of overflow flag. Used only within [.IF](../vs140/.IF.md), [.WHILE](../vs140/.WHILE.md), or [.REPEAT](../vs140/.REPEAT.md) blocks and evaluated at run time, not at assembly time.  
  
## Syntax  
  
```  
  
OVERFLOW?  
  
```
