# CD2DLayer::Get
Returns ID2D1Layer interface  
  
## Syntax  
  
```  
ID2D1Layer* Get();  
```  
  
## Return Value  
 Pointer to an ID2D1Layer interface or NULL if object is not initialized yet.  
  
## Requirements  
 **Header:** afxrendertarget.h
