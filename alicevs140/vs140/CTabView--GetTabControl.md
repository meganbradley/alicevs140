# CTabView::GetTabControl
Returns a reference to the tab control associated with the view.  
  
## Syntax  
  
```  
DECLARE_DYNCREATE CMFCTabCtrl& GetTabControl();  
```  
  
## Return Value  
 A reference to the tab control associated with the view.  
  
## Requirements  
 **Header:** afxTabView.h
