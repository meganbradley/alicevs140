# CMFCRibbonBaseElement::GetKeyTipSize
Retrieves the size of the keytip text.  
  
## Syntax  
  
```  
virtual CSize GetKeyTipSize(  
   CDC* pDC  
);  
```  
  
#### Parameters  
 [in] `pDC`  
 Pointer to a device context.  
  
## Return Value  
 The size of the keytip text.

## Requirements
