# CD2DBitmap::GetPixelSize
Returns the size, in device-dependent units (pixels), of the bitmap  
  
## Syntax  
  
```  
CD2DSizeU GetPixelSize() const;  
```  
  
## Return Value  
 The size, in pixels, of the bitmap..  
  
## Requirements  
 **Header:** afxrendertarget.h
