# .CONTINUE
Generates code to jump to the top of a [.WHILE](../vs140/.WHILE.md) or [.REPEAT](../vs140/.REPEAT.md) block if `condition` is true.  
  
## Syntax  
  
```  
  
.CONTINUE [[.IF condition]]  
```
