# CRichEditView::GetPrintWidth
Call this function to determine the width of the printing area.  
  
## Syntax  
  
```  
  
int GetPrintWidth( ) const;  
  
```  
  
## Return Value  
 The width of the printing area, measured in `MM_TWIPS`.  
  
## Requirements  
 **Header:** afxrich.h
