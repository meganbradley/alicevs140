# CAnimationGroup::CAnimationGroup
Constructs an animation group.  
  
## Syntax  
  
```  
CAnimationGroup(  
   CAnimationController* pParentController,  
   UINT32 nGroupID  
);  
```  
  
#### Parameters  
 `pParentController`  
 A pointer to animation controller that creates a group.  
  
 `nGroupID`  
 Specifies GroupID.  
  
## Requirements  
 **Header:** afxanimationcontroller.h
