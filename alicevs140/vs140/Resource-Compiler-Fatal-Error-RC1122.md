# Resource Compiler Fatal Error RC1122
I/O error writing file  
  
 The Resource Compiler could not write to a file.  
  
### To fix by checking the following possible causes  
  
1.  Insufficient disk space. Free space must equal about twice the size of the executable file you are creating.  
  
2.  Volume is read-only.  
  
3.  Bad sector.  
  
4.  Sharing violation.
