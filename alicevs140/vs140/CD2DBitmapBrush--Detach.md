# CD2DBitmapBrush::Detach
Detaches resource interface from the object  
  
## Syntax  
  
```  
ID2D1BitmapBrush* Detach();  
```  
  
## Return Value  
 Pointer to detached resource interface.  
  
## Requirements  
 **Header:** afxrendertarget.h
