# CUrl::GetUrlPath
Call this method to get the URL path.  
  
## Syntax  
  
```  
  
inline LPCTSTR GetUrlPath( ) const throw( );  
  
```  
  
## Return Value  
 Returns the URL path.  
  
## Requirements  
 **Header:** atlutil.h
