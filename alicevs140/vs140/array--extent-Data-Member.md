Gets the [extent](../vs140/extent-Class--C---AMP-.md) object that defines the shape of the [array](../vs140/array-Class.md).  
  
## Syntax  
  
```  
__declspec(property(get=get_extent)) Concurrency::extent<_Rank> extent;  
```
