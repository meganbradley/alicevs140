# COleServerDoc::IsDocObject
Determines if the document is a DocObject.  
  
## Syntax  
  
```  
  
BOOL IsDocObject( ) const;  
  
```  
  
## Return Value  
 **TRUE** if the document is a DocObject; otherwise **FALSE**.  
  
## Requirements  
 **Header:** afxole.h
