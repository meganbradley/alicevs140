# CHtmlView::GetHeight
Call this member function to retrieve the height, in pixels, of the WebBrowser control's frame window.  
  
## Syntax  
  
```  
  
long GetHeight( ) const;  
  
```  
  
## Return Value  
 The control's frame window height, in pixels.  
  
## Requirements  
 **Header:** afxhtml.h
