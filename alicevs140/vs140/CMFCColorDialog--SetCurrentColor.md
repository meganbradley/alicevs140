# CMFCColorDialog::SetCurrentColor
Sets the current color of the dialog box.  
  
## Syntax  
  
```  
void SetCurrentColor(  
   COLORREF rgb   
);  
```  
  
#### Parameters  
 [in] `rgb`  
 An RGB color value
