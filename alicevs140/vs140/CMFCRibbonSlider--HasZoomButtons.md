# CMFCRibbonSlider::HasZoomButtons
Specifies whether the slider has zoom buttons.  
  
## Syntax  
  
```  
BOOL HasZoomButtons() const;  
```  
  
## Return Value  
 `TRUE` if the slider has zoom buttons; `FALSE` otherwise.  
  
## Requirements  
 **Header:** afxribbonslider.h
