# CFileFind::CFileFind
This member function is called when a `CFileFind` object is constructed.  
  
## Syntax  
  
```  
  
      CFileFind( );  
CFileFind(  
   CAtlTransactionManager* pTM  
);  
```  
  
#### Parameters  
 `pTM`  
 Pointer to CAtlTransactionManager object
