# CKeyFrame::m_offset
Specifies offset of this keyframe from a keyframe stored in m_pExistingKeyFrame.  
  
## Syntax  
  
```  
UI_ANIMATION_SECONDS m_offset;  
```  
  
## Requirements  
 **Header:** afxanimationcontroller.h
