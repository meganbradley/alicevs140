# CSliderCtrl::GetRangeMax
Retrieves the maximum position for the slider in a slider control.  
  
## Syntax  
  
```  
  
int GetRangeMax( ) const;  
```  
  
## Return Value  
 The control's maximum position.  
  
## Requirements  
 **Header:** afxcmn.h
