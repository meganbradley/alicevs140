# END_CONNECTION_PART
Ends the declaration of your connection point.  
  
## Syntax  
  
```  
  
END_CONNECTION_PART(  
localClass )  
```  
  
#### Parameters  
 *localClass*  
 Specifies the name of the local class that implements the connection point.  
  
## Requirements  
 **Header:** afxdisp.h
