# fabs Function (fast_math)
Returns the absolute value of the argument  
  
## Syntax  
  
```  
inline float fabs(  
   float _X  
) restrict(amp);  
  
```  
  
#### Parameters  
 `_X`  
 Integer value  
  
## Return Value  
 Returns the absolute value of the argument  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::fast_math
