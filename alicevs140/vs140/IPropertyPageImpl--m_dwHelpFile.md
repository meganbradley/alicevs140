# IPropertyPageImpl::m_dwHelpFile
Stores the resource identifier associated with the name of the help file describing the property page.  
  
## Syntax  
  
```  
  
UINT m_dwHelpFile;  
  
```  
  
## Requirements  
 **Header:** atlctl.h
