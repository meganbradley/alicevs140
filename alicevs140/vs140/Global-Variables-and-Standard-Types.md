# Global Variables and Standard Types
The Microsoft run-time library contains definitions for [global variables](../vs140/Global-Variables.md), [control flags](../vs140/Control-Flags.md), and [standard types](../vs140/Standard-Types.md) used by library routines. Access these variables, flags, and types by declaring them in your program or by including the appropriate header files.
