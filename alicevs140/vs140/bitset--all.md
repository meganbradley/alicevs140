# bitset::all
Tests all of the bits in this bitset to determine if they are all set to true.  
  
## Syntax  
  
```  
  
bool all() const;  
  
```  
  
## Return Value  
 Returns true if all bits in this set are true. Returns **false** if one or more bits are false.  
  
## Requirements  
 **Header:** <bitset\>  
  
 **Namespace:** std
