# CMDIFrameWndEx::GetMDITabs
Returns a reference to the underlined tabbed window.  
  
## Syntax  
  
```  
CMFCTabCtrl& GetMDITabs();  
```  
  
## Return Value  
 A reference to the underlined tabbed window.  
  
## Requirements  
 **Header:** afxMDIFrameWndEx.h
