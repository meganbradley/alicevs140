# CMFCBaseToolBar::OnAfterChangeParent
Called by the framework after the pane's parent changes.  
  
## Syntax  
  
```  
virtual void OnAfterChangeParent(  
   CWnd* pWndOldParent  
);  
```  
  
#### Parameters  
 [in] `pWndOldParent`  
 A pointer to the previous parent window.  
  
## Requirements  
 **Header:** afxbasetoolbar.h
