# CMFCAutoHideBar::SetActiveInGroup
Flags an auto-hide bar as active.  
  
## Syntax  
  
```  
virtual void SetActiveInGroup(  
		    BOOL bActive  
		);  
  
```  
  
#### Parameters  
 [in] BOOL `bActive`  
 TRUE to set to active; otherwise FALSE.  
  
## Remarks  
 See [CPane::SetActiveInGroup](../vs140/CPane--SetActiveInGroup.md).  
  
## Requirements  
 **Header:** afxautohidebar.h
