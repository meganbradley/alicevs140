# condition_variable::notify_all Method
Unblocks all threads that are waiting for the `condition_variable` object.  
  
## Syntax  
  
```  
void notify_all() _NOEXCEPT;  
```  
  
## Requirements  
 **Header:** condition_variable  
  
 **Namespace:** std
