# CTreeCtrl::GetIndent
Call this function to retrieve the amount, in pixels, that child items are indented relative to their parent items.  
  
## Syntax  
  
```  
  
UINT GetIndent( ) const;  
```  
  
## Return Value  
 The amount of indentation measured in pixels.
