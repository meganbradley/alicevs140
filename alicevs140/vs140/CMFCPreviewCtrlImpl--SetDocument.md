# CMFCPreviewCtrlImpl::SetDocument
Called by the preview handler to create a relationship between the document implementation and the preview control.  
  
## Syntax  
  
```  
void SetDocument(  
   IDocument* pDocument  
);  
```  
  
#### Parameters  
 `pDocument`  
 A pointer to the document implementation.
