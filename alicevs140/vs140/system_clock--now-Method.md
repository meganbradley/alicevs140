# system_clock::now Method
Static method that returns the current time.  
  
## Syntax  
  
```  
static time_point now() _NOEXCEPT;  
```  
  
## Return Value  
 A [time_point](../vs140/time_point-Class.md) object that represents the current time.  
  
## Requirements  
 **Header:** chrono  
  
 **Namespace:** std::chrono
