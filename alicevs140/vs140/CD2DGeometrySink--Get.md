# CD2DGeometrySink::Get
Returns ID2D1GeometrySink interface  
  
## Syntax  
  
```  
ID2D1GeometrySink* Get();  
```  
  
## Return Value  
 Pointer to an ID2D1GeometrySink interface or NULL if object is not initialized yet.

 **Header:** afxrendertarget.h
