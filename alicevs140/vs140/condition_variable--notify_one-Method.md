# condition_variable::notify_one Method
Unblocks one of the threads that are waiting on the `condition_variable` object.  
  
## Syntax  
  
```  
void notify_one() _NOEXCEPT;  
```  
  
## Requirements  
 **Header:** condition_variable  
  
 **Namespace:** std
