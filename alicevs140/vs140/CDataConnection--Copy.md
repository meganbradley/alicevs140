# CDataConnection::Copy
Creates a copy of an existing data connection.  
  
## Syntax  
  
```  
  
      CDataConnection& Copy(   
   const CDataConnection & ds    
) throw( );  
```  
  
#### Parameters  
 `ds`  
 [in] A reference to an existing data connection to copy.  
  
## Requirements  
 **Header:** atldbcli.h
