# CMFCTabCtrl::ActivateMDITab
Displays the specified tab of the current tab control and sets the focus on that tab.  
  
## Syntax  
  
```  
void ActivateMDITab(  
   int nTab = -1  
);  
```  
  
#### Parameters  
 [in] `nTab`  
 The zero-based index of a tab to display, or -1 to specify the currently active tab.  
  
## Requirements  
 **Header:** afxtabctrl.h
