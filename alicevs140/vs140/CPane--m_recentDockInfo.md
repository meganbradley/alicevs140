# CPane::m_recentDockInfo
Contains recent docking information.  
  
## Syntax  
  
```  
CRecentDockSiteInfo m_recentDockInfo;  
```  
  
## Remarks  
 The framework stores the latest docking state information for the pane in this member.  
  
## Requirements  
 **Header:** afxPane.h
