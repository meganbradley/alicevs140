# CMFCRibbonBaseElement::IsHighlighted
Specifies whether ribbon element is highlighted.  
  
## Syntax  
  
```  
virtual BOOL IsHighlighted() const;  
```  
  
## Return Value  
 `TRUE` if the ribbon element is highlighted; otherwise `FALSE`.

## Requirements
