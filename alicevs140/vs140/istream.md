# istream
A type `basic_istream` specialized on `char`.  
  
## Syntax  
  
```  
  
typedef basic  
_  
istream<char, char  
_  
traits<char> > istream;  
  
```  
  
## Remarks  
 The type is a synonym for template class [basic_istream](../vs140/basic_istream-Class.md), specialized for elements of type `char` with default character traits.  
  
## Requirements  
 **Header:** <istream\>  
  
 **Namespace:** std
