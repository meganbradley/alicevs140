# time_put::char_type
A type that is used to describe a character used by a locale.  
  
## Syntax  
  
```  
  
typedef CharType char  
_  
type;  
  
```  
  
## Remarks  
 The type is a synonym for the template parameter **CharType**.  
  
## Requirements  
 **Header:** <locale\>  
  
 **Namespace:** std
