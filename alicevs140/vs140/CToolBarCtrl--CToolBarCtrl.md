# CToolBarCtrl::CToolBarCtrl
Constructs a `CToolBarCtrl` object.  
  
## Syntax  
  
```  
  
CToolBarCtrl( );  
```  
  
## Remarks  
 You must call [Create](../vs140/CToolBarCtrl--Create.md) to make the toolbar usable.
