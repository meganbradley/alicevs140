# forward_list::value_type
A type that represents the type of element stored in a forward list.  
  
## Syntax  
  
```  
typedef typename Allocator::value_type value_type;  
```  
  
## Remarks  
 The type is a synonym for the template parameter _`Ty`.  
  
## Requirements  
 **Header:** <forward_list>  
  
 **Namespace:** std
