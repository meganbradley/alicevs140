# Compiler Error CS1583
'file' is not a valid Win32 resource file  
  
 This error occurs when you specify a filename with the [/win32res](../vs140/-win32res--C#-Compiler-Options-.md) compiler option that is not a valid or correctly formatted Win32 resource file. In Visual Studio, the filename is specified in the Application pane of the Project Designer.
