# CWnd::GetRenderTarget
Gets a render target that is associated with this window.  
  
## Syntax  
  
```  
CHwndRenderTarget* GetRenderTarget();  
```  
  
## Return Value  
 Pointer to the render target or NULL.  
  
## Requirements  
 **Header:** afxwin.h
