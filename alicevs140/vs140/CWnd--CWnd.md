# CWnd::CWnd
Constructs a `CWnd` object.  
  
## Syntax  
  
```  
  
CWnd( );  
```  
  
## Remarks  
 The Windows window is not created and attached until the [CreateEx](../vs140/CWnd--CreateEx.md) or [Create](../vs140/CWnd--Create.md) member function is called.  
  
## Requirements  
 **Header:** afxwin.h
