# HandleT::IsValid Method
Indicates whether the current HandleT object represents a handle.  
  
## Syntax  
  
```  
bool IsValid() const;  
```  
  
## Return Value  
 `true` if the HandleT represents a handle; otherwise, `false`.  
  
## Requirements  
 **Header:** corewrappers.h  
  
 **Namespace:** Microsoft::WRL::Wrappers
