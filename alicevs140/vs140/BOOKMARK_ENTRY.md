# BOOKMARK_ENTRY
Binds the bookmark column.  
  
## Syntax  
  
```  
  
BOOKMARK_ENTRY(  
variable  
 )  
  
```  
  
#### Parameters  
 *variable*  
 [in] The variable to be bound to the bookmark column.
