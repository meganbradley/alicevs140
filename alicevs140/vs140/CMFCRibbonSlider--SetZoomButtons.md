# CMFCRibbonSlider::SetZoomButtons
Display or hide zoom buttons.  
  
## Syntax  
  
```  
void SetZoomButtons(  
   BOOL bSet=TRUE   
);  
```  
  
#### Parameters  
 [in].`bSet`  
 `TRUE` to display zoom buttons; `FALSE` to hide them.  
  
## Requirements  
 **Header:** afxribbonslider.h
