# CMFCToolBarInfo::m_uiLargeDisabledResID
Specifies a resource ID for all the large disabled button images of a toolbar.  
  
## Syntax  
  
```  
UINT m_uiLargeDisabledResID;  
```  
  
## Requirements  
 **Header:** afxtoolbar.h
