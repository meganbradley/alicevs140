# CSacl::GetAceCount
Returns the number of access-control entries (ACEs) in the `CSacl` object.  
  
## Syntax  
  
```  
  
UINT GetAceCount( ) const throw( );  
  
```  
  
## Return Value  
 Returns the number of ACEs contained in the `CSacl` object.  
  
## Requirements  
 **Header:** atlsecurity.h
