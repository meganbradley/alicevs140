# CMFCToolBarImages::IsUserImagesList
Specifies whether this set of toolbar images contains user-defined images.  
  
## Syntax  
  
```  
BOOL IsUserImagesList() const;  
```  
  
## Return Value  
 `TRUE` if the [CMFCToolbarImages](../vs140/CMFCToolBarImages-Class.md) object contains user-defined toolbar images; otherwise `FALSE`.  
  
## Requirements  
 **Header:** afxtoolbarimages.h
