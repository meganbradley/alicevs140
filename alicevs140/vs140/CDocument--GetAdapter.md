# CDocument::GetAdapter
Returns a pointer to an object implementing the `IDocument` interface.  
  
## Syntax  
  
```  
virtual ATL::IDocument* GetAdapter();  
```  
  
## Return Value  
 A pointer to an object implementing the `IDocument` interface.
