# CPathT::RemoveBlanks
Call this method to remove all leading and trailing spaces from the path.  
  
## Syntax  
  
```  
  
void RemoveBlanks( );  
  
```  
  
## Remarks  
 For more information, see [PathRemoveBlanks](http://msdn.microsoft.com/library/windows/desktop/bb773745).  
  
## Requirements  
 **Header:** atlpath.h
