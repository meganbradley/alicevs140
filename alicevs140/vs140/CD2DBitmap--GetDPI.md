# CD2DBitmap::GetDPI
Return the dots per inch (DPI) of the bitmap  
  
## Syntax  
  
```  
CD2DSizeF GetDPI() const;  
```  
  
## Return Value  
 The horizontal and vertical DPI of the bitmap.  
  
## Requirements  
 **Header:** afxrendertarget.h
