# CWindow::DlgDirSelect
Retrieves the current selection from a list box.  
  
## Syntax  
  
```  
  
      BOOL DlgDirSelect(  
   LPTSTR lpString,  
   int nCount,  
   int nIDListBox   
) throw();  
```  
  
## Remarks  
 See [DlgDirSelectEx](http://msdn.microsoft.com/library/windows/desktop/bb761368) in the [!INCLUDE[winSDK](../vs140/includes/winSDK_md.md)].
