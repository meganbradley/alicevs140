# IDispatchImpl::IDispatchImpl
The constructor. Calls `AddRef` on the protected member variable that manages the type information for the dual interface. The destructor calls **Release**.  
  
## Syntax  
  
```  
  
IDispatchImpl( );  
  
```  
  
## Requirements  
 **Header:** atlcom.h
