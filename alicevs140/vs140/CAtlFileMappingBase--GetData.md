# CAtlFileMappingBase::GetData
Call this method to get the data from a file-mapping object.  
  
## Syntax  
  
```  
  
void* GetData( ) const throw( );  
  
```  
  
## Return Value  
 Returns a pointer to the data.  
  
## Requirements  
 **Header:** atlfile.h
