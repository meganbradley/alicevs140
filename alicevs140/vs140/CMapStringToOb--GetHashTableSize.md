# CMapStringToOb::GetHashTableSize
Determines the current number of elements in the hash table.  
  
## Syntax  
  
```  
  
UINT GetHashTableSize( ) const;  
  
```  
  
## Return Value  
 Returns the number of elements in the hash table.
