# CFileDialog::AddEditBox
Adds an edit box to the dialog.  
  
## Syntax  
  
```  
HRESULT AddEditBox(  
   DWORD dwIDCtl,  
   const CString& strText  
);  
```  
  
#### Parameters  
 `dwIDCtl`  
 The ID of the edit box to add.  
  
 `strText`  
 The edit box name.
