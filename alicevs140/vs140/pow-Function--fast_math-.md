# pow Function (fast_math)
Calculates _X raised to the power of _Y  
  
## Syntax  
  
```  
inline float pow(  
   float _X,  
   float _Y  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value, base  
  
 `_Y`  
 Floating-point value, exponent  
  
## Return Value  
 Returns the value of _X raised to the power of _Y  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::fast_math
