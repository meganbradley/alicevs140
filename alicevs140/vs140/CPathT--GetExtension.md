# CPathT::GetExtension
Call this method to get the file extension from the path.  
  
## Syntax  
  
```  
  
StringType GetExtension( ) const;  
  
```  
  
## Return Value  
 Returns the file extension.  
  
## Requirements  
 **Header:** atlpath.h
