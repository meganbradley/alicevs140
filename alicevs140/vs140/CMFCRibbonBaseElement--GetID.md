# CMFCRibbonBaseElement::GetID
Returns the command ID of the ribbon element.  
  
## Syntax  
  
```  
UINT GetID() const;  
```  
  
## Return Value  
 The command ID of the ribbon element.  
  
## Requirements  
 **Header:** afxbaseribbonelement.h
