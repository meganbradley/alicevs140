# CToolTipCtrl::GetToolCount
Retrieves a count of the tools registered with the tool tip control.  
  
## Syntax  
  
```  
  
int GetToolCount( ) const;  
```  
  
## Return Value  
 A count of tools registered with the tool tip control.  
  
## Requirements  
 **Header:** afxcmn.h
