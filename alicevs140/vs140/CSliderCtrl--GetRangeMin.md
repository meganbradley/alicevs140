# CSliderCtrl::GetRangeMin
Retrieves the minimum position for the slider in a slider control.  
  
## Syntax  
  
```  
  
int GetRangeMin( ) const;  
```  
  
## Return Value  
 The control's minimum position.  
  
## Requirements  
 **Header:** afxcmn.h
