# CMFCPreviewCtrlImpl::DoPaint
Called by the framework to render the preview.  
  
## Syntax  
  
```  
virtual void DoPaint(  
   CPaintDC* pDC  
);  
```  
  
#### Parameters  
 `pDC`  
 A pointer to a device context for painting.
