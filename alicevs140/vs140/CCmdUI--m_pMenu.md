# CCmdUI::m_pMenu
Pointer (of `CMenu` type) to the menu represented by the `CCmdUI` object.  
  
## Syntax  
  
```  
  
CMenu* m_pMenu;  
  
```  
  
## Remarks  
 **NULL** if the item is not a menu.  
  
## Requirements  
 **Header:** afxwin.h
