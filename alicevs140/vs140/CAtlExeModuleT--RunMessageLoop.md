# CAtlExeModuleT::RunMessageLoop
This method executes the message loop.  
  
## Syntax  
  
```  
  
void RunMessageLoop( ) throw( );  
  
```  
  
## Remarks  
 This method can be overridden to change the behavior of the message loop.
