# hypotf Function
Computes the square root of the sum of the squares of _X and _Y  
  
## Syntax  
  
```  
inline float hypotf(  
   float _X,  
   float _Y  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
 `_Y`  
 Floating-point value  
  
## Return Value  
 Returns the square root of the sum of the squares of _X and _Y  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::precise_math
