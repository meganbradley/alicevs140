# CWnd::IsIconic
Specifies whether `CWnd` is minimized (iconic).  
  
## Syntax  
  
```  
  
BOOL IsIconic( ) const;  
```  
  
## Return Value  
 Nonzero if `CWnd` is minimized; otherwise 0.
