# .LISTMACRO
Starts listing of macro expansion statements that generate code or data.  
  
## Syntax  
  
```  
  
.LISTMACRO  
  
```  
  
## Remarks  
 This is the default. Same as [.XALL](../vs140/.XALL.md).
