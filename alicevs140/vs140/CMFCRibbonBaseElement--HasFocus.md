# CMFCRibbonBaseElement::HasFocus
Indicates whether the parent element has keyboard focus.  
  
## Syntax  
  
```  
virtual BOOL HasFocus() const;  
```  
  
## Return Value  
 `TRUE` if the ribbon element is focused; otherwise `FALSE`.
