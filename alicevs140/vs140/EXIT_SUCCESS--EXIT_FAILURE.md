# EXIT_SUCCESS, EXIT_FAILURE
## Syntax  
  
```  
  
#include <stdlib.h>  
```  
  
## Remarks  
 These are arguments for the **exit** and `_exit` functions and the return values for the `atexit` and `_onexit` functions.
