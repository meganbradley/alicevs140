# CAccessToken::GetHandle
Call this method to retrieve a handle to the access token.  
  
## Syntax  
  
```  
  
HANDLE GetHandle( ) const throw( );  
  
```  
  
## Return Value  
 Returns a handle to the `CAccessToken` object's access token.  
  
## Requirements  
 **Header:** atlsecurity.h
