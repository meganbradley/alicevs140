# CMDIChildWndEx::OnUpdateFrameTitle
Called by the framework to update the frame title.  
  
## Syntax  
  
```  
virtual void OnUpdateFrameTitle(  
   BOOL bAddToTitle  
);  
```  
  
#### Parameters  
 [in] `bAddToTitle`  
 If `TRUE`, add the document name to the title.  
  
## Requirements  
 **Header:** afxmdichildwndex.h
