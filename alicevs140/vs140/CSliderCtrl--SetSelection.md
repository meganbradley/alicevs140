# CSliderCtrl::SetSelection
Sets the starting and ending positions for the current selection in a slider control.  
  
## Syntax  
  
```  
  
      void SetSelection(  
   int nMin,  
   int nMax   
);  
```  
  
#### Parameters  
 `nMin`  
 Starting position for the slider.  
  
 `nMax`  
 Ending position for the slider.  
  
## Requirements  
 **Header:** afxcmn.h
