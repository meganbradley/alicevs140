# #endregion (C# Reference)
`#endregion` marks the end of a [#region](../vs140/#region--C#-Reference-.md) block. For example:  
  
```  
  
      #region MyClass definition  
class MyClass   
{  
    static void Main()   
    {  
    }  
}  
#endregion  
```
