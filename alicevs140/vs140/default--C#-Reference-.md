# default (C# Reference)
The `default` keyword.  
  
 The `default` keyword can be used in the `switch` statement or in generic code:  
  
-   [The switch statement](../vs140/switch--C#-Reference-.md): Specifies the default label.  
  
-   [Generic code](../Topic/default%20Keyword%20in%20Generic%20Code%20\(C%23%20Programming%20Guide\).md): Specifies the default value of the type parameter. This will be null for reference types and zero for value types.
