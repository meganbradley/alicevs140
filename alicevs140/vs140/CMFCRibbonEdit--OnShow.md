# CMFCRibbonEdit::OnShow
Called by the framework to show or hide the [CMFCRibbonEdit](../vs140/CMFCRibbonEdit-Class.md) control.  
  
## Syntax  
  
```  
virtual void OnShow(  
   BOOL bShow  
);  
```  
  
#### Parameters  
 [in] `bShow`  
 `TRUE` to show the control; `FALSE` to hide the control.  
  
## Remarks  
  
## Requirements  
 **Header:** afxribbonedit.h
