# COlePropertyPage::SetModifiedFlag
Indicates whether the user has modified the property page.  
  
## Syntax  
  
```  
  
      void SetModifiedFlag(  
   BOOL bModified = TRUE   
);  
```  
  
#### Parameters  
 `bModified`  
 Specifies the new value for the property page's modified flag.  
  
## Requirements  
 **Header:** afxctl.h
