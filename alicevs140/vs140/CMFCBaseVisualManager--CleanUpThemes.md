# CMFCBaseVisualManager::CleanUpThemes
Calls `CloseThemeData` for all handles obtained in `UpdateSystemColors`.  
  
## Syntax  
  
```  
void CleanUpThemes();  
```  
  
## Remarks  
 For internal use only.
