# CAxWindow2T::operator =
Assigns an `HWND` to an existing `CAxWindow2T` object.  
  
## Syntax  
  
```  
  
      CAxWindow2T<   
      TBase  
       >& operator= (  
   HWND hWnd   
);  
```  
  
#### Parameters  
 `hWnd`  
 A handle of an existing window.  
  
## Requirements  
 **Header:** atlwin.h
