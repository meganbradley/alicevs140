# CStringData::nDataLength
Current length of the string object.  
  
## Syntax  
  
```  
  
int nDataLength;  
  
```  
  
## Remarks  
 Stores the length of currently used data in `XCHAR`s (not including terminating null).  
  
## Requirements  
 **Header:** atlsimpstr.h
