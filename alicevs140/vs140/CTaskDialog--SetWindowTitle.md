# CTaskDialog::SetWindowTitle
Sets the title of the `CTaskDialog`.  
  
## Syntax  
  
```  
void SetWindowTitle(  
   CString& strWindowTitle  
);  
```  
  
#### Parameters  
 [in] `strWindowTitle`  
 The new title for the `CTaskDialog`.
