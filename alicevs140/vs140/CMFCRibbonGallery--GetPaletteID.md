# CMFCRibbonGallery::GetPaletteID
Returns the command ID of the current palette.  
  
## Syntax  
  
```  
int GetPaletteID() const;  
```  
  
## Return Value  
 The command ID of the currently selected palette.

## Requirements  
 **Header:** afxRibbonPaletteGallery.h
