# END_DHTML_EVENT_MAP_INLINE
Marks the end of the DHTML event map.  
  
## Syntax  
  
```  
  
END_DHTML_EVENT_MAP_INLINE( )  
  
```  
  
## Remarks  
 Must be used in conjunction with [BEGIN_DHTML_EVENT_MAP_INLINE](../vs140/BEGIN_DHTML_EVENT_MAP_INLINE.md).  
  
## Requirements  
 **Header:** afxdhtml.h
