# CHtmlView::SetHeight
Call this member function to set the height of the Internet Explorer main window.  
  
## Syntax  
  
```  
  
      void SetHeight(  
   long nNewValue   
);  
```  
  
#### Parameters  
 `nNewValue`  
 The height, in pixels, of the main window.  
  
## Remarks  
 Applies to Internet Explorer and WebBrowser.  
  
## Requirements  
 **Header:** afxhtml.h
