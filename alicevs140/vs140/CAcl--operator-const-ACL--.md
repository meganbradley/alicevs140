# CAcl::operator const ACL *
Casts a `CAcl` object to an **ACL** (access-control list) structure.  
  
## Syntax  
  
```  
  
operator const ACL *( ) const throw(...);  
  
```  
  
## Remarks  
 Returns the address of the **ACL** structure.  
  
## Requirements  
 **Header:** atlsecurity.h
