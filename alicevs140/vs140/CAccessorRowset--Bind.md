# CAccessorRowset::Bind
Creates the bindings if you specified **bBind** as false in [CCommand::Open](../vs140/CCommand--Open.md).  
  
## Syntax  
  
```  
  
HRESULT Bind( );  
  
```  
  
## Return Value  
 A standard `HRESULT`.  
  
## Requirements  
 **Header:** atldbcli.h
