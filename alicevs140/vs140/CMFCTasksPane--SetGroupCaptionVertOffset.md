# CMFCTasksPane::SetGroupCaptionVertOffset
Sets the vertical offset of a group caption.  
  
## Syntax  
  
```  
void SetGroupCaptionVertOffset(  
    int n = -1  
);  
```  
  
#### Parameters  
 [in] `n`  
 Specifies the vertical offset, in pixels, of the group caption.  
  
## Requirements  
 **Header:** afxTasksPane.h
