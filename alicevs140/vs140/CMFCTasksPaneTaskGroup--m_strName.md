# CMFCTasksPaneTaskGroup::m_strName
Specifies the name of the group.  
  
## Syntax  
  
```  
CString m_strName;  
```  
  
## Remarks  
 If this value is empty, the group caption is not displayed and the group cannot be collapsed.  
  
## Requirements  
 **Header:** afxTasksPane.h
