# CAtlMap::IsEmpty
Call this method to test for an empty map object.  
  
## Syntax  
  
```  
  
bool IsEmpty( ) const throw( );  
  
```  
  
## Return Value  
 Returns **true** if the map is empty, **false** otherwise.  
  
## Requirements  
 **Header:** atlcoll.h
