# CFileDialog::SetEditBoxText
Sets the current text in an edit box control.  
  
## Syntax  
  
```  
HRESULT SetEditBoxText(  
   DWORD dwIDCtl,  
   const CString& strText  
);  
```  
  
#### Parameters  
 `dwIDCtl`  
 The ID of the edit box.  
  
 `strText`  
 The text value.
