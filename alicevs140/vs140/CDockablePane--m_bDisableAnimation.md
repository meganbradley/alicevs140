# CDockablePane::m_bDisableAnimation
Specifies whether autohide animation of the dockable pane is disabled.  
  
## Syntax  
  
```  
AFX_IMPORT_DATA static BOOL m_bDisableAnimation;  
```  
  
## Requirements  
 **Header:** afxDockablePane.h
