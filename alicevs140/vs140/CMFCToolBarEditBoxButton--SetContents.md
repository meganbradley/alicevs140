# CMFCToolBarEditBoxButton::SetContents
Sets the text in the text box control.  
  
## Syntax  
  
```  
virtual void SetContents(  
   const CString& sContents   
);  
```  
  
#### Parameters  
 `[in] sContents`  
 Specifies the new text to set.  
  
## Requirements  
 **Header:** afxtoolbareditboxbutton.h
