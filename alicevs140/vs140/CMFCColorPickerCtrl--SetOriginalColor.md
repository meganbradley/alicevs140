# CMFCColorPickerCtrl::SetOriginalColor
Sets the initial selected color.  
  
## Syntax  
  
```  
void SetOriginalColor(  
   COLORREF ref   
);  
```  
  
#### Parameters  
 [in] `ref`  
 An RGB color value.  
  
## Remarks  
 Call this method when the color picker control is initialized.  
  
## Requirements  
 **Header:** afxcolorpickerctrl.h
