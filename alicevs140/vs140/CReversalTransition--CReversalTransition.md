# CReversalTransition::CReversalTransition
Constructs a reversal transition object and initializes its duration.  
  
## Syntax  
  
```  
CReversalTransition(  
   UI_ANIMATION_SECONDS duration  
);  
```  
  
#### Parameters  
 `duration`  
 The duration of the transition.  
  
## Requirements  
 **Header:** afxanimationcontroller.h
