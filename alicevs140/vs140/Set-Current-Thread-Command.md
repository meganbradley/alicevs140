# Set Current Thread Command
Sets the specified thread as the current thread.  
  
## Syntax  
  
```  
Debug.SetCurrentThread index  
```  
  
## Arguments  
 `index`  
 Required. Selects a thread by its index.  
  
## Example  
  
```  
>Debug.SetCurrentThread 1  
```
