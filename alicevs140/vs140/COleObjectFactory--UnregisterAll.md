# COleObjectFactory::UnregisterAll
Unregisters all of an application's object factories.  
  
## Syntax  
  
```  
  
static BOOL PASCAL UnregisterAll( );  
  
```  
  
## Return Value  
 TRUE if successful; otherwise FALSE.  
  
## Requirements  
 **Header:** afxdisp.h
