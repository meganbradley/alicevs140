# CUrl::GetHostNameLength
Call this method to get the length of the host name.  
  
## Syntax  
  
```  
  
inline DWORD GetHostNameLength( ) const throw( );  
  
```  
  
## Return Value  
 Returns the host name length.  
  
## Requirements  
 **Header:** atlutil.h
