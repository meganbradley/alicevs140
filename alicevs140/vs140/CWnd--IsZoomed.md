# CWnd::IsZoomed
Determines whether `CWnd` has been maximized.  
  
## Syntax  
  
```  
  
BOOL IsZoomed( ) const;  
```  
  
## Return Value  
 Nonzero if `CWnd` is maximized; otherwise 0.  
  
## Requirements  
 **Header:** afxwin.h
