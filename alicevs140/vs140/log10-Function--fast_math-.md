# log10 Function (fast_math)
Calculates the base-10 logarithm of the argument  
  
## Syntax  
  
```  
inline float log10(  
   float _X  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
## Return Value  
 Returns the base-10 logarithm of the argument  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::fast_math
