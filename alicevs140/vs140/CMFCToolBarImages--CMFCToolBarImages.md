# CMFCToolBarImages::CMFCToolBarImages
Constructs a `CMFCToolBarImages` object.  
  
## Syntax  
  
```  
CMFCToolBarImages();  
```  
  
## Remarks  
 Constructs a `CMFCToolBarImages` object, initializes its rendering engine and sets the image size to its default value 16x15 pixels. Use [CMFCToolBarImages::SetImageSize](../vs140/CMFCToolBarImages--SetImageSize.md) to change the image size before you add images.  
  
## Requirements  
 **Header:** afxtoolbarimages.h
