# CGdiObject::CGdiObject
Constructs a `CGdiObject` object.  
  
## Syntax  
  
```  
  
CGdiObject( );  
```  
  
## Remarks  
 You never create a `CGdiObject` directly. Rather, you create an object from one of its derived classes, such as `CPen` or **Cbrush**.  
  
## Requirements  
 **Header:** afxwin.h
