# CWnd::IsD2DSupportEnabled
Determines whether D2D support is enabled.  
  
## Syntax  
  
```  
BOOL IsD2DSupportEnabled();  
```  
  
## Return Value  
 TRUE if the feature is enabled; otherwise FALSE.  
  
## Requirements  
 **Header:** afxwin.h
