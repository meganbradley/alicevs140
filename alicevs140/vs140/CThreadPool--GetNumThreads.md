# CThreadPool::GetNumThreads
Call this method to get the number of threads in the pool.  
  
## Syntax  
  
```  
  
int GetNumThreads( ) throw( );  
  
```  
  
## Return Value  
 Returns the number of threads in the pool.  
  
## Requirements  
 **Header:** atlutil.h
