# CFrameWnd::SetActivePreviewView
Designates the specified view to be the active view for Rich Preview.  
  
## Syntax  
  
```  
void SetActivePreviewView(  
   CView* pViewNew  
);  
```  
  
#### Parameters  
 `pViewNew`  
 A pointer to a view to be activated.
