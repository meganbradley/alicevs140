# CCriticalSection::operator CRITICAL_SECTION*
Retrieves a **CRITICAL_SECTION** object.  
  
## Syntax  
  
```  
  
operator CRITICAL_SECTION*( );  
  
```  
  
## Remarks  
 Call this function to retrieve a pointer to the internal **CRITICAL_SECTION** object.  
  
## Requirements  
 **Header:** afxmt.h
