# CUserTool::GetCommandId
Returns the command ID of the user tool.  
  
## Syntax  
  
```  
UINT GetCommandId() const;  
```  
  
## Return Value  
 The command ID of this user tool.  
  
## Requirements  
 **Header:** afxusertool.h
