# CComCoClass::GetObjectCLSID
Provides a consistent way of retrieving the object's CLSID.  
  
## Syntax  
  
```  
  
static const CLSID& WINAPI GetObjectCLSID( );  
  
```  
  
## Return Value  
 The object's class identifier.  
  
## Requirements  
 **Header:** atlcom.h
