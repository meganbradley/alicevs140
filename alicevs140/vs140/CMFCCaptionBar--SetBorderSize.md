# CMFCCaptionBar::SetBorderSize
Sets the border size of the caption bar.  
  
## Syntax  
  
```  
void SetBorderSize(  
   int nSize   
);  
```  
  
#### Parameters  
 [in] `nSize`  
 The new size, in pixels, of the caption bar border.  
  
## Requirements  
 **Header:** afxcaptionbar.h
