# CMFCToolBar::UpdateButton
Updates the state of the specified button.  
  
## Syntax  
  
```  
void UpdateButton(  
   int nIndex  
);  
```  
  
#### Parameters  
 [in] `nIndex`  
 Specifies the zero-based index of the button to update.
