# message::msg_id Method
Returns the ID of the `message` object.  
  
## Syntax  
  
```  
runtime_object_identity msg_id() const;  
```  
  
## Return Value  
 The `runtime_object_identity` of the `message` object.

 **Namespace:** concurrency
