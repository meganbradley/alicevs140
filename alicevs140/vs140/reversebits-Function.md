# reversebits Function
Reverses the order of the bits in _X  
  
## Syntax  
  
```  
inline unsigned int reversebits(  
   unsigned int _X  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Unsigned integer value  
  
## Return Value  
 Returns the value with the bit order reversed in _X  
  
## Requirements  
 **Header:** amp.h  
  
 **Namespace:** Concurrency::direct3d
