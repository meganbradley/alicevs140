# CComFakeCriticalSection::Term
Does nothing since there is no critical section.  
  
## Syntax  
  
```  
  
HRESULT Term( ) throw( );  
  
```  
  
## Return Value  
 Returns S_OK.  
  
## Requirements  
 **Header:** atlcore.h
