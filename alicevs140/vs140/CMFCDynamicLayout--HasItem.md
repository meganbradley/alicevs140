# CMFCDynamicLayout::HasItem
Checks if a child control was added to dynamic layout.  
  
## Syntax  
  
```  
  
  BOOL HasItem(  
HWND hwnd);  
```  
  
#### Parameters  
 `hwnd`  
 The window handle for the control.  
  
## Return Value  
 TRUE if layout already has this item; otherwise FALSE.  
  
## Remarks  
  
## Requirements  
 **Header:** afxlayout.h
