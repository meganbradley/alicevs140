# CMFCImageEditorPaletteBar::GetRowHeight
Returns the height of toolbar buttons.  
  
## Syntax  
  
```  
virtual int GetRowHeight() const;  
```  
  
## Return Value  
 The height of each button on the toolbar.  
  
## Requirements  
 **Header:** afximageeditordialog.h
