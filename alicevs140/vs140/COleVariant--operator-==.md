# COleVariant::operator ==
This operator compares two variant values and returns nonzero if they are equal; otherwise 0.  
  
## Syntax  
  
```  
  
      BOOL operator ==(  
   const VARIANT& varSrc   
) const;  
BOOL operator ==(  
   LPCVARIANT pSrc   
) const;  
```  
  
## Requirements  
 **Header:** afxdisp.h
