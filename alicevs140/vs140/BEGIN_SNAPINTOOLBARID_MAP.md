# BEGIN_SNAPINTOOLBARID_MAP
Declares the beginning of the toolbar ID map for the Snap-In object.  
  
## Syntax  
  
```  
  
BEGIN_SNAPINTOOLBARID_MAP( _class )  
```  
  
#### Parameters  
 `_class`  
 [in] Specifies the Snap-In object class.
