# CFrameWnd::rectDefault
Pass this static `CRect` as a parameter when creating a window to allow Windows to choose the window's initial size and position.  
  
## Syntax  
  
```  
  
static AFX_DATA const CRect rectDefault;  
  
```  
  
## Requirements  
 **Header:** afxwin.h
