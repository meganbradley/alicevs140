# END_DISPATCH_MAP
Ends the definition of your dispatch map.  
  
## Syntax  
  
```  
  
END_DISPATCH_MAP( )  
```  
  
## Remarks  
 It must be used in conjunction with `BEGIN_DISPATCH_MAP`.  
  
## Requirements  
 **Header:** afxdisp.h
