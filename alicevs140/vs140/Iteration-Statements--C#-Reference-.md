# Iteration Statements (C# Reference)
You can create loops by using the iteration statements. Iteration statements cause embedded statements to be executed a number of times, subject to the loop-termination criteria. These statements are executed in order, except when a [jump statement](../vs140/Jump-Statements--C#-Reference-.md) is encountered.  
  
 The following keywords are used in iteration statements:  
  
-   [do](../vs140/do--C#-Reference-.md)  
  
-   [for](../vs140/for--C#-Reference-.md)  
  
-   [foreach](../Topic/foreach,%20in%20\(C%23%20Reference\).md)  
  
-   [in](../Topic/foreach,%20in%20\(C%23%20Reference\).md)  
  
-   [while](../vs140/while--C#-Reference-.md)
