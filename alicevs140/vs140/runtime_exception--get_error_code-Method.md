# runtime_exception::get_error_code Method
Returns the error code that caused the exception.  
  
## Syntax  
  
```  
HRESULT get_error_code() const throw();  
```  
  
## Return Value  
 The HRESULT of error that caused the exception.  
  
## Requirements  
 **Header:** amprt.h  
  
 **Namespace:** Concurrency
