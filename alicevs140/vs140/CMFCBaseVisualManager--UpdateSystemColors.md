# CMFCBaseVisualManager::UpdateSystemColors
Calls `OpenThemeData` to obtain handles for drawing various controls: windows, toolbars, buttons, and so on.  
  
## Syntax  
  
```  
void UpdateSystemColors();  
```  
  
## Remarks  
 For internal use only.
