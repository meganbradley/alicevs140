# CJumpList::ClearAll
Removes all tasks and destinations that have been added to the current instance of `CJumpList` so far.  
  
## Syntax  
  
```  
void ClearAll();  
```  
  
## Remarks  
 This method clears and releases all data and internal interfaces.  
  
## Requirements  
 **Header:** afxadv.h
