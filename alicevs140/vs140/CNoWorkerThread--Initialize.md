# CNoWorkerThread::Initialize
Non-functional equivalent of [CWorkerThread::Initialize](../vs140/CWorkerThread--Initialize.md).  
  
## Syntax  
  
```  
  
HRESULT Initialize( ) throw( );  
  
```  
  
## Return Value  
 Always returns S_OK.  
  
## Remarks  
 The implementation provided by this class does nothing.  
  
## Requirements  
 **Header:** atlutil.h
