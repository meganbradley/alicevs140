# STRUCT (MASM)
Declares a structure type having the specified *fielddeclarations*. Each field must be a valid data definition. Same as [STRUC](../vs140/STRUC.md).  
  
## Syntax  
  
```  
  
   name STRUCT [[alignment]] [[, NONUNIQUE]]  
fielddeclarations  
name ENDS  
```
