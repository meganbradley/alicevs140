# Module::GetModule Method
Creates an instance of a module.  
  
## Syntax  
  
```  
static Module& GetModule();  
WRL_NOTHROW static Module& GetModule();  
```  
  
## Return Value  
 A reference to a module.  
  
## Requirements  
 **Header:** module.h  
  
 **Namespace:** Microsoft::WRL
