# CMFCRibbonComboBox::GetCount
Returns the number of items in the list box.  
  
## Syntax  
  
```  
INT_PTR GetCount() const;  
```  
  
## Return Value  
 The number of items in the list box, or 0 if the list box contains no items.

## Requirements  
 **Header:** afxribboncombobox.h
