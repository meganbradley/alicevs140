# CMFCRibbonLinkCtrl::SetLink
Sets the value of the hyperlink.  
  
## Syntax  
  
```  
void SetLink(  
    LPCTSTR lpszLink   
);  
```  
  
#### Parameters  
 [in] `lpszLink`  
 Specifies the hyperlink text.  
  
## Requirements  
 **Header:** afxRibbonLinkCtrl.h
