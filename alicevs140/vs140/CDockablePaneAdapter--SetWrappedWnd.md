# CDockablePaneAdapter::SetWrappedWnd
Sets the underlying window for the dockable pane adapter.  
  
## Syntax  
  
```  
virtual BOOL SetWrappedWnd(  
   CWnd* pWnd  
);  
```  
  
#### Parameters  
 [in] `pWnd`  
 A pointer to the window for the pane adapter to wrap.
