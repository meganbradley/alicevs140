# CMFCToolBar::GetResourceID
Retrieves the resource ID of the toolbar.  
  
## Syntax  
  
```  
UINT GetResourceID() const;  
```  
  
## Return Value  
 The resource ID of the toolbar.  
  
## Remarks  
 Call the [CMFCToolBar::LoadToolBarEx](../vs140/CMFCToolBar--LoadToolBarEx.md) method to set the resource ID of the toolbar.  
  
## Requirements  
 **Header:** afxtoolbar.h
