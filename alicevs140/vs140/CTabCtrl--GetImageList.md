# CTabCtrl::GetImageList
Retrieves the image list that's associated with a tab control.  
  
## Syntax  
  
```  
  
CImageList* GetImageList( ) const;  
  
```  
  
## Return Value  
 If successful, a pointer to the image list of the tab control; otherwise, **NULL**.  
  
## Requirements  
 **Header:** afxcmn.h
