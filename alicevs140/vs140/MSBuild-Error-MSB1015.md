# MSBuild Error MSB1015
**MSBuild does not run on this version of the operating system. It is only supported on Windows 2000, Windows XP, and later versions.**  
  
 [!INCLUDE[vstecmsbuild](../vs140/includes/vstecmsbuild_md.md)] is not supported on this version of the operating system.  
  
### To correct this error  
  
1.  Install and run [!INCLUDE[vstecmsbuild](../vs140/includes/vstecmsbuild_md.md)] on one of the supported versions of the operating system.
