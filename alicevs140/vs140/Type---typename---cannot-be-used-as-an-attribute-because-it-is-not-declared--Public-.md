# Type &#39;&lt;typename&gt;&#39; cannot be used as an attribute because it is not declared &#39;Public&#39;
Attributes must be public.  
  
 **Error ID:** BC31516  
  
### To correct this error  
  
1.  Add the `Public` modifier to the attribute class.
