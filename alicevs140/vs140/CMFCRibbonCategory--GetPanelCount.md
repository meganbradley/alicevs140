# CMFCRibbonCategory::GetPanelCount
Returns the number of ribbon panels in the ribbon category.  
  
## Syntax  
  
```  
int GetPanelCount() const;  
```  
  
## Return Value  
 The number of ribbon panels in the ribbon category.  
  
## Requirements  
 **Header:** afxribboncategory.h
