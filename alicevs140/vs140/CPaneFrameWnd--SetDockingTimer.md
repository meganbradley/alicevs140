# CPaneFrameWnd::SetDockingTimer
Sets the docking timer.  
  
## Syntax  
  
```  
void SetDockingTimer(  
   UINT nTimeOut  
);  
```  
  
#### Parameters  
 [in] `nTimeOut`  
 Timeout value in milliseconds.  
  
## Requirements  
 **Header:** afxpaneframewnd.h
