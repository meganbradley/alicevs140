# CWnd::GetOleControlSite
Retrieves the custom site for the specified ActiveX control.  
  
## Syntax  
  
```  
  
      COleControlSite* GetOleControlSite(  
   UINT idControl  
) const;  
```  
  
#### Parameters  
 `idControl`  
 The ID of the ActiveX control.  
  
## Requirements  
 **Header:** afxwin.h
