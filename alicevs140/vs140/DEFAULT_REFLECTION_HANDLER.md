# DEFAULT_REFLECTION_HANDLER
Provides a default handler for the child window (control) that will receive reflected messages; the handler will properly pass unhandled messages to `DefWindowProc`.  
  
## Syntax  
  
```  
  
DEFAULT_REFLECTION_HANDLER( )  
  
```  
  
## Requirements  
 **Header:** atlwin.h
