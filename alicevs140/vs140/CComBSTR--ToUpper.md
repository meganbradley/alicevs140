# CComBSTR::ToUpper
Converts the contained string to uppercase.  
  
## Syntax  
  
```  
  
HRESULT ToUpper( ) throw( );  
  
```  
  
## Return Value  
 A standard `HRESULT` value.  
  
## Remarks  
 See **CharUpperBuff** for more information on how the conversion is performed.  
  
## Requirements  
 **Header:** atlbase.h
