##  <a name="swap"></a>  swap  
 Exchanges the elements of two deques.  
  
```  
void swap(  
   deque<Type, Allocator>& _Left,  
   deque<Type, Allocator>& _Right,  
);  
```  
  
### Parameters  
 `_Left`  
 An object of type `deque`.  
  
 `_Right`  
 An object of type `deque`.  
  
### Example  
 See the example for [deque::swap](../vs140/deque-Class.md#deque__swap).
