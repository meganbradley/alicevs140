# CMFCColorBar::SetColor
Sets the color that is currently selected.  
  
## Syntax  
  
```  
void SetColor(  
   COLORREF color   
);  
```  
  
#### Parameters  
 [in] `color`  
 An RGB color value.  
  
## Requirements  
 **Header:** afxcolorbar.h
