# CMFCTasksPane::GetGroupCaptionHeight
Returns the height of the group caption.  
  
## Syntax  
  
```  
int GetGroupCaptionHeight() const;  
```  
  
## Return Value  
 The height of the group caption, in pixels.  
  
## Requirements  
 **Header:** afxTasksPane.h
