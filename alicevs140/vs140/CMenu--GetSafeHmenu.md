# CMenu::GetSafeHmenu
Returns the `HMENU` wrapped by this `CMenu` object, or a **NULL** `CMenu` pointer.  
  
## Syntax  
  
```  
  
HMENU GetSafeHmenu( ) const;  
```  
  
## Example  
 See the example for [CMenu::LoadMenu](../vs140/CMenu--LoadMenu.md).  
  
## Requirements  
 **Header:** afxwin.h
