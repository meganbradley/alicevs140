# CWindow::operator =
Assigns an `HWND` to the `CWindow` object by setting the [m_hWnd](../vs140/CWindow--m_hWnd.md) member to `hWnd`.  
  
## Syntax  
  
```  
  
      CWindow& operator =(  
   HWND hWnd   
) throw();  
```  
  
## Requirements  
 **Header:** atlwin.h
