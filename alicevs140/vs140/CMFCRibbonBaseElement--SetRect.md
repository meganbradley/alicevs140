# CMFCRibbonBaseElement::SetRect
Sets the dimensions of the display rectangle for the ribbon element.  
  
## Syntax  
  
```  
void SetRect(  
   CRect rect  
);  
```  
  
#### Parameters  
 [in] `rect`  
 The dimensions of the rectangle.
