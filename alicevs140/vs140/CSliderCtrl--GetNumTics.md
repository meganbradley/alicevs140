# CSliderCtrl::GetNumTics
Retrieves the number of tick marks in a slider control.  
  
## Syntax  
  
```  
  
UINT GetNumTics( ) const;  
```  
  
## Return Value  
 The number of tick marks in the slider control.  
  
## Requirements  
 **Header:** afxcmn.h
