# CRenderTarget::GetDpi
Returns the render target's dots per inch (DPI)  
  
## Syntax  
  
```  
CD2DSizeF GetDpi() const;  
```  
  
## Return Value  
 The render target's dots per inch (DPI).  
  
## Requirements  
 **Header:** afxrendertarget.h
