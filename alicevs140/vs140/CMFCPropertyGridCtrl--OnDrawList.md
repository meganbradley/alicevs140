# CMFCPropertyGridCtrl::OnDrawList
Called by the framework to display the list of properties in the property grid control.  
  
## Syntax  
  
```  
virtual void OnDrawList(  
   CDC* pDC   
);  
```  
  
#### Parameters  
 [in] `pDC`  
 A pointer to a device context.
