# CMFCColorButton::SetPalette
Specifies the standard colors to display on the popup color bar.  
  
## Syntax  
  
```  
void SetPalette(  
   CPalette* pPalette   
);  
```  
  
#### Parameters  
 [in] `pPalette`  
 A pointer to a color palette.
