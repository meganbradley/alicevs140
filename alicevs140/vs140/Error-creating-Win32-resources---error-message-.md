# Error creating Win32 resources: &lt;error message&gt;
The [!INCLUDE[vbprvb](../vs140/includes/vbprvb_md.md)] compiler calls the Assembly Linker (Al.exe, also known as Alink) to generate an assembly with a manifest. The linker has reported an error creating an in-memory resource. This might be a problem with the environment, or your computer might be low on memory.  
  
 **Error ID:** BC30136  
  
### To correct this error  
  
1.  Examine the quoted error message and consult the topic [Al.exe Tool Errors and Warnings](assetId:///7f125d49-0a03-47a6-9ba9-d61a679a7d4b) for further explanation and advice.  
  
2.  If the error persists, gather information about the circumstances and notify Microsoft Product Support Services.
