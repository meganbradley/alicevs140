# CDC::IsPrinting
Determines whether the device context is being used for printing.  
  
## Syntax  
  
```  
  
BOOL IsPrinting( ) const;  
```  
  
## Return Value  
 Nonzero if the `CDC` object is a printer DC; otherwise 0.  
  
## Requirements  
 **Header:** afxwin.h
