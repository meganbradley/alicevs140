# OperatorIntrinsics.PowUIntPtr Function (F#)
This is a library intrinsic. Calls to this function may be generated by uses of the generic [pown](../vs140/Operators.pown-^T--Function--F#-.md) operator on values of type `unativeint`.  
  
 **Namespace/Module Path:** Microsoft.FSharp.Core.Operators.OperatorIntrinsics  
  
 **Assembly:** FSharp.Core (in FSharp.Core.dll)  
  
## Syntax  
  
```  
// Signature:  
PowUIntPtr : unativeint -> int -> unativeint  
  
// Usage:  
PowUIntPtr x n  
```  
  
#### Parameters  
 `x`  
 Type: [unativeint](../vs140/Core.unativeint-Type-Abbreviation--F#-.md)  
  
 The value to raise to a power.  
  
 `n`  
 Type: [int](../vs140/Core.int-Type-Abbreviation--F#-.md)  
  
 The power to raise the value to.  
  
## Return Value  
 The result of the exponentiation operation.  
  
## Remarks  
 This function is for use by compiled F# code and should not be used directly.
