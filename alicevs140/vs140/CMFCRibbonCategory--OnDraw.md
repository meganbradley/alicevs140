# CMFCRibbonCategory::OnDraw
Called by the framework to draw the ribbon category.  
  
## Syntax  
  
```  
virtual void OnDraw(  
   CDC* pDC  
);  
```  
  
#### Parameters  
 [in] `pDC`  
 Pointer to a device context for the ribbon category.

## Requirements  
 **Header:** afxribboncategory.h
