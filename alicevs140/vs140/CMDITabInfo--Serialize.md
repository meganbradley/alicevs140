# CMDITabInfo::Serialize
Reads or writes this object from an archive or to an archive.  
  
## Syntax  
  
```  
void Serialize(  
   CArchive& ar  
);  
```  
  
#### Parameters  
 [in] `ar`  
 A [CArchive Class](../vs140/CArchive-Class.md) object to serialize.
