# CMFCButton::IsRadioButton
Indicates whether a button is a radio button.  
  
## Syntax  
  
```  
BOOL IsRadioButton() const;  
```  
  
## Return Value  
 TRUE if the button style is BS_RADIOBUTTON or BS_AUTORADIOBUTTON; otherwise, FALSE.  
  
## Remarks  
  
## Requirements  
 **Header:** afxbutton.h
