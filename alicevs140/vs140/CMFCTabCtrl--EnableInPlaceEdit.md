# CMFCTabCtrl::EnableInPlaceEdit
Enables or disables editable tab labels.  
  
## Syntax  
  
```  
virtual void EnableInPlaceEdit(  
   BOOL bEnable   
);  
```  
  
#### Parameters  
 [in] `bEnable`  
 `TRUE` to enable editable tab labels; `FALSE` to disable editable tab labels.
