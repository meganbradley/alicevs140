# CMFCPopupMenu::ShowAllCommands
Forces the pop-up menu to display all commands.  
  
## Syntax  
  
```  
void ShowAllCommands();  
```  
  
## Remarks  
 This is not a global setting and affects only the current pop-up menu.  
  
## Requirements  
 **Header:** afxpopupmenu.h
