# CSplitterWnd::OnDraw
Called by the framework to draw the splitter window.  
  
## Syntax  
  
```  
virtual void OnDraw(  
   CDC* pDC  
);  
```  
  
#### Parameters  
 `pDC`  
 A pointer to a device context.
