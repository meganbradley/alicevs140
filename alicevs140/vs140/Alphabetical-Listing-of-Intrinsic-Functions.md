# Alphabetical Listing of Intrinsic Functions
The following sections describe the Microsoft-specific intrinsic functions available on some or all architectures. Other supported intrinsics are documented by processor manufacturers, either in the header files or on their websites. See the [ARM intrinsics](../vs140/ARM-Intrinsics.md), [x86 intrinsics](../vs140/x86-Intrinsics-List.md), and [x64 intrinsics](../vs140/x64--amd64--Intrinsics-List.md) sections for links to manufacturer documentation. C Run-time Library (CRT) functions that have intrinsics implemented for them are not documented here; these functions are documented in the [CRT reference](../vs140/C-Run-Time-Library-Reference.md).
