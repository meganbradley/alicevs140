# CSnapInPropertyPageImpl::OnHelp
This member function is called when the user clicks the **Help** button for the property page.  
  
## Syntax  
  
```  
  
void OnHelp( );  
  
```  
  
## Remarks  
 Override this member function to display help for the property page.  
  
## Requirements  
 **Header:** atlsnap.h
