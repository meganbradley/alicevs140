# CAtlList::GetCount
Call this method to return the number of objects in the list.  
  
## Syntax  
  
```  
  
size_t GetCount( ) const throw( );  
  
```  
  
## Return Value  
 Returns the number of elements in the list.  
  
## Example  
 See the example for [CAtlList::Find](../vs140/CAtlList--Find.md).

 **Header:** atlcoll.h
