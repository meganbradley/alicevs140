# CMFCBaseTabCtrl::GetHighlightedTab
Retrieves the index of the currently highlighted tab.  
  
## Syntax  
  
```  
int GetHighlightedTab() const;  
```  
  
## Return Value  
 The zero-based index of the highlighted tab.  
  
## Requirements  
 **Header:** afxbasetabctrl.h
