# CMFCRibbonCategory::RecalcLayout
Adjusts the layout of all controls on the ribbon category.  
  
## Syntax  
  
```  
virtual void RecalcLayout(  
   CDC* pDC  
);  
```  
  
#### Parameters  
 [in] `pDC`  
 Pointer to a device context for the ribbon category.

## Requirements  
 **Header:** afxribboncategory.h
