# CD2DGeometrySink::CD2DGeometrySink
Constructs a CD2DGeometrySink object from CD2DPathGeometry object.  
  
## Syntax  
  
```  
CD2DGeometrySink(  
   CD2DPathGeometry& pathGeometry  
);  
```  
  
#### Parameters  
 `pathGeometry`  
 An existing CD2DPathGeometry object.  
  
## Requirements  
 **Header:** afxrendertarget.h
