# Compiler Warning (levels 1 and 4) C4223
nonstandard extension used : non-lvalue array converted to pointer  
  
 In standard C, you cannot convert a non-lvalue array to a pointer. With the default Microsoft extensions ([/Ze](../Topic/-Za,%20-Ze%20\(Disable%20Language%20Extensions\).md)), you can.
