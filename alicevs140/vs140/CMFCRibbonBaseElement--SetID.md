# CMFCRibbonBaseElement::SetID
Sets the command ID of the ribbon element.  
  
## Syntax  
  
```  
virtual void SetID(  
   UINT nID   
);  
```  
  
#### Parameters  
 [in] `nID`  
 The command ID.  
  
## Requirements  
 **Header:** afxbaseribbonelement.h
