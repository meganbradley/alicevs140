# AFX_GLOBAL_DATA::UpdateFonts
Reintializes the logical fonts that are used by the framework.  
  
## Syntax  
  
```  
void UpdateFonts();  
```  
  
## Remarks  
 For more information about logical fonts, see [CFont::CreateFontIndirect](../vs140/CFont--CreateFontIndirect.md).  
  
## Requirements  
 **Header:** afxglobals.h
