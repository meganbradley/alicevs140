# log Function
Calculates the base-e logarithm of the argument  
  
## Syntax  
  
```  
inline float log(  
   float _X  
) restrict(amp);  
inline double log(  
   double _X  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
## Return Value  
 Returns the base-e logarithm of the argument  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::precise_math
