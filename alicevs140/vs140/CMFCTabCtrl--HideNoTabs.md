# CMFCTabCtrl::HideNoTabs
Enables or disables drawing of the tab area if there are no visible tabs.  
  
## Syntax  
  
```  
void HideNoTabs(  
   BOOL bHide=TRUE   
);  
```  
  
#### Parameters  
 [in] `bHide`  
 `TRUE` to enable drawing the tab area; `FALSE` to disable drawing. The default value is `TRUE`.
