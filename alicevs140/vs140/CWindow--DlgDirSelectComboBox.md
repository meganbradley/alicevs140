# CWindow::DlgDirSelectComboBox
Retrieves the current selection from a combo box.  
  
## Syntax  
  
```  
  
      BOOL DlgDirSelectComboBox(  
   LPTSTR lpString,  
   int nCount,  
   int nIDComboBox   
) throw();  
```
