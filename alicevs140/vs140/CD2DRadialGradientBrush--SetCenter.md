# CD2DRadialGradientBrush::SetCenter
Specifies the center of the gradient ellipse in the brush's coordinate space  
  
## Syntax  
  
```  
void SetCenter(  
   CD2DPointF point  
);  
```  
  
#### Parameters  
 `point`  
 The center of the gradient ellipse, in the brush's coordinate space  
  
## Requirements  
 **Header:** afxrendertarget.h
