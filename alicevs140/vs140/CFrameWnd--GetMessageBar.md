# CFrameWnd::GetMessageBar
Call this member function to get a pointer to the status bar.  
  
## Syntax  
  
```  
  
virtual CWnd* GetMessageBar( );  
  
```  
  
## Return Value  
 Pointer to the status-bar window.  
  
## Requirements  
 **Header:** afxwin.h
