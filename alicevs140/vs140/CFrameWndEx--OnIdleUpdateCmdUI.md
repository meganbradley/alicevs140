# CFrameWndEx::OnIdleUpdateCmdUI
Called by the framework to update the frame display when command processing is idle.  
  
## Syntax  
  
```  
afx_msg LRESULT OnIdleUpdateCmdUI(  
   WPARAM wParam = 0,  
   LPARAM lParam = 0  
);  
```  
  
#### Parameters  
 [in] `wParam`  
 This parameter is not used.  
  
 [in] `lParam`  
 This parameter is not used.  
  
## Return Value  
 Always returns 0.

## Requirements  
 **Header:** afxframewndex.h
