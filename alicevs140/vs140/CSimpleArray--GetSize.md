# CSimpleArray::GetSize
Returns the number of elements stored in the array.  
  
## Syntax  
  
```  
  
int GetSize( ) const;  
  
```  
  
## Return Value  
 Returns the number of elements stored in the array.

 **Header:** atlsimpcoll.h
