# CMFCRibbonBar::IsKeyTipEnabled
Indicates whether the keytips feature is enabled.  
  
## Syntax  
  
```  
BOOL IsKeyTipEnabled() const;  
```  
  
## Return Value  
 `TRUE` if the keytips feature is enabled; otherwise `FALSE`.  
  
## Requirements  
 **Header:** afxribbonbar.h
