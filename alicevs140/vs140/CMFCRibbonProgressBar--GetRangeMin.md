# CMFCRibbonProgressBar::GetRangeMin
Returns the progress bar's current minimum range value.  
  
## Syntax  
  
```  
int GetRangeMin() const;  
```  
  
## Return Value  
 The minimum value of the current range.  
  
## Requirements  
 **Header:** afxRibbonProgressBar.h
