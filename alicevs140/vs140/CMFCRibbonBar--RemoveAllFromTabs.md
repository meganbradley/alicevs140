# CMFCRibbonBar::RemoveAllFromTabs
Removes all ribbon elements from the tab area.  
  
## Syntax  
  
```  
void RemoveAllFromTabs();  
```  
  
## Remarks  
 Use this function if you want to remove all the elements that you added to the tab area by using [CMFCRibbonBar::AddToTabs](../vs140/CMFCRibbonBar--AddToTabs.md) method.  
  
## Requirements  
 **Header:** afxribbonbar.h
