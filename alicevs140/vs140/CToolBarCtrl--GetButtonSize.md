# CToolBarCtrl::GetButtonSize
Gets the size of a toolbar button.  
  
## Syntax  
  
```  
  
DWORD GetButtonSize( ) const;  
  
```  
  
## Return Value  
 A `DWORD` value that contains the width and height values in the LOWORD and HIWORD, respectively.  
  
## Requirements  
 **Header:** afxcmn.h
