# CMFCPropertyGridProperty::IsSelected
Indicates whether the current property is selected.  
  
## Syntax  
  
```  
virtual BOOL IsSelected() const;  
```  
  
## Return Value  
 `TRUE` if the current property is selected; otherwise, `FALSE`.

## Requirements  
 **Header:** afxpropertygridctrl.h
