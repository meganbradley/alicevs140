# CAnimateCtrl::Stop
Call this function to stop playing an AVI clip in an animation control.  
  
## Syntax  
  
```  
  
BOOL Stop( );  
```  
  
## Return Value  
 Nonzero if successful; otherwise zero.  
  
## Example  
 See the example for [CAnimateCtrl::CAnimateCtrl](../vs140/CAnimateCtrl--CAnimateCtrl.md).
