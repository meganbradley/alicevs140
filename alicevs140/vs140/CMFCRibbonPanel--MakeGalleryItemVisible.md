# CMFCRibbonPanel::MakeGalleryItemVisible
Scrolls the gallery to make the specified Ribbon element visible.  
  
## Syntax  
  
```  
void MakeGalleryItemVisible(  
   CMFCRibbonBaseElement* pItem  
);  
```  
  
#### Parameters  
 `pItem`  
 A pointer to a Ribbon element to show.
