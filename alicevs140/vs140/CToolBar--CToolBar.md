# CToolBar::CToolBar
This member function constructs a `CToolBar` object and sets the default sizes.  
  
## Syntax  
  
```  
  
CToolBar( );  
```  
  
## Remarks  
 Call the [Create](../vs140/CToolBar--Create.md) member function to create the toolbar window.  
  
## Requirements  
 **Header:** afxext.h
