# CMFCRibbonBaseElement::IsSeparator
Indicates whether the ribbon element is a display separator.  
  
## Syntax  
  
```  
virtual BOOL IsSeparator() const;  
```  
  
## Return Value  
 `TRUE` if the ribbon element is a display separator; otherwise `FALSE`.

## Requirements
