# CFrameWnd::SetProgressBarRange
Sets the range for the Windows 7 progress bar displayed on the taskbar.  
  
## Syntax  
  
```  
void SetProgressBarRange(  
   int nRangeMin,  
   int nRangeMax  
);  
```  
  
#### Parameters  
 `nRangeMin`  
 Minimal value.  
  
 `nRangeMax`  
 Maximal value.
