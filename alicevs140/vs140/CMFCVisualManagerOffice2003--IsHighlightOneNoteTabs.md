# CMFCVisualManagerOffice2003::IsHighlightOneNoteTabs
Indicates whether OneNote tabs should be highlighted.  
  
## Syntax  
  
```  
virtual BOOL IsHighlightOneNoteTabs() const;  
```  
  
## Return Value  
 Returns `TRUE`.
