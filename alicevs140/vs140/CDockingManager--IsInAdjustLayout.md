# CDockingManager::IsInAdjustLayout
Determines if the layouts of all panes are adjusted.  
  
## Syntax  
  
```  
BOOL IsInAdjustLayout() const;  
```  
  
## Return Value  
 `TRUE` if the layouts of all panes are adjusted; `FALSE` otherwise.  
  
## Requirements  
 **Header:** afxdockingmanager.h
