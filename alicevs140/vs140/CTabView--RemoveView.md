# CTabView::RemoveView
Removes the view from the tab control.  
  
## Syntax  
  
```  
BOOL RemoveView(  
   int iTabNum   
);  
```  
  
#### Parameters  
 [in] `iTabNum`  
 The index of the view to remove.  
  
## Return Value  
 The index of the removed view if this method succeeds. Otherwise -1.

## Requirements  
 **Header:** afxTabView.h
