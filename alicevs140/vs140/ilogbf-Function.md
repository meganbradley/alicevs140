# ilogbf Function
Extract the exponent of _X as a signed int value  
  
## Syntax  
  
```  
inline int ilogbf(  
   float _X  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
## Return Value  
 Returns the exponent of _X as a signed int value  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::precise_math
