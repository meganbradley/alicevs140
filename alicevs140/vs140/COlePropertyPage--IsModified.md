# COlePropertyPage::IsModified
Determines whether the user has changed any values on the property page.  
  
## Syntax  
  
```  
  
BOOL IsModified( );  
```  
  
## Return Value  
 **TRUE** if the property page has been modified.  
  
## Requirements  
 **Header:** afxctl.h
