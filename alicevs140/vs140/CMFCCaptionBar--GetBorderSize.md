# CMFCCaptionBar::GetBorderSize
Returns the border size of the caption bar.  
  
## Syntax  
  
```  
int GetBorderSize() const;  
```  
  
## Return Value  
 The size, in pixels, of the border.  
  
## Requirements  
 **Header:** afxcaptionbar.h
