# source_link_manager::count Method
Counts the number of linked blocks in the `source_link_manager` object.  
  
## Syntax  
  
```  
size_t count();  
```  
  
## Return Value  
 The number of linked blocks in the `source_link_manager` object.  
  
## Requirements  
 **Header:** agents.h  
  
 **Namespace:** concurrency
