# CHtmlView::Stop
Call this member function to cancel any pending navigation or download operation and stop any dynamic page elements, such as background sounds and animations.  
  
## Syntax  
  
```  
  
void Stop( );  
  
```  
  
## Remarks  
 Applies to Internet Explorer and WebBrowser.  
  
## Requirements  
 **Header:** afxhtml.h
