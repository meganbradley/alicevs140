# CMFCTasksPane::RemovePage
Removes a specified page from the task pane.  
  
## Syntax  
  
```  
void RemovePage(  
    int nPageIdx  
);  
```  
  
#### Parameters  
 [in] `nPageIdx`  
 Specifies the zero-based index of the page to remove.  
  
## Requirements  
 **Header:** afxTasksPane.h
