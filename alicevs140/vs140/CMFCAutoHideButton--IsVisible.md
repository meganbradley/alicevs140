# CMFCAutoHideButton::IsVisible
Indicates whether the auto-hide button is visible.  
  
## Syntax  
  
```  
virtual BOOL IsVisible() const;  
```  
  
## Return Value  
 `TRUE` if the button is visible; `FALSE` otherwise.  
  
## Requirements  
 **Header:** afxautohidebutton.h
