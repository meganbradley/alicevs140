# CMFCStatusBar::GetCount
Retrieves the number of panes in the status bar.  
  
## Syntax  
  
```  
int GetCount() const;  
```  
  
## Return Value  
 The number of panes in the status bar.  
  
## Requirements  
 **Header:** afxstatusbar.h
