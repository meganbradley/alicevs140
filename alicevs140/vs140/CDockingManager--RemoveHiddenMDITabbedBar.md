# CDockingManager::RemoveHiddenMDITabbedBar
Removes the specified hidden bar pane.  
  
## Syntax  
  
```  
void RemoveHiddenMDITabbedBar(  
   CDockablePane* pBar  
);  
```  
  
#### Parameters  
 [in] `pBar`  
 A pointer to a bar pane to remove.  
  
## Requirements  
 **Header:** afxdockingmanager.h
