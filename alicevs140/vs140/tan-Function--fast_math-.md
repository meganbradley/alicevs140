# tan Function (fast_math)
Calculates the tangent value of the argument  
  
## Syntax  
  
```  
inline float tan(  
   float _X  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
## Return Value  
 Returns the tangent value of the argument  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::fast_math
