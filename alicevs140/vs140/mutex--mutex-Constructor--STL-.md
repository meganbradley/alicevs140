# mutex::mutex Constructor (STL)
Constructs a `mutex` object that is not locked.  
  
## Syntax  
  
```cpp  
constexpr mutex() _NOEXCEPT;  
```  
  
## Requirements  
 **Header:** mutex  
  
 **Namespace:** std
