# CMFCMenuBar::GetHMenu
Retrieves the handle to the menu attached to the [CMFCMenuBar](../vs140/CMFCMenuBar-Class.md) object.  
  
## Syntax  
  
```  
HMENU GetHMenu() const;  
```  
  
## Requirements  
 **Header:** afxmenubar.h
