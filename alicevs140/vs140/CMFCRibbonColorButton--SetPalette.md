# CMFCRibbonColorButton::SetPalette
Specifies the standard colors to display in the color table that the color button displays.  
  
## Syntax  
  
```  
void SetPalette(  
   CPalette* pPalette  
);  
```  
  
#### Parameters  
 [in] `pPalette`  
 A pointer to a color palette.
