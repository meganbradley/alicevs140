# CDaoDatabase::IsOpen
Call this member function to determine whether the `CDaoDatabase` object is currently open on a database.  
  
## Syntax  
  
```  
  
BOOL IsOpen( ) const;  
  
```  
  
## Return Value  
 Nonzero if the `CDaoDatabase` object is currently open; otherwise 0.
