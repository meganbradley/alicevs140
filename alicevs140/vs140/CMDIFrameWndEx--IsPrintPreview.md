# CMDIFrameWndEx::IsPrintPreview
Determines whether the frame window is in print-preview mode.  
  
## Syntax  
  
```  
BOOL IsPrintPreview();  
```  
  
## Return Value  
 `TRUE` if the frame window is in print-preview mode; otherwise, `FALSE`.

## Requirements  
 **Header:** afxmdiframewndex.h
