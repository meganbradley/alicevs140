# CMFCRibbonPanel::GetData
Returns the user-defined data associated with the panel.  
  
## Syntax  
  
```  
DWORD_PTR GetData() const;  
```  
  
## Return Value  
 The user-defined data associated with the panel.  
  
## Requirements  
 **Header:** afxRibbonPanel.h
