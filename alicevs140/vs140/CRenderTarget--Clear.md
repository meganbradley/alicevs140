# CRenderTarget::Clear
Clears the drawing area to the specified color.  
  
## Syntax  
  
```  
void Clear(  
   D2D1_COLOR_F color  
);  
```  
  
#### Parameters  
 `color`  
 The color to which the drawing area is cleared.  
  
## Requirements  
 **Header:** afxrendertarget.h
