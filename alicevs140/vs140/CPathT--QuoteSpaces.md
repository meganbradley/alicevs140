# CPathT::QuoteSpaces
Call this method to enclose the path in quotation marks if it contains any spaces.  
  
## Syntax  
  
```  
  
void QuoteSpaces( );  
  
```  
  
## Remarks  
 For more information, see [PathQuoteSpaces](http://msdn.microsoft.com/library/windows/desktop/bb773739).  
  
## Requirements  
 **Header:** atlpath.h
