# CBitmapRenderTarget::Attach
Attaches existing render target interface to the object  
  
## Syntax  
  
```  
void Attach(  
   ID2D1BitmapRenderTarget* pTarget  
);  
```  
  
#### Parameters  
 `pTarget`  
 Existing render target interface. Cannot be NULL  
  
## Requirements  
 **Header:** afxrendertarget.h
