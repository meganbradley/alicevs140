# CPaneFrameWnd::RedrawAll
Redraws all mini-frame windows.  
  
## Syntax  
  
```  
static void RedrawAll();  
```  
  
## Remarks  
 This method updates all mini-frame windows by calling [CWnd::RedrawWindow](../vs140/CWnd--RedrawWindow.md) for each window.  
  
## Requirements  
 **Header:** afxPaneFrameWnd.h
