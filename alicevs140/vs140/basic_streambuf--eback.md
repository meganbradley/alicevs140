# basic_streambuf::eback
A protected function that returns a pointer to the beginning of the input buffer.  
  
## Syntax  
  
```  
  
char  
_  
type *eback( ) const;  
  
```  
  
## Return Value  
 A pointer to the beginning of the input buffer.  
  
## Requirements  
 **Header:** <streambuf\>  
  
 **Namespace:** std
