# CFileTimeSpan::operator =
The assignment operator.  
  
## Syntax  
  
```  
  
      CFileTimeSpan& operator =(  
   const CFileTimeSpan& span   
) throw( );  
```  
  
#### Parameters  
 `span`  
 A `CFileTimeSpan` object.  
  
## Return Value  
 Returns the updated `CFileTimeSpan` object.  
  
## Requirements  
 **Header:** atltime.h
