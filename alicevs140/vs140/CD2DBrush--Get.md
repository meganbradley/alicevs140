# CD2DBrush::Get
Returns ID2D1Brush interface  
  
## Syntax  
  
```  
ID2D1Brush* Get();  
```  
  
## Return Value  
 Pointer to an ID2D1Brush interface or NULL if object is not initialized yet.  
  
## Requirements  
 **Header:** afxrendertarget.h
