# CTabCtrl::SetPadding
Sets the amount of space (padding) around each tab's icon and label in a tab control.  
  
## Syntax  
  
```  
  
      void SetPadding(  
  CSize size   
);  
```  
  
#### Parameters  
 `size`  
 Sets the amount of space (padding) around each tab's icon and label in a tab control.  
  
## Requirements  
 **Header:** afxcmn.h
