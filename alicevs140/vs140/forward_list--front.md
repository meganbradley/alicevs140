# forward_list::front
Returns a reference to the first element in a forward list.  
  
## Syntax  
  
```  
reference front();  
const_reference front() const;  
```  
  
## Return Value  
 A reference to the first element of the controlled sequence, which must be non-empty.  
  
## Requirements  
 **Header:** <forward_list>  
  
 **Namespace:** std
