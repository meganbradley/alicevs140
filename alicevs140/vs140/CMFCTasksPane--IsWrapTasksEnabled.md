# CMFCTasksPane::IsWrapTasksEnabled
Specifies whether the framework wraps the task string.  
  
## Syntax  
  
```  
BOOL IsWrapTasksEnabled() const;  
```  
  
## Return Value  
 `TRUE` if the task string is wrapped; otherwise, `FALSE`.  
  
## Requirements  
 **Header:** afxTasksPane.h
