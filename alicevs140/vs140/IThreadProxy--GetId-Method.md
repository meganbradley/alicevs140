# IThreadProxy::GetId Method
Returns a unique identifier for the thread proxy.  
  
## Syntax  
  
```  
virtual unsigned int GetId() const =0;  
```  
  
## Return Value  
 A unique integer identifier.  
  
## Requirements  
 **Header:** concrtrm.h  
  
 **Namespace:** concurrency
