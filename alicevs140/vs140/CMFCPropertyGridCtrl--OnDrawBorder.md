# CMFCPropertyGridCtrl::OnDrawBorder
Called by the framework to draw a border around a property grid control.  
  
## Syntax  
  
```  
virtual void OnDrawBorder(  
   CDC* pDC   
);  
```  
  
#### Parameters  
 [in] `pDC`  
 A pointer to a device context.

## Requirements
