# CAsyncMonikerFile::Close
Call this function to close and release all resources.  
  
## Syntax  
  
```  
  
virtual void Close( );  
```  
  
## Remarks  
 Can be called on unopened or already closed files.  
  
## Requirements  
 **Header:** afxole.h
