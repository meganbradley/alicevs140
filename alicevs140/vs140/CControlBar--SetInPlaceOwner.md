# CControlBar::SetInPlaceOwner
Changes the in-place owner of a control bar.  
  
## Syntax  
  
```  
  
      void SetInPlaceOwner(  
   CWnd *pWnd  
);  
```  
  
#### Parameters  
 `pWnd`  
 A pointer to a `CWnd` object.
