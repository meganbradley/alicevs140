# COleControl::COleControl
Constructs a `COleControl` object.  
  
## Syntax  
  
```  
  
COleControl( );  
```  
  
## Remarks  
 This function is normally not called directly. Instead the OLE control is usually created by its class factory.  
  
## Requirements  
 **Header:** afxctl.h
