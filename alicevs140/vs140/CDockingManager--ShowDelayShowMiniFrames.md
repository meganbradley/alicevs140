# CDockingManager::ShowDelayShowMiniFrames
Shows or hides the windows of the mini frames.  
  
## Syntax  
  
```  
void ShowDelayShowMiniFrames(  
    BOOL bshow  
);  
```  
  
#### Parameters  
 [in] `bShow`  
 `TRUE` to make the window of the shown frame active; `FALSE to` hide the window of the frame.
