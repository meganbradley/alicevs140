# CPaneFrameWnd::GetPaneCount
Returns the number of panes that are contained in a mini-frame window.  
  
## Syntax  
  
```  
virtual int GetPaneCount() const;  
```  
  
## Return Value  
 The number of panes in the mini-frame window. This value can be zero.
