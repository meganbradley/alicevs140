# diagnostic Namespace
The `diagnostics` namespace provides functionality for emitting Concurrency Visualizer markers.  
  
## Syntax  
  
```  
namespace diagnostic;  
```  
  
## Members  
  
### Classes  
  
|Name|Description|  
|----------|-----------------|  
|[marker_series Class](../vs140/marker_series-Class.md)|Represents a serial channel of events generated by a single provider.|  
|[span Class](../vs140/span-Class.md)|Defines a phase of the application.|  
  
### Enumerations  
  
|Name|Description|  
|----------|-----------------|  
|[marker_importance Enumeration](../vs140/marker_importance-Enumeration.md)|Represents the importance level of a Concurrency Visualizer marker.|  
  
## Requirements  
 **Header:** cvmarkersobj.h  
  
 **Namespace:** Concurrency
