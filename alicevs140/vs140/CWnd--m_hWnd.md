# CWnd::m_hWnd
The handle of the Windows window attached to this `CWnd`.  
  
## Syntax  
  
```  
  
HWND m_hWnd;  
  
```  
  
## Remarks  
 The `m_hWnd` data member is a public variable of type `HWND`.  
  
## Requirements  
 **Header:** afxwin.h
