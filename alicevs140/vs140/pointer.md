# pointer
A synonym for `Del::pointer` if defined, otherwise `Type *`.  
  
## Syntax  
  
```  
typedef T1 pointer;  
```  
  
## Remarks  
 The type is a synonym for `Del::pointer` if defined, otherwise `Type *`.  
  
## Requirements  
 **Header:** <memory\>  
  
 **Namespace:** std
