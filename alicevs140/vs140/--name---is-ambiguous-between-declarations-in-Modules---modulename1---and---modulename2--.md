# &#39;&lt;name&gt;&#39; is ambiguous between declarations in Modules &#39;&lt;modulename1&gt;&#39; and &#39;&lt;modulename2&gt;&#39;
You have provided a name that is ambiguous and therefore conflicts with another name. The [!INCLUDE[vbprvb](../vs140/includes/vbprvb_md.md)] compiler does not have any conflict resolution rules; you must disambiguate names yourself.  
  
 **Error ID:** BC30562  
  
### To correct this error  
  
-   Fully qualify the name.
