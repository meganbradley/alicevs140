# CMFCRibbonCheckBox::CMFCRibbonCheckBox
Constructor of a ribbon check box object  
  
## Syntax  
  
```  
CMFCRibbonCheckBox(  
   UINT nID,  
   LPCTSTR lpszText   
);  
```  
  
#### Parameters  
 [in] `nID`  
 Specifies command ID.  
  
 [in] `lpszText`  
 Specifies text label.  
  
## Return Value  
 Constructs a ribbon check box object.
