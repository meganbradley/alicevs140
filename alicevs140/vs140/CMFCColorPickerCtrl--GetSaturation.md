# CMFCColorPickerCtrl::GetSaturation
Retrieves the saturation value of the color that the user selects.  
  
## Syntax  
  
```  
double GetSaturation() const;  
```  
  
## Return Value  
 The saturation component of the selected color.

## Requirements  
 **Header:** afxcolorpickerctrl.h
