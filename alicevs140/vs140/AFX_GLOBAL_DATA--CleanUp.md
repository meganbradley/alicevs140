# AFX_GLOBAL_DATA::CleanUp
Releases resources that are allocated by the framework, such as brushes, fonts, and DLLs.  
  
## Syntax  
  
```  
void CleanUp();  
```  
  
## Requirements  
 **Header:** afxglobals.h
