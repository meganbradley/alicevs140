# CDockingManager::GetDockSiteFrameWnd
Gets a pointer to the parent window frame.  
  
## Syntax  
  
```  
CFrameWnd* GetDockSiteFrameWnd() const;  
```  
  
## Return Value  
 A pointer to the parent window frame.  
  
## Requirements  
 **Header:** afxdockingmanager.h
