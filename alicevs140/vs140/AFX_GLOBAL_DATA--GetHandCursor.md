# AFX_GLOBAL_DATA::GetHandCursor
Retrieves the predefined cursor that resembles a hand and whose identifier is `IDC_HAND`.  
  
## Syntax  
  
```  
HCURSOR GetHandCursor();  
```  
  
## Return Value  
 The handle of the hand cursor.  
  
## Requirements  
 **Header:** afxglobals.h
