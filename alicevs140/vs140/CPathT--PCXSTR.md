# CPathT::PCXSTR
A constant string type.  
  
## Syntax  
  
```  
  
typedef StringType::PCXSTR PCXSTR;  
  
```  
  
## Remarks  
 `StringType` is the template parameter to `CPathT`.  
  
## Requirements  
 **Header:** atlpath.h
