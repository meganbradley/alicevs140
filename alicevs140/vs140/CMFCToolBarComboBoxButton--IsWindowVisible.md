# CMFCToolBarComboBoxButton::IsWindowVisible
Returns the visibility state of the combo box button.  
  
## Syntax  
  
```  
virtual BOOL IsWindowVisible();  
```  
  
## Return Value  
 The visibility state of the combo box button.

 **Header:** afxtoolbarcomboboxbutton.h
