# CMFCRibbonSeparator::OnDraw
Called by the system to draw the separator on either the ribbon or the Quick Access Toolbar.  
  
## Syntax  
  
```  
virtual void OnDraw(  
   CDC* pDC  
);  
```  
  
#### Parameters  
 [in] `pDC`  
 A pointer to a device context.  
  
## Requirements  
 **Header:** afxbaseribbonelement.h
