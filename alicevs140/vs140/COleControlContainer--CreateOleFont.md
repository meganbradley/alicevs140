# COleControlContainer::CreateOleFont
Creates an OLE font.  
  
## Syntax  
  
```  
  
      void CreateOleFont(  
   CFont* pFont   
);  
```  
  
#### Parameters  
 `pFont`  
 A pointer to the font to be used by the control container.  
  
## Requirements  
 **Header:** afxocc.h
