# sinh Function
Calculates the hyperbolic sine value of the argument  
  
## Syntax  
  
```  
inline float sinh(  
   float _X  
) restrict(amp);  
inline double sinh(  
   double _X  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
## Return Value  
 Returns the hyperbolic sine value of the argument  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::precise_math
