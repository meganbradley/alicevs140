# CDataConnection::operator BOOL
Determines whether the current session is open or not.  
  
## Syntax  
  
```  
  
operator BOOL( ) throw( );  
  
```  
  
## Remarks  
 Returns **BOOL** (MFC typedef) value. **TRUE** means the current session is open; **FALSE** means the current session is closed.  
  
## Requirements  
 **Header:** atldbcli.h
