# CMFCDynamicLayout::GetHostWnd
Returns a pointer to a host window.  
  
## Syntax  
  
```  
  
CWnd* GetHostWnd();  
```  
  
## Return Value  
 A pointer to the host window.  
  
## Remarks  
 By default all child control positions recalculated relative to this window.  
  
## Requirements  
 **Header:** afxwin.h
