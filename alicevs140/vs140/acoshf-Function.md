# acoshf Function
Calculates the inverse hyperbolic cosine of the argument  
  
## Syntax  
  
```  
inline float acoshf(  
   float _X  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
## Return Value  
 Returns the inverse hyperbolic cosine value of the argument  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::precise_math
