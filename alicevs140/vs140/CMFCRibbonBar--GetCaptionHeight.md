# CMFCRibbonBar::GetCaptionHeight
Retrieves the height of the caption area for the ribbon bar.  
  
## Syntax  
  
```  
int GetCaptionHeight() const;  
```  
  
## Return Value  
 The height, in pixels, of the caption area for the ribbon bar.  
  
## Remarks  
  
## Requirements  
 **Header:** afxribbonbar.h
