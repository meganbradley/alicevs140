# rotate (STL-CLR)
Exchanges the elements in two adjacent ranges.  
  
## Syntax  
  
```  
template<class _FwdIt> inline  
    void rotate(_FwdIt _First, _FwdIt _Mid, _FwdIt _Last);  
```  
  
## Remarks  
 This function behaves the same as the STL function `rotate`. For more information, see [rotate](../vs140/rotate.md).  
  
## Requirements  
 **Header:** <cliext/algorithm>  
  
 **Namespace:** cliext
