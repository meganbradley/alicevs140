# CMFCPropertyGridProperty::SetValue
Sets the value of a property grid property.  
  
## Syntax  
  
```  
virtual void SetValue(  
   const _variant_t& varValue   
);  
```  
  
#### Parameters  
 [in] `varValue`  
 A reference to a value.
