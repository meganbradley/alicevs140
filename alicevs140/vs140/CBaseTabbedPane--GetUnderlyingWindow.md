# CBaseTabbedPane::GetUnderlyingWindow
Gets the underlying (wrapped) tab window.  
  
## Syntax  
  
```  
virtual CMFCBaseTabCtrl* GetUnderlyingWindow();  
```  
  
## Return Value  
 A pointer to the underlying tab window.  
  
## Requirements  
 **Header:** afxBaseTabbedPane.h
