# CShellManager::GetItemCount
Returns the number of items in an item list.  
  
## Syntax  
  
```  
UINT GetItemCount(  
   LPCITEMIDLIST pidl   
);  
```  
  
#### Parameters  
 [in] `pidl`  
 A pointer to an item list.  
  
## Return Value  
 The number of items in the item list.  
  
## Requirements  
 **Header:** afxshellmanager.h
