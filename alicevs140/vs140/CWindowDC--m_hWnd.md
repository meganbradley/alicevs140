# CWindowDC::m_hWnd
The `HWND` of the `CWnd` pointer is used to construct the `CWindowDC` object.  
  
## Syntax  
  
```  
  
HWND m_hWnd;  
  
```  
  
## Remarks  
 `m_hWnd` is a protected variable of type `HWND`.

 See the example for [CWindowDC::CWindowDC](../vs140/CWindowDC--CWindowDC.md).
