# CCmdTarget::EnableConnections
Enables event firing over connection points.  
  
## Syntax  
  
```  
  
void EnableConnections( );  
```  
  
## Remarks  
 To enable connection points, call this member function in the constructor of your derived class.  
  
## Requirements  
 **Header:** afxwin.h
