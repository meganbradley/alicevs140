# CPane::GetMinSize
Retrieves the minimum allowed size for the pane.  
  
## Syntax  
  
```  
virtual void GetMinSize(  
   CSize& size  
) const;  
```  
  
#### Parameters  
 [out] `size`  
 A `CSize` object that is filled with the minimum allowed size.
