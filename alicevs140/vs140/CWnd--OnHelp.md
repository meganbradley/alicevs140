# CWnd::OnHelp
Handles F1 Help within the application (using the current context).  
  
## Syntax  
  
```  
  
afx_msg void OnHelp( );  
  
```  
  
## Remarks  
 See [CWinApp::OnHelp](../vs140/CWinApp--OnHelp.md) for more information.  
  
## Requirements  
 **Header:** afxwin.h
