# CDumpContext::GetDepth
Determines whether a deep or shallow dump is in process.  
  
## Syntax  
  
```  
  
int GetDepth( ) const;  
```  
  
## Return Value  
 The depth of the dump as set by `SetDepth`.  
  
## Example  
 See the example for [SetDepth](../vs140/CDumpContext--SetDepth.md).
