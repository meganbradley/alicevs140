# basic_streambuf::pptr
A protected function that returns a pointer to the next element of the output buffer.  
  
## Syntax  
  
```  
  
char  
_  
type *pptr( ) const;  
  
```  
  
## Return Value  
 A pointer to the next element of the output buffer.  
  
## Requirements  
 **Header:** <streambuf\>  
  
 **Namespace:** std
