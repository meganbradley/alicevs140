# .LISTALL
Starts listing of all statements.  
  
## Syntax  
  
```  
  
.LISTALL  
  
```  
  
## Remarks  
 Equivalent to the combination of [.LIST](../vs140/.LIST.md), [.LISTIF](../vs140/.LISTIF.md), and [.LISTMACROALL](../vs140/.LISTMACROALL.md).
