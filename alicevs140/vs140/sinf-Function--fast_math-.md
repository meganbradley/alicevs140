# sinf Function (fast_math)
Calculates the sine value of the argument  
  
## Syntax  
  
```  
inline float sinf(  
   float _X  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
## Return Value  
 Returns the sine value of the argument  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::fast_math
