# CMFCPropertyGridCtrl::GetHeaderHeight
Retrieves the height of the header of a property grid control.  
  
## Syntax  
  
```  
int GetHeaderHeight() const;  
```  
  
## Return Value  
 The height of the header, in pixels.

## Requirements  
 **Header:** afxpropertygridctrl.h
