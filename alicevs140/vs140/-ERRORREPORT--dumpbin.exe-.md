# -ERRORREPORT (dumpbin.exe)
```  
/ERRORREPORT[NONE | PROMPT | QUEUE | SEND ]  
```  
  
## Remarks  
 If dumpbin.exe fails at runtime, you can use **/ERRORREPORT** to send information to Microsoft about these internal errors.  
  
 For more information about **/ERRORREPORT**, see [/errorReport (Report Internal Compiler Errors)](../vs140/-errorReport--Report-Internal-Compiler-Errors-.md).
