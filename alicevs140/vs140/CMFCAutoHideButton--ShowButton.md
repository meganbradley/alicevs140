# CMFCAutoHideButton::ShowButton
Shows or hides the auto-hide button.  
  
## Syntax  
  
```  
virtual void ShowButton(  
   BOOL bShow   
);  
```  
  
#### Parameters  
 [in] `bShow`  
 A Boolean that specifies whether to show the auto-hide button.  
  
## Requirements  
 **Header:** afxautohidebutton.h
