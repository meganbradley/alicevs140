# codecvt::intern_type
A character type that is used for internal representations.  
  
## Syntax  
  
```  
typedef CharType intern_type;  
```  
  
## Remarks  
 The type is a synonym for the template parameter **CharType**.  
  
## Requirements  
 **Header:** <locale\>  
  
 **Namespace:** std
