# COleSafeArray::GetByteArray
Copies the contents of the safe array into a `CByteArray`.  
  
## Syntax  
  
```  
  
      void GetByteArray(   
   CByteArray& bytes   
);  
```  
  
#### Parameters  
 `bytes`  
 A reference to a [CByteArray](../vs140/CByteArray-Class.md) object.  
  
## Requirements  
 **Header:** afxdisp.h
