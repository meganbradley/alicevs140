# COleControl::SetFont
Sets the stock Font property of your control.  
  
## Syntax  
  
```  
  
      void SetFont(  
   LPFONTDISP pFontDisp   
);  
```  
  
#### Parameters  
 *pFontDisp*  
 A pointer to a Font dispatch interface.  
  
## Requirements  
 **Header:** afxctl.h
