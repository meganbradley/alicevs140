# basic_string::shrink_to_fit
Discards the excess capacity of the string.  
  
## Syntax  
  
```  
void shrink_to_fit();  
```  
  
## Remarks  
 This member function eliminates any unneeded storage in the container.  
  
## Requirements  
 **Header:** <string\>  
  
 **Namespace:** std
