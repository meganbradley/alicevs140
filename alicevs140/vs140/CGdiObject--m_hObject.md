# CGdiObject::m_hObject
A `HANDLE` containing the `HBITMAP`, **HRGN**, `HBRUSH`, `HPEN`, `HPALETTE`, or **HFONT** attached to this object.  
  
## Syntax  
  
```  
  
HGDIOBJ m_hObject;  
  
```  
  
## Requirements  
 **Header:** afxwin.h
