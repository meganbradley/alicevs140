# CMFCPopupMenuBar::GetLastCommandIndex
Gets the index of the most recently invoked menu command.  
  
## Syntax  
  
```  
static int __stdcall GetLastCommandIndex( );  
```  
  
## Return Value  
 Returns the index of the last menu command that has been invoked.

## Requirements
