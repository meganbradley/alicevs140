# Sids::TerminalServer
Returns the SECURITY_TERMINAL_SERVER_RID SID.  
  
> [!IMPORTANT]
>  This function cannot be used in applications that execute in the [!INCLUDE[wrt](../vs140/includes/wrt_md.md)].  
  
## Syntax  
  
```  
  
CSid TerminalServer( ) throw(...);  
  
```
