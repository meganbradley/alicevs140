# CUrl::GetUserNameLength
Call this method to get the length of the user name.  
  
## Syntax  
  
```  
  
inline DWORD GetUserNameLength( ) const throw( );  
  
```  
  
## Return Value  
 Returns the user name length.  
  
## Requirements  
 **Header:** atlutil.h
