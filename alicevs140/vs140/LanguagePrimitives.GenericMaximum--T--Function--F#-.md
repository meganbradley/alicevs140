# LanguagePrimitives.GenericMaximum&lt;&#39;T&gt; Function (F#)
Take the maximum of two values structurally according to the order given by [GenericComparison](../vs140/LanguagePrimitives.GenericComparison--T--Function--F#-.md).  
  
 **Namespace/Module Path:** Microsoft.FSharp.Core.LanguagePrimitives  
  
 **Assembly:** FSharp.Core (in FSharp.Core.dll)  
  
## Syntax  
  
```  
// Signature:  
GenericMaximum : 'T -> 'T -> 'T (requires comparison)  
  
// Usage:  
GenericMaximum e1 e2  
```  
  
#### Parameters  
 `e1`  
 Type: `'T`  
  
 The first value.  
  
 `e2`  
 Type: `'T`  
  
 The second value.  
  
## Return Value  
 The maximum value.
