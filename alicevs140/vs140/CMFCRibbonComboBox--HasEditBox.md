# CMFCRibbonComboBox::HasEditBox
Indicates whether the control contains an edit box.  
  
## Syntax  
  
```  
BOOL HasEditBox() const;  
```  
  
## Return Value  
 `TRUE` if the control contains an edit box; otherwise, `FALSE`.  
  
## Remarks  
  
## Requirements  
 **Header:** afxribboncombobox.h
