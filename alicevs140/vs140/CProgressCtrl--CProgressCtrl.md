# CProgressCtrl::CProgressCtrl
Constructs a `CProgressCtrl` object.  
  
## Syntax  
  
```  
  
CProgressCtrl( );  
```  
  
## Remarks  
 After constructing the `CProgressCtrl` object, call `CProgressCtrl::Create` to create the progress bar control.
