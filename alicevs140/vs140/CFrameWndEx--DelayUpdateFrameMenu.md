# CFrameWndEx::DelayUpdateFrameMenu
Sets the frame menu and then updates it when command processing is idle.  
  
## Syntax  
  
```  
virtual void DelayUpdateFrameMenu(  
   HMENU hMenuAlt  
);  
```  
  
#### Parameters  
 [in] `hMenuAlt`  
 Handle to an alternative menu.
