# CMFCRibbonBar::SaveToXMLFile
Saves the Ribbon Bar to an XML file.  
  
## Syntax  
  
```  
BOOL SaveToXMLFile(  
   LPCTSTR lpszFilePath  
) const;  
```  
  
#### Parameters  
 `lpszFilePath`  
 Specifies the output file.  
  
## Return Value  
 `TRUE` if successful; otherwise `FALSE`.

## Requirements  
 **Header:** afxribbonbar.h
