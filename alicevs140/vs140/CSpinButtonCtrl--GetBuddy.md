# CSpinButtonCtrl::GetBuddy
Retrieves a pointer to the current buddy window.  
  
## Syntax  
  
```  
  
CWnd* GetBuddy( ) const;  
```  
  
## Return Value  
 A pointer to the current buddy window.  
  
## Requirements  
 **Header:** afxcmn.h
