# CLongBinary::m_hData
Stores a Windows `HGLOBAL` handle to the actual binary large object data.  
  
## Syntax  
  
```  
  
HGLOBAL m_hData;  
  
```  
  
## Requirements  
 **Header:** afxdb_.h
