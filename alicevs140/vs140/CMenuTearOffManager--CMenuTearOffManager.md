# CMenuTearOffManager::CMenuTearOffManager
Constructs a [CMenuTearOffManager](../vs140/CMenuTearOffManager-Class.md) object.  
  
## Syntax  
  
```  
CMenuTearOffManager();  
```  
  
## Remarks  
 In most cases, you should not create a `CMenuTearOffManager` manually. The framework of your application creates the `CMenuTearOffManager` object when you call [CWinAppEx::EnableTearOffMenus](../vs140/CWinAppEx--EnableTearOffMenus.md).  
  
## Requirements  
 **Header:** afxmenutearoffmanager.h
