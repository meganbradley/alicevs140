# CFrameWnd::CFrameWnd
Constructs a `CFrameWnd` object, but does not create the visible frame window.  
  
## Syntax  
  
```  
  
CFrameWnd( );  
```  
  
## Remarks  
 Call **Create** to create the visible window.  
  
## Requirements  
 **Header:** afxwin.h
