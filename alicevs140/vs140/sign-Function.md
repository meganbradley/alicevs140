# sign Function
Determines the sign of the specified argument.  
  
## Syntax  
  
```  
inline int sign(  
   int _X  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Integer value  
  
## Return Value  
 The sign of the argument.  
  
## Requirements  
 **Header:** amp.h  
  
 **Namespace:** Concurrency::direct3d
