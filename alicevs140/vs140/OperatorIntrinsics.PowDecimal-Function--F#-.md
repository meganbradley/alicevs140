# OperatorIntrinsics.PowDecimal Function (F#)
This is a library intrinsic. Calls to this function may be generated by uses of the generic [pown](../vs140/Operators.pown-^T--Function--F#-.md) operator on values of type `decimal`.  
  
 **Namespace/Module Path:** Microsoft.FSharp.Core.Operators.OperatorIntrinsics  
  
 **Assembly:** FSharp.Core (in FSharp.Core.dll)  
  
## Syntax  
  
```  
// Signature:  
PowDecimal : decimal -> int -> decimal  
  
// Usage:  
PowDecimal x n  
```  
  
#### Parameters  
 `x`  
 Type: [decimal](../vs140/Core.decimal-Type-Abbreviation--F#-.md)  
  
 `n`  
 Type: [int](../vs140/Core.int-Type-Abbreviation--F#-.md)  
  
## Return Value  
 The result of the exponentiation operation.  
  
## Remarks  
 This function is for use by compiled F# code and should not be used directly.
