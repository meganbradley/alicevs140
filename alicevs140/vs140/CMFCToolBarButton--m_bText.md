# CMFCToolBarButton::m_bText
Specifies whether a text label is displayed on the button.  
  
## Syntax  
  
```  
BOOL m_bText;  
```  
  
## Remarks  
 If this data member is set to `TRUE`, the framework displays the text label of the toolbar button; otherwise the framework does not display the text label. This member affects the return value of the [CMFCToolBarButton::m_bText](../vs140/CMFCToolBarButton--m_bText.md) method.  
  
## Requirements  
 **Header:** afxtoolbarbutton.h
