# END_CATCH_ALL
Marks the end of the last `CATCH_ALL` or `AND_CATCH_ALL` block.  
  
## Syntax  
  
```  
  
END_CATCH_ALL  
  
```  
  
## Requirements  
 **Header:** afx.h
