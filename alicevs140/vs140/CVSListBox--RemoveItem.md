# CVSListBox::RemoveItem
Removes an item from an editable list control.  
  
## Syntax  
  
```  
virtual BOOL RemoveItem(  
   int iIndex   
);  
```  
  
#### Parameters  
 [in] `iIndex`  
 The zero-based index of an editable list control item.  
  
## Return Value  
 `TRUE` if the specified item is removed; otherwise, `FALSE`.
