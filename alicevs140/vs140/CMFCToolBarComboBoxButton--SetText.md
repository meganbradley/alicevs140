# CMFCToolBarComboBoxButton::SetText
Sets the text in the edit box of the combo box button.  
  
## Syntax  
  
```  
void SetText(  
   LPCTSTR lpszText  
);  
```  
  
#### Parameters  
 [in] `lpszText`  
 Pointer to a string that that contains the text for the edit box.  
  
## Requirements  
 **Header:** afxtoolbarcomboboxbutton.h
