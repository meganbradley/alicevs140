# CMFCTasksPaneTask::m_bIsBold
Determines whether a task label is drawn in bold text.  
  
## Syntax  
  
```  
BOOL m_bIsBold;  
```  
  
## Remarks  
 Set this member to `TRUE` to display bold text for the task label.  
  
## Requirements  
 **Header:** afxTasksPane.h
