# CComBSTR::ToLower
Converts the contained string to lowercase.  
  
## Syntax  
  
```  
  
HRESULT ToLower( ) throw( );  
  
```  
  
## Return Value  
 A standard `HRESULT` value.  
  
## Remarks  
 See **CharLowerBuff** for more information on how the conversion is performed.  
  
## Requirements  
 **Header:** atlbase.h
