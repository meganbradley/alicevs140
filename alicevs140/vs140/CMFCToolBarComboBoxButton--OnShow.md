# CMFCToolBarComboBoxButton::OnShow
Called by the framework when the combo box button is hidden or displayed.  
  
## Syntax  
  
```  
virtual void OnShow(  
   BOOL bShow  
);  
```  
  
#### Parameters  
 [in] `bShow`  
 Whether to hide or display the combo box button.  
  
## Requirements  
 **Header:** afxtoolbarcomboboxbutton.h
