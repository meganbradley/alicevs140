# concurrent_queue::clear Method
Clears the concurrent queue, destroying any currently enqueued elements. This method is not concurrency-safe.  
  
## Syntax  
  
```  
void clear();  
```  
  
## Requirements  
 **Header:** concurrent_queue.h  
  
 **Namespace:** concurrency
