# CDockablePane::IsAutoHideMode
Determines whether a pane is in autohide mode.  
  
## Syntax  
  
```  
virtual BOOL IsAutoHideMode() const;  
```  
  
## Return Value  
 `TRUE` if the dockable pane is in autohide mode; otherwise, `FALSE`.  
  
## Requirements  
 **Header:** afxDockablePane.h
