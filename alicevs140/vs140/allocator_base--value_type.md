# allocator_base::value_type
A type that is managed by the allocator.  
  
## Syntax  
  
```  
typedef Type value_type;  
```  
  
## Remarks  
 The object type implements  as a typedef for `Type`.  
  
## Requirements  
 **Header:** <allocators\>  
  
 **Namespace:** stdext
