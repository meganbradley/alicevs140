# COlePropertyPage::SetDialogResource
Sets the property page's dialog resource.  
  
## Syntax  
  
```  
  
      void SetDialogResource(  
   HGLOBAL hDialog   
);  
```  
  
#### Parameters  
 *hDialog*  
 Handle to the property page's dialog resource.  
  
## Requirements  
 **Header:** afxctl.h
