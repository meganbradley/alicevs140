# CComSafeArray::GetDimensions
Returns the number of dimensions in the array.  
  
## Syntax  
  
```  
  
UINT GetDimensions( ) const;  
  
```  
  
## Return Value  
 Returns the number of dimensions in the array.  
  
## Requirements  
 **Header:** atlsafe.h
