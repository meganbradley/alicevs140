# CFileDialog::IsPickFoldersMode
Determines if the current dialog is in folder picker mode.  
  
## Syntax  
  
```  
BOOL IsPickFoldersMode() const;  
```  
  
## Return Value  
 `TRUE` if the dialog is in folder picker mode; otherwise `FALSE`.
