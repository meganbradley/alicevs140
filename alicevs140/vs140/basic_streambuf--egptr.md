# basic_streambuf::egptr
A protected function that returns a pointer just past the end of the input buffer.  
  
## Syntax  
  
```  
  
char  
_  
type *egptr( ) const;  
  
```  
  
## Return Value  
 A pointer just past the end of the input buffer.  
  
## Requirements  
 **Header:** <streambuf\>  
  
 **Namespace:** std
