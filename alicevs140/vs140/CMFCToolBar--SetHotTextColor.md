# CMFCToolBar::SetHotTextColor
Sets the text color for hot toolbar buttons.  
  
## Syntax  
  
```  
static void SetHotTextColor(  
   COLORREF clrText   
);  
```  
  
#### Parameters  
 [in] `clrText`  
 Specifies the text color for toolbar buttons that are hot-tracked.  
  
## Remarks  
 For more information about hot-tracked toolbar buttons, see [CMFCToolBar::GetHotBorder](../vs140/CMFCToolBar--GetHotBorder.md) and [CMFCToolBar::SetHotBorder](../vs140/CMFCToolBar--SetHotBorder.md).  
  
## Requirements  
 **Header:** afxtoolbar.h
