# CWnd::GetWindowlessChildCount
Retrieves the number of associated windowless child windows.  
  
## Syntax  
  
```  
  
long GetWindowlessChildCount( );  
  
```  
  
## Return Value  
 The number of windowless child windows associated with the `CWnd` object.  
  
## Requirements  
 **Header:** afxwin.h
