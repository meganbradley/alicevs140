# CMFCPropertyGridCtrl::ExpandAll
Expands or collapses all property grid control nodes.  
  
## Syntax  
  
```  
void ExpandAll(  
   BOOL bExpand=TRUE   
);  
```  
  
#### Parameters  
 [in] `bExpand`  
 `TRUE` to expand all nodes; `FALSE` to collapse all nodes. The default value is `TRUE`.

 **Header:** afxpropertygridctrl.h
