# CMFCRibbonEdit::DropDownList
Drops down a list box.  
  
## Syntax  
  
```  
virtual void DropDownList();  
```  
  
## Remarks  
 By default this method does nothing. Override this method to drop down a list box.  
  
## Requirements  
 **Header:** afxribbonedit.h
