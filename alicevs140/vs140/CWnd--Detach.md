# CWnd::Detach
Detaches a Windows handle from a `CWnd` object and returns the handle.  
  
## Syntax  
  
```  
  
HWND Detach( );  
```  
  
## Return Value  
 A `HWND` to the Windows object.  
  
## Example  
 See the example for [CWnd::Attach](../vs140/CWnd--Attach.md).  
  
## Requirements  
 **Header:** afxwin.h
