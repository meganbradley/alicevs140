# CMFCRibbonColorButton::GetColor
Returns the currently selected color.  
  
## Syntax  
  
```  
COLORREF GetColor() const;  
```  
  
## Return Value  
 The color selected by clicking the button.  
  
## Requirements  
 **Header:** afxribboncolorbutton.h
