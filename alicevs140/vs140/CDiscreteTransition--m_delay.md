# CDiscreteTransition::m_delay
The amount of time by which to delay the instantaneous switch to the final value.  
  
## Syntax  
  
```  
UI_ANIMATION_SECONDS m_delay;  
```  
  
## Requirements  
 **Header:** afxanimationcontroller.h
