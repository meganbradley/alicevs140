# CMFCPropertyGridProperty::SetDescription
Specifies the text that describes the current property.  
  
## Syntax  
  
```  
void SetDescription(  
   const CString& strDescr  
);  
```  
  
#### Parameters  
 [in] `strDescr`  
 Text that describes the current property.
