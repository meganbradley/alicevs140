# REFLECT_NOTIFICATIONS
Reflects notification messages back to the child window (control) that sent them.  
  
## Syntax  
  
```  
  
REFLECT_NOTIFICATIONS( )  
  
```  
  
## Remarks  
 Specify this macro as part of the parent window's message map.  
  
## Requirements  
 **Header:** atlwin.h
