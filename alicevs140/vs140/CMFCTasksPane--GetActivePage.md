# CMFCTasksPane::GetActivePage
Returns the zero-based index for the active page.  
  
## Syntax  
  
```  
int GetActivePage() const;  
```  
  
## Return Value  
 The zero-based index of the active page.  
  
## Requirements  
 **Header:** afxTasksPane.h
