# CDefaultCharTraits::CharToUpper
Call this function to convert a character to uppercase.  
  
## Syntax  
  
```  
  
      static wchar_t CharToUpper(  
   wchar_t x  
);  
static char CharToUpper(  
   char x  
);  
```  
  
#### Parameters  
 *x*  
 The character to convert to uppercase.  
  
## Requirements  
 **Header:** atlcoll.h
