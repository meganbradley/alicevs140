# CMFCDesktopAlertWndInfo::m_nURLCmdID
The command ID associated with a link on the desktop alert window.  
  
## Syntax  
  
```  
UINT m_nURLCmdID;  
```  
  
## Remarks  
 The command ID is sent to the owner of the popup window when the user clicks on the link specified by [CMFCDesktopAlertWndInfo::m_strURL](../vs140/CMFCDesktopAlertWndInfo--m_strURL.md).  
  
## Requirements  
 **Header:** afxDesktopAlertDialog.h
