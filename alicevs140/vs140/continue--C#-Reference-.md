# continue (C# Reference)
The `continue` statement passes control to the next iteration of the enclosing [while](../vs140/while--C#-Reference-.md), [do](../vs140/do--C#-Reference-.md), [for](../vs140/for--C#-Reference-.md), or [foreach](../Topic/foreach,%20in%20\(C%23%20Reference\).md) statement in which it appears.  
  
## Example  
 In this example, a counter is initialized to count from 1 to 10. By using the `continue` statement in conjunction with the expression `(i < 9)`, the statements between `continue` and the end of the `for` body are skipped.
