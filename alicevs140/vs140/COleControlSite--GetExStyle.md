# COleControlSite::GetExStyle
Retrieves the window's extended styles.  
  
## Syntax  
  
```  
  
virtual DWORD GetExStyle( ) const;  
  
```  
  
## Return Value  
 The control window's extended styles.  
  
## Remarks  
 To retrieve the regular styles, call [COleControlSite::GetStyle](../vs140/COleControlSite--GetStyle.md).  
  
## Requirements  
 **Header:** afxocc.h
