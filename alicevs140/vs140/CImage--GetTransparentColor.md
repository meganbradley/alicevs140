# CImage::GetTransparentColor
Retrieves the indexed location of the transparent color in the color palette.  
  
## Syntax  
  
```  
  
LONG GetTransparentColor( ) const throw( );  
  
```  
  
## Return Value  
 The index of the transparent color.  
  
## Requirements  
 **Header:** atlimage.h
