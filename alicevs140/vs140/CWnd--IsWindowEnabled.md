# CWnd::IsWindowEnabled
Specifies whether `CWnd` is enabled for mouse and keyboard input.  
  
## Syntax  
  
```  
  
BOOL IsWindowEnabled( ) const;  
```  
  
## Return Value  
 Nonzero if `CWnd` is enabled; otherwise 0.
