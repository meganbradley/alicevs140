# DD
Allocates and optionally initializes a double word (4 bytes) of storage for each `initializer`. `DD` is a synonym of [DWORD](../vs140/DWORD.md).  
  
## Syntax  
  
```  
[[name]] DD initializer [[, initializer]]...  
```  
  
## Remarks  
 Can also be used as a type specifier anywhere a type is legal.
