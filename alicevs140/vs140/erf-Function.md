# erf Function
Computes the error function of _X  
  
## Syntax  
  
```  
inline float erf(  
   float _X  
) restrict(amp);  
inline double erf(  
   double _X  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
## Return Value  
 Returns the error function of _X  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::precise_math
