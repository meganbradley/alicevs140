# &#39;&lt;specifier&gt;&#39; is not valid on a Structure declaration
You have used a specifier, such as `Dim` or `Override`, that is not valid on a `Structure` declaration.  
  
 **Error ID:** BC30395  
  
### To correct this error  
  
-   Remove the specifier.
