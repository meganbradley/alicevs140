# CHwndRenderTarget::Detach
Detaches render target interface from the object  
  
## Syntax  
  
```  
ID2D1HwndRenderTarget* Detach();  
```  
  
## Return Value  
 Pointer to detached render target interface.  
  
## Requirements  
 **Header:** afxrendertarget.h
