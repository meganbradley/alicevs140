# CFrameWnd::ShowOwnedWindows
Call this member function to show all windows that are descendants of the `CFrameWnd` object.  
  
## Syntax  
  
```  
  
      void ShowOwnedWindows(  
   BOOL bShow   
);  
```  
  
#### Parameters  
 `bShow`  
 Specifies whether the owned windows are to be shown or hidden.  
  
## Requirements  
 **Header:** afxwin.h
