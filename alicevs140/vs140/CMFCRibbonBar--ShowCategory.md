# CMFCRibbonBar::ShowCategory
Shows or hides the specified ribbon category.  
  
## Syntax  
  
```  
void ShowCategory(  
   int nIndex,  
   BOOL bShow=TRUE   
);  
```  
  
#### Parameters  
 [in] `nIndex`  
 The index of the ribbon category.  
  
 [in] `bShow`  
 If `TRUE`, show the ribbon category; otherwise, hide the ribbon category.  
  
## Requirements  
 **Header:** afxribbonbar.h
