# CAtlBaseModule::GetModuleInstance
Returns the module instance from a `CAtlBaseModule` object.  
  
## Syntax  
  
```  
  
HINSTANCE GetModuleInstance( ) throw( );  
  
```  
  
## Return Value  
 Returns the module instance.  
  
## Requirements  
 **Header:** atlcore.h
