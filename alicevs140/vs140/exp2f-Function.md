# exp2f Function
Calculates the base-2 exponential of the argument  
  
## Syntax  
  
```  
inline float exp2f(  
   float _X  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
## Return Value  
 Returns the base-2 exponential of the argument  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::precise_math
