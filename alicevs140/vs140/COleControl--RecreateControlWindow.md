# COleControl::RecreateControlWindow
Destroys and re-creates the control's window.  
  
## Syntax  
  
```  
  
void RecreateControlWindow( );  
```  
  
## Remarks  
 This may be necessary if you need to change the window's style bits.  
  
## Requirements  
 **Header:** afxctl.h
