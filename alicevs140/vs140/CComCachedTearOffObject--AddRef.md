# CComCachedTearOffObject::AddRef
Increments the reference count of the `CComCachedTearOffObject` object by 1.  
  
## Syntax  
  
```  
  
STDMETHOD_(ULONG, AddRef)( );  
  
```  
  
## Return Value  
 A value that may be useful for diagnostics and testing.  
  
## Requirements  
 **Header:** atlcom.h
