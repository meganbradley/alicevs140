# Compiler Error CS1724
Value specified for the argument to 'System.Runtime.InteropServices.DefaultCharSetAttribute' is not valid  
  
 This error is generated by an invalid argument to the <xref:System.Runtime.InteropServices.DefaultCharSetAttribute?qualifyHint=False> class.  
  
## Example  
 The following example generates CS1724.  
  
```  
// CS1724.cs  
using System.Runtime.InteropServices;  
// To resolve, replace 42 with a valid CharSet value.  
 [module:DefaultCharSetAttribute((CharSet)42)]   // CS1724  
class C {   
  
[DllImport("F.Dll")]  
extern static void FW1Named();  
  
static void Main() {}  
}  
```
