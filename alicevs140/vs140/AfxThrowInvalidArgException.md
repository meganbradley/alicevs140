# AfxThrowInvalidArgException
Throws an invalid argument exception.  
  
## Syntax  
  
```  
  
void AfxThrowInvalidArgException( );  
```  
  
## Remarks  
 This function is called when invalid arguments are used.  
  
## Requirements  
 **Header:** afx.h
