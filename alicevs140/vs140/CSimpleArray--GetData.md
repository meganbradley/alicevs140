# CSimpleArray::GetData
Returns a pointer to the data stored in the array.  
  
## Syntax  
  
```  
  
T* GetData( ) const;  
  
```  
  
## Return Value  
 Returns a pointer to the data in the array.  
  
## Requirements  
 **Header:** atlsimpcoll.h
