# unordered_multiset::iterator
A type that provides a constant [forward iterator](../vs140/forward_iterator_tag-Struct.md) that can read elements in an unordered_multiset.  
  
## Syntax  
  
```  
typedef implementation-defined iterator;  
```  
  
## Example  
 See the example for [begin](../vs140/multiset--begin.md) for an example of how to declare and use an **iterator**.  
  
## Requirements  
 **Header:** <unordered_set>  
  
 **Namespace:** std
