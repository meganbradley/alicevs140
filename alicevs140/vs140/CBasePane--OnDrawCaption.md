# CBasePane::OnDrawCaption
The framework calls this method when the caption is drawn.  
  
## Syntax  
  
```  
virtual void OnDrawCaption();  
```  
  
## Requirements  
 **Header:** afxbasepane.h  
  
## Remarks  
 This method has no functionality for the `CBasePane` class.
