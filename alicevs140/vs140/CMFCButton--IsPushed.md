# CMFCButton::IsPushed
Indicates whether a button is pushed.  
  
## Syntax  
  
```  
BOOL IsPushed() const;  
```  
  
## Return Value  
 TRUE if the button is pushed; otherwise, FALSE.  
  
## Remarks  
  
## Requirements  
 **Header:** afxbutton.h
