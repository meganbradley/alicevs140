# CDataRecoveryHandler::GetAutosaveInterval
Returns the interval between autosave tries.  
  
## Syntax  
  
```  
virtual int GetAutosaveInterval() const;  
```  
  
## Return Value  
 The number of milliseconds between autosave tries.  
  
## Requirements  
 **Header:** afxdatarecovery.h
