# CMFCRibbonBaseElement::GetText
Returns the text associated with the ribbon element.  
  
## Syntax  
  
```  
LPCTSTR GetText() const;  
```  
  
## Return Value  
 The text associated with the ribbon element.  
  
## Requirements  
 **Header:** afxbaseribbonelement.h
