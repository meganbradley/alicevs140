# CClientDC::m_hWnd
The `HWND` of the `CWnd` pointer used to construct the `CClientDC` object.  
  
## Syntax  
  
```  
  
HWND m_hWnd;  
  
```  
  
## Remarks  
 `m_hWnd` is a protected variable.

 See the example for [CClientDC::CClientDC](../vs140/CClientDC--CClientDC.md).
