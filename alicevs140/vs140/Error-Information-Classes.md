# Error Information Classes
The following class indicates how error information is handled:  
  
-   [ISupportErrorInfoImpl](../vs140/ISupportErrorInfoImpl-Class.md) Determines whether the object supports the [IErrorInfo](assetId:///4dda6909-2d9a-4727-ae0c-b5f90dcfa447) interface. **IErrorInfo** allows error information to be propagated back to the client.
