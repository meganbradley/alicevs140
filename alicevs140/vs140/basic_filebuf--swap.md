# basic_filebuf::swap
Exchanges the contents of this `basic_filebuf` for the contents of the provided `basic_filebuf`.  
  
## Syntax  
  
```  
void swap(basic_filebuf& _Right);  
```  
  
#### Parameters  
 `_Right`  
 An `lvalue` reference to another `basic_filebuf`.  
  
## Requirements  
 **Header:** <fstream\>  
  
 **Namespace:** std
