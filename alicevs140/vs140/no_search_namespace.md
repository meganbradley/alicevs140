# no_search_namespace
**C++ Specific**  
  
 Has the same functionality as the [no_namespace](../vs140/no_namespace.md) attribute but is used on type libraries that you use the #import directive with the [auto_search](../vs140/auto_search.md) attribute.  
  
## Syntax  
  
```  
no_search_namespace  
```
