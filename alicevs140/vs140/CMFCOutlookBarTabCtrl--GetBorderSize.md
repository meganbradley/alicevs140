# CMFCOutlookBarTabCtrl::GetBorderSize
Returns the border size of the Outlook tab control.  
  
## Syntax  
  
```  
int GetBorderSize() const;  
```  
  
## Return Value  
 The border size, in pixels.  
  
## Requirements  
 **Header:** afxOutlookBarTabCtrl.h
