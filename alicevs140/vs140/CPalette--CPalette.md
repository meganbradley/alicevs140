# CPalette::CPalette
Constructs a `CPalette` object.  
  
## Syntax  
  
```  
  
CPalette( );  
```  
  
## Remarks  
 The object has no attached palette until you call `CreatePalette` to attach one.  
  
## Requirements  
 **Header:** afxwin.h
