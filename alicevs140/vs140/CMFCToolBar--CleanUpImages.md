# CMFCToolBar::CleanUpImages
Frees the system resources allocated for toolbar images.  
  
## Syntax  
  
```  
static void CMFCToolBar::CleanUpImages();  
```  
  
## Remarks  
 The framework calls this method when an application shuts down.  
  
## Requirements  
 **Header:** afxtoolbar.h
