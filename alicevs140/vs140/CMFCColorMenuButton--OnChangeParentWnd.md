# CMFCColorMenuButton::OnChangeParentWnd
Called by the framework when the parent window changes.  
  
## Syntax  
  
```  
virtual void OnChangeParentWnd(  
   CWnd* pWndParent   
);  
```  
  
#### Parameters  
 [in] `pWndParent`  
 A pointer to the new parent window.
