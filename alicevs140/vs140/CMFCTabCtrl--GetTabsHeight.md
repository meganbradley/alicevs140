# CMFCTabCtrl::GetTabsHeight
Retrieves the height of the tab area of the current tab control.  
  
## Syntax  
  
```  
virtual int GetTabsHeight() const;  
```  
  
## Return Value  
 The height of the tab area if any tab is visible, or zero if no tab is visible.  
  
## Requirements  
 **Header:** afxtabctrl.h
