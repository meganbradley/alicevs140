# wstring_convert::int_type
A type that represents an integer.  
  
## Syntax  
  
```  
typedef typename wide_string::traits_type::int_type int_type;  
```  
  
## Remarks  
 The type is a synonym for `wide_string::traits_type::int_type`.  
  
## Requirements  
 **Header:** <locale\>  
  
 **Namespace:** std
