# CMFCPropertyGridProperty::IsVisible
Indicates whether the current property is visible.  
  
## Syntax  
  
```  
BOOL IsVisible() const;  
```  
  
## Return Value  
 `TRUE` if the current property is visible; otherwise; `FALSE`.

## Requirements  
 **Header:** afxpropertygridctrl.h
