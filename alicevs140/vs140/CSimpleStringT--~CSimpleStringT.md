# CSimpleStringT::~CSimpleStringT
Destroys a `CSimpleStringT` object.  
  
## Syntax  
  
```  
  
~CSimpleStringT( ) throw( );  
  
```  
  
## Remarks  
 Call this method to destroy the `CSimpleStringT` object.
