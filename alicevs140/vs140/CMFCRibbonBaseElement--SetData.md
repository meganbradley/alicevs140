# CMFCRibbonBaseElement::SetData
Associates a data item with the ribbon element.  
  
## Syntax  
  
```  
void SetData(  
   DWORD_PTR dwData   
);  
```  
  
#### Parameters  
 [in] `dwData`  
 The data value.  
  
## Requirements  
 **Header:** afxbaseribbonelement.h
