# CWindow::CheckRadioButton
Checks the specified radio button.  
  
## Syntax  
  
```  
  
      BOOL CheckRadioButton(  
   int nIDFirstButton,  
   int nIDLastButton,  
   int nIDCheckButton   
) throw();  
```
