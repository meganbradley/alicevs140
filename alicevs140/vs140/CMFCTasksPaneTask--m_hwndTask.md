# CMFCTasksPaneTask::m_hwndTask
A handle to the task window.  
  
## Syntax  
  
```  
HWND m_hwndTask;  
```  
  
## Remarks  
 To add a task window, call [CMFCTasksPane::AddWindow](../vs140/CMFCTasksPane--AddWindow.md).

 **Header:** afxTasksPane.h
