# CMFCColorPickerCtrl::SetColor
Sets the current color to the specified RGB color value.  
  
## Syntax  
  
```  
void SetColor(  
   COLORREF Color   
);  
```  
  
#### Parameters  
 [in] `Color`  
 An RGB color value.
