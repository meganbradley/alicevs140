# CMFCColorBar::GetHorzMargin
Retrieves the horizontal margin, which is the space between the left or right color cell and the client area boundary.  
  
## Syntax  
  
```  
int GetHorzMargin();  
```  
  
## Return Value  
 The horizontal margin.  
  
## Requirements  
 **Header:** afxcolorbar.h
