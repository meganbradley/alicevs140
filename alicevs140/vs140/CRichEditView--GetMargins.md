# CRichEditView::GetMargins
Call this function to retrieve the current margins used in printing.  
  
## Syntax  
  
```  
  
CRect GetMargins( ) const;  
  
```  
  
## Return Value  
 The margins used in printing, measured in `MM_TWIPS`.  
  
## Requirements  
 **Header:** afxrich.h
