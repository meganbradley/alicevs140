# codecvt::extern_type
A character type that is used for external representations.  
  
## Syntax  
  
```  
typedef Byte extern_type;  
```  
  
## Remarks  
 The type is a synonym for the template parameter **Byte**.  
  
## Requirements  
 **Header:** <locale\>  
  
 **Namespace:** std
