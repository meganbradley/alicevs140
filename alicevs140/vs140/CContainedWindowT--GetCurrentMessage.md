# CContainedWindowT::GetCurrentMessage
Returns the current message (**m_pCurrentMsg**).  
  
## Syntax  
  
```  
  
const _ATL_MSG* GetCurrentMessage( );  
  
```  
  
## Return Value  
 The current message, packaged in the `MSG` structure.  
  
## Requirements  
 **Header:** atlwin.h
