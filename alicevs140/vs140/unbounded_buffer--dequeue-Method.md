# unbounded_buffer::dequeue Method
Removes an item from the `unbounded_buffer` messaging block.  
  
## Syntax  
  
```  
_Type dequeue();  
```  
  
## Return Value  
 The payload of the message removed from the `unbounded_buffer`.  
  
## Requirements  
 **Header:** agents.h  
  
 **Namespace:** concurrency
