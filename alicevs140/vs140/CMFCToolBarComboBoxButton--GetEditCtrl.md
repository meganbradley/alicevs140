# CMFCToolBarComboBoxButton::GetEditCtrl
Returns a pointer to the edit box in the combo box button.  
  
## Syntax  
  
```  
virtual CEdit* GetEditCtrl();  
```  
  
## Return Value  
 A pointer to the edit box if the method was successful; otherwise, `NULL`.
