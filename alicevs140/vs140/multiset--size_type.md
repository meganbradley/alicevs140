# multiset::size_type
An unsigned integer type that can represent the number of elements in a multiset.  
  
## Syntax  
  
```  
typedef typename allocator_type::size_type size_type;  
```  
  
## Example  
 See example for [size](../vs140/multiset--size.md) for an example of how to declare and use `size_type`  
  
## Requirements  
 **Header:** <set\>  
  
 **Namespace:** std
