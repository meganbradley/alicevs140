# CDockingManager::RecalcLayout
Recalculates the internal layout of the controls present in the list of controls.  
  
## Syntax  
  
```  
virtual void RecalcLayout(  
   BOOL bNotify = TRUE  
);  
```  
  
#### Parameters  
 [in] `bNotify`  
 This parameter is not used.  
  
## Requirements  
 **Header:** afxdockingmanager.h
