# .ENDPROLOG
Signals the end of the prologue declarations.  
  
## Syntax  
  
```  
.ENDPROLOG  
```  
  
## Remarks  
 It is an error to use any of the prologue declarations outside of the region between [PROC](../vs140/PROC.md) FRAME and .ENDPROLOG.  
  
 For more information, see [MASM for x64 (ml64.exe)](../vs140/MASM-for-x64--ml64.exe-.md).
