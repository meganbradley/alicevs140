# &lt;c&gt; (C# Programming Guide)
## Syntax  
  
```  
<c>text</c>  
```  
  
#### Parameters  
 `text`  
 The text you would like to indicate as code.  
  
## Remarks  
 The <c\> tag gives you a way to indicate that text within a description should be marked as code. Use [<code\>](../vs140/-code---C#-Programming-Guide-.md) to indicate multiple lines as code.  
  
 Compile with [/doc](../Topic/-doc%20\(C%23%20Compiler%20Options\).md) to process documentation comments to a file.
