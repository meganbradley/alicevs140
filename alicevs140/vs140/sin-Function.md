# sin Function
Calculates the sine value of the argument  
  
## Syntax  
  
```  
inline float sin(  
   float _X  
) restrict(amp);  
inline double sin(  
   double _X  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
## Return Value  
 Returns the sine value of the argument  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::precise_math
