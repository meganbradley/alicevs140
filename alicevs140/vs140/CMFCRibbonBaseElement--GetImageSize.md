# CMFCRibbonBaseElement::GetImageSize
Returns the image size of the ribbon element.  
  
## Syntax  
  
```  
virtual CSize GetImageSize(  
   RibbonImageType R   
) const;  
```  
  
## Return Value  
 The image size of the ribbon element.
