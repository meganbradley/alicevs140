# CStatusBar::CStatusBar
Constructs a `CStatusBar` object, creates a default status-bar font if necessary, and sets the font characteristics to default values.  
  
## Syntax  
  
```  
  
CStatusBar( );  
```  
  
## Requirements  
 **Header:** afxext.h
