# CMFCImagePaintArea::GetMode
Retrieves the current drawing mode.  
  
## Syntax  
  
```  
IMAGE_EDIT_MODE GetMode() const;  
```  
  
## Return Value  
 An `IMAGE_EDIT_MODE` value that specifies the current drawing mode.  
  
## Requirements  
 **Header:** afximagepaintarea.h
