# CPane::ToggleAutoHide
Toggles auto-hide mode.  
  
## Syntax  
  
```  
virtual void ToggleAutoHide();  
```  
  
## Remarks  
 Call this method to toggle auto-hide mode. A pane must be docked to a main frame window in order to be switch to auto-hide mode.  
  
## Requirements  
 **Header:** afxPane.h
