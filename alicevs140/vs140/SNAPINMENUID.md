# SNAPINMENUID
Use this macro to declare the context menu resource of the Snap-In object.  
  
## Syntax  
  
```  
  
      SNAPINMENUID(   
   id    
)  
```  
  
#### Parameters  
 `id`  
 [in] Identifies the context menu of the Snap-In object.  
  
## Requirements  
 **Header:** atlsnap.h
