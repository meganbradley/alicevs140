# CWindow::WinHelp
Starts Windows Help.  
  
## Syntax  
  
```  
  
      BOOL WinHelp(  
   LPCTSTR lpszHelp,  
   UINT nCmd = HELP_CONTEXT,  
   DWORD dwData = 0   
) throw();  
```  
  
## Remarks  
 See [WinHelp](http://msdn.microsoft.com/library/windows/desktop/bb762267) in the [!INCLUDE[winSDK](../vs140/includes/winSDK_md.md)].

 **Header:** atlwin.h
