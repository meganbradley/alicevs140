# CAcl::operator =
Assignment operator.  
  
## Syntax  
  
```  
  
      CAcl & operator =(  
   const CAcl & rhs   
) throw(...);  
```  
  
#### Parameters  
 `rhs`  
 The `CAcl` to assign to the existing object.  
  
## Return Value  
 Returns a reference to the updated `CAcl` object.  
  
## Requirements  
 **Header:** atlsecurity.h
