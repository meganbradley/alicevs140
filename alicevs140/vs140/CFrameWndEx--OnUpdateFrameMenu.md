# CFrameWndEx::OnUpdateFrameMenu
Called by the framework to set the frame menu.  
  
## Syntax  
  
```  
virtual void OnUpdateFrameMenu(  
   HMENU hMenuAlt  
);  
```  
  
#### Parameters  
 [in] `hMenuAlt`  
 Handle to the alternative menu.
