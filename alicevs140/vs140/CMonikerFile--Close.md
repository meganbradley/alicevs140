# CMonikerFile::Close
Call this function to detach and release the stream and to release the moniker.  
  
## Syntax  
  
```  
  
virtual void Close( );  
```  
  
## Remarks  
 Can be called on unopened or already closed streams.  
  
## Requirements  
 **Header:** afxole.h
