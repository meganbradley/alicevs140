# CComGITPtr::GetCookie
Call this method to return the cookie from the `CComGITPtr` object.  
  
## Syntax  
  
```  
  
DWORD GetCookie( ) const;  
  
```  
  
## Return Value  
 Returns the cookie.  
  
## Remarks  
 The cookie is a variable used to identify an interface and its location.  
  
## Requirements  
 **Header:** atlbase.h
