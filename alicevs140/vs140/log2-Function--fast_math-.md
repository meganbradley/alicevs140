# log2 Function (fast_math)
Calculates the base-2 logarithm of the argument  
  
## Syntax  
  
```  
inline float log2(  
   float _X  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
## Return Value  
 Returns the base-2 logarithm of the argument  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::fast_math
