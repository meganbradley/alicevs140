# CComDynamicUnkArray::CComDynamicUnkArray
The constructor.  
  
## Syntax  
  
```  
  
CComDynamicUnkArray( );  
```  
  
## Remarks  
 Sets the collection size to zero and initializes the values to **NULL**. The destructor frees the collection, if necessary.  
  
## Requirements  
 **Header:** atlcom.h
