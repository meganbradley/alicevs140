# UnitNames.sievert Type Abbreviation (F#)
The SI unit of dose equivalent of radiation.  
  
 **Namespace/Module Path**: Microsoft.FSharp.Data.UnitSystems.SI.UnitNames  
  
 **Assembly**: FSharp.Core (in FSharp.Core.dll)  
  
## Syntax  
  
```  
[<Measure>] type sievert = float  
```  
  
## Remarks  
 In terms of fundamental units, the sievert has units of m<sup>2</sup> / s<sup>2</sup>. The sievert differs from the gray in that the gray measures absolute radiation dose, but the sievert measures *dose equivalent*, which corrects for the differing effects of different types of ionizing radiation.
