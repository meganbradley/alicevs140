# CUrl::GetSchemeNameLength
Call this method to get the length of the URL scheme name.  
  
## Syntax  
  
```  
  
inline DWORD GetSchemeNameLength( ) const throw( );  
  
```  
  
## Return Value  
 Returns the URL scheme name length.  
  
## Requirements  
 **Header:** atlutil.h
