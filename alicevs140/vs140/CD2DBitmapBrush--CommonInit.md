# CD2DBitmapBrush::CommonInit
Initializes the object  
  
## Syntax  
  
```  
void CommonInit(  
   D2D1_BITMAP_BRUSH_PROPERTIES* pBitmapBrushProperties  
);  
```  
  
#### Parameters  
 `pBitmapBrushProperties`  
 A pointer to the bitmap brush properties.
