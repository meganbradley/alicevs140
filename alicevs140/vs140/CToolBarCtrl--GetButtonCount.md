# CToolBarCtrl::GetButtonCount
Retrieves a count of the buttons currently in the toolbar control.  
  
## Syntax  
  
```  
  
int GetButtonCount( ) const;  
```  
  
## Return Value  
 The count of the buttons.  
  
## Requirements  
 **Header:** afxcmn.h
