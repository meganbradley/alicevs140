# promise::swap Method
Exchanges the *associated asynchronous state* of this promise object with that of a specified object.  
  
## Syntax  
  
```  
void swap(promise& Other) _NOEXCEPT;  
```  
  
#### Parameters  
 `Other`  
 A `promise` object.  
  
## Requirements  
 **Header:** future  
  
 **Namespace:** std
