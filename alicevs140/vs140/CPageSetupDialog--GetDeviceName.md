# CPageSetupDialog::GetDeviceName
Call this function after `DoModal` to retrieve the name of the currently selected printer.  
  
## Syntax  
  
```  
  
CString GetDeviceName( ) const;  
  
```  
  
## Return Value  
 The device name used by the **CPageSetupDialog** object.  
  
## Requirements  
 **Header:** afxdlgs.h
