# CMap::GetSize
Returns the number of map elements.  
  
## Syntax  
  
```  
  
INT_PTR GetSize( ) const;  
  
```  
  
## Return Value  
 The number of items in the map.  
  
## Remarks  
 Call this method to retrieve the number of elements in the map.
