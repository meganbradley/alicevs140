# CComSafeArray::IsSizable
Tests if a `CComSafeArray` object can be resized.  
  
## Syntax  
  
```  
  
bool IsSizable( ) const;  
  
```  
  
## Return Value  
 Returns **true** if the `CComSafeArray` can be resized, **false** if it cannot.  
  
## Requirements  
 **Header:** atlsafe.h
