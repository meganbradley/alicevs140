# CD2DRectF::IsNull
Returns a Boolean value that indicates whether an expression contains no valid data (Null).  
  
## Syntax  
  
```  
BOOL IsNull() const;  
```  
  
## Return Value  
 TRUE if rectangle's top, left, bottom, and right values are all equal to 0; otherwise FALSE.  
  
## Requirements  
 **Header:** afxrendertarget.h
