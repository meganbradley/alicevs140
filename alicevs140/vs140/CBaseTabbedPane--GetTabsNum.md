# CBaseTabbedPane::GetTabsNum
Returns the count of tabs in a tab window.  
  
## Syntax  
  
```  
virtual int GetTabsNum() const;  
```  
  
## Return Value  
 The number of tabs in the tabbed pane.  
  
## Requirements  
 **Header:** afxBaseTabbedPane.h
