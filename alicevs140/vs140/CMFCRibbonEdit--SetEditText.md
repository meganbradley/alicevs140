# CMFCRibbonEdit::SetEditText
Sets the text in the text box.  
  
## Syntax  
  
```  
void SetEditText(  
    CString strText  
);  
```  
  
#### Parameters  
 [in] `strText`  
 The text for the text box.  
  
## Requirements  
 **Header:** afxRibbonEdit.h
