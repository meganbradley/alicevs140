# CTabCtrl::DeleteAllItems
Removes all items from a tab control.  
  
## Syntax  
  
```  
  
BOOL DeleteAllItems( );  
```  
  
## Return Value  
 Nonzero if successful; otherwise 0.  
  
## Requirements  
 **Header:** afxcmn.h
