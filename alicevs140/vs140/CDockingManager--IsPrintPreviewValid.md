# CDockingManager::IsPrintPreviewValid
Determines if the print preview mode is set.  
  
## Syntax  
  
```  
BOOL IsPrintPreviewValid() const;  
```  
  
## Return Value  
 `TRUE` if the print preview mode is set; `FALSE` otherwise.  
  
## Requirements  
 **Header:** afxdockingmanager.h
