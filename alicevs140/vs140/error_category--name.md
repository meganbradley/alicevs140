# error_category::name
Returns the name of the category.  
  
## Syntax  
  
```  
virtual const char *name() const = 0;  
```  
  
## Return Value  
 Returns the name of the category as a null-terminated byte string.

## Requirements  
 **Header:** <system_error>  
  
 **Namespace:** std
