# CMFCToolBarButton::GetParentWnd
Retrieves the parent window of the button.  
  
## Syntax  
  
```  
CWnd* GetParentWnd() const;  
```  
  
## Return Value  
 The parent window of the button.  
  
## Requirements  
 **Header:** afxtoolbarbutton.h
