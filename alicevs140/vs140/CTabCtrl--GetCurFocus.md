# CTabCtrl::GetCurFocus
Retrieves the index of the tab with the current focus.  
  
## Syntax  
  
```  
  
int GetCurFocus( ) const;  
```  
  
## Return Value  
 The zero-based index of the tab with the current focus.  
  
## Requirements  
 **Header:** afxcmn.h
