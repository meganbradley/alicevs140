# forward_list::empty
Tests whether a forward list is empty.  
  
## Syntax  
  
```  
bool empty() const;  
```  
  
## Return Value  
 `true` if the forward list is empty; otherwise, `false`.  
  
## Requirements  
 **Header:** <forward_list>  
  
 **Namespace:** std
