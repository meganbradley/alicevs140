# CMFCRibbonCategory::GetHighlighted
Returns a highlighted element.  
  
## Syntax  
  
```  
CMFCRibbonBaseElement* GetHighlighted();  
```  
  
## Return Value  
 A pointer to a highlighted element or `NULL` if no elements are highlighted.

## Requirements
