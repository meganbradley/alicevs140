# CToolBarCtrl::SetMaxTextRows
Sets the maximum number of text rows displayed on a toolbar button.  
  
## Syntax  
  
```  
  
      BOOL SetMaxTextRows(  
   int iMaxRows   
);  
```  
  
#### Parameters  
 *iMaxRows*  
 Maximum number of rows to be set.  
  
## Return Value  
 Nonzero if successful; otherwise zero.  
  
## Requirements  
 **Header:** afxcmn.h
