# CMFCTabCtrl::GetTabMaxWidth
Retrieves the maximum width of a tab.  
  
## Syntax  
  
```  
int GetTabMaxWidth() const;  
```  
  
## Return Value  
 Maximum width of a tab, in pixels. If the return value is 0, the tab width is unlimited.  
  
## Remarks  
 Use the [CMFCTabCtrl::SetTabMaxWidth](../vs140/CMFCTabCtrl--SetTabMaxWidth.md) method to set maximum tab width.  
  
## Requirements  
 **Header:** afxtabctrl.h
