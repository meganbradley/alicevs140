# CMFCBaseTabCtrl::GetToolTipCtrl
Retrieves a reference to the tooltip contorl.  
  
## Syntax  
  
```  
CToolTipCtrl& GetToolTipCtrl() const;  
```  
  
## Return Value  
 A reference to the tooltip control.
