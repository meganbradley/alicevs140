# IPropertyPageImpl::m_dwTitle
Stores the resource identifier associated with the text string that appears in the tab for the property page.  
  
## Syntax  
  
```  
  
UINT m_dwTitle;  
  
```  
  
## Requirements  
 **Header:** atlctl.h
