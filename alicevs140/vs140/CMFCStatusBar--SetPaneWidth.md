# CMFCStatusBar::SetPaneWidth
Set the width of the status bar pane.  
  
## Syntax  
  
```  
void SetPaneWidth(  
   int nIndex,  
   int cx   
);  
```  
  
#### Parameters  
 [in] `nIndex`  
 The index of the status bar pane for which to set a new width.  
  
 [in] `cx`  
 The new width of the status bar pane, in pixels.  
  
## Requirements  
 **Header:** afxstatusbar.h
