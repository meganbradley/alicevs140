# CMFCMaskedEdit::DisableMask
Disables validating user input.  
  
## Syntax  
  
```  
void DisableMask();  
```  
  
## Remarks  
 If user input validation is disabled, the masked edit control behaves like the standard edit control.  
  
## Requirements  
 **Header:** afxmaskededit.h
