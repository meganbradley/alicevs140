# CMFCPropertySheet::GetNavBarWidth
Gets the width of the navigation bar.  
  
## Syntax  
  
```  
int GetNavBarWidth() const;  
```  
  
## Return Value  
 The width of the navigation bar in pixels.  
  
## Requirements  
 **Header:** afxpropertysheet.h
