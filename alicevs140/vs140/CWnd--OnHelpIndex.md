# CWnd::OnHelpIndex
Handles the **ID_HELP_INDEX** command and provides a default Help topic.  
  
## Syntax  
  
```  
  
afx_msg void OnHelpIndex( );  
  
```  
  
## Remarks  
 See [CWinApp::OnHelpIndex](../vs140/CWinApp--OnHelpIndex.md) for more information.  
  
## Requirements  
 **Header:** afxwin.h
