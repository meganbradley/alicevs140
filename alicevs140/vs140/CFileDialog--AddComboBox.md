# CFileDialog::AddComboBox
Adds a combo box to the dialog.  
  
## Syntax  
  
```  
HRESULT AddComboBox(  
   DWORD dwIDCtl  
);  
```  
  
#### Parameters  
 `dwIDCtl`  
 The ID of the combo box to add.
