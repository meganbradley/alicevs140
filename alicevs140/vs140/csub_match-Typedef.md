# csub_match Typedef
Type definition for char sub_match.  
  
## Syntax  
  
```  
typedef sub_match<const char*> csub_match;  
```  
  
## Remarks  
 The type describes a specialization of template class [sub_match](../vs140/sub_match-Class.md) for iterators of type `const char*`.  
  
## Requirements  
 **Header:** <regex\>  
  
 **Namespace:** std
