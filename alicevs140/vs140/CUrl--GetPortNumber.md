# CUrl::GetPortNumber
Call this method to get the port number.  
  
## Syntax  
  
```  
  
inline ATL_URL_PORT GetPortNumber( ) const throw( );  
  
```  
  
## Return Value  
 Returns the port number.  
  
## Requirements  
 **Header:** atlutil.h
