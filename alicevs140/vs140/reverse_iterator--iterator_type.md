# reverse_iterator::iterator_type
A type that provides the underlying iterator for a `reverse_iterator`.  
  
## Syntax  
  
```  
typedef RandomIterator iterator_type;  
```  
  
## Remarks  
 The type is a synonym for the template parameter `Iterator`.  
  
## Example  
 See [reverse_iterator::base](../vs140/reverse_iterator--base.md) for an example of how to declare and use `iterator_type`.  
  
## Requirements  
 **Header:** <iterator\>  
  
 **Namespace:** std
