# ceilf Function (fast_math)
Calculates the ceiling of the argument  
  
## Syntax  
  
```  
inline float ceilf(  
   float _X  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
## Return Value  
 Returns the ceiling of the argument  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::fast_math
