# CMFCHeaderCtrl::OnFillBackground
Called by the framework to fill the background of a header control column.  
  
## Syntax  
  
```  
virtual void OnFillBackground(  
   CDC* pDC   
);  
```  
  
#### Parameters  
 [in] `pDC`  
 A pointer to a device context.
