# CDHtmlDialog::IsExternalDispatchSafe
Indicates whether the host's `IDispatch` interface is safe for scripting.  
  
## Syntax  
  
```  
  
virtual BOOL IsExternalDispatchSafe( );  
  
```  
  
## Return Value  
 Returns **FALSE**.  
  
## Requirements  
 **Header:** afxdhtml.h
