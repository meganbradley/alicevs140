# CDialog::SetHelpID
Sets a context-sensitive help ID for the dialog box.  
  
## Syntax  
  
```  
  
      void SetHelpID(  
   UINT nIDR   
);  
```  
  
#### Parameters  
 *nIDR*  
 Specifies the context-sensitive help ID.  
  
## Requirements  
 **Header:** afxwin.h
