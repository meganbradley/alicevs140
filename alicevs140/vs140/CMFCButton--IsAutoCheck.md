# CMFCButton::IsAutoCheck
Indicates whether a check box or radio button is an automatic button.  
  
## Syntax  
  
```  
BOOL IsAutoCheck() const;  
```  
  
## Return Value  
 `TRUE` if the button has style BS_AUTOCHECKBOX or BS_AUTORADIOBUTTON; otherwise, `FALSE`.  
  
## Remarks  
  
## Requirements  
 **Header:** afxbutton.h
