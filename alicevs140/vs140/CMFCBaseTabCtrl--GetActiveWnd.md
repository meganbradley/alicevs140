# CMFCBaseTabCtrl::GetActiveWnd
Retrieves a pointer to the currently active tab window.  
  
## Syntax  
  
```  
virtual CWnd* GetActiveWnd() const;  
```  
  
## Return Value  
 A pointer to a window.  
  
## Requirements  
 **Header:** afxbasetabctrl.h
