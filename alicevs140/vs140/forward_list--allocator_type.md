# forward_list::allocator_type
A type that represents the allocator class for a forward list object.  
  
## Syntax  
  
```  
typedef Allocator allocator_type;  
```  
  
## Remarks  
 `allocator_type` is a synonym for the template parameter Allocator.  
  
## Requirements  
 **Header:** <forward_list>  
  
 **Namespace:** std
