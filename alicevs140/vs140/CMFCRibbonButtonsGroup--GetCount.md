# CMFCRibbonButtonsGroup::GetCount
Returns the number of buttons in the group.  
  
## Syntax  
  
```  
int GetCount() const;  
```  
  
## Return Value  
 The number of buttons in the group.  
  
## Requirements  
 **Header:** afxribbonbuttonsgroup.h
