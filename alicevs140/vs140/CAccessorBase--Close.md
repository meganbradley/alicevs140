# CAccessorBase::Close
Closes the accessors.  
  
## Syntax  
  
```  
  
void Close( );  
  
```  
  
## Remarks  
 You must call [ReleaseAccessors](../vs140/CAccessorBase--ReleaseAccessors.md) first.  
  
## Requirements  
 **Header:** atldbcli.h
