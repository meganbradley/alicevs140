# CMFCRibbonBaseElement::SetToolTipText
Sets the tooltip text for the ribbon element.  
  
## Syntax  
  
```  
virtual void SetToolTipText(  
   LPCTSTR lpszText   
);  
```  
  
#### Parameters  
 [in] `lpszText`  
 The tooltip text.  
  
## Requirements  
 **Header:** afxbaseribbonelement.h
