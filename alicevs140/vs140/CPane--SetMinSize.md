# CPane::SetMinSize
Sets the minimum allowed size for the pane.  
  
## Syntax  
  
```  
void SetMinSize(  
   const CSize& size  
);  
```  
  
#### Parameters  
 [in] `size`  
 A `CSize` object that contains the minimum allowed size for the pane.
