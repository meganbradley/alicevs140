# CShellManager::GetItemSize
Returns the size of an item list.  
  
## Syntax  
  
```  
UINT GetItemSize(  
   LPCITEMIDLIST pidl   
);  
```  
  
#### Parameters  
 [in] `pidl`  
 A pointer to an item list.  
  
## Return Value  
 The size of the item list.  
  
## Requirements  
 **Header:** afxshellmanager.h
