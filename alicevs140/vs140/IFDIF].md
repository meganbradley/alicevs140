# IFDIF]
Grants assembly if the text items are different.  
  
## Syntax  
  
```  
  
IFDIF[[I]] textitem1, textitem2  
```  
  
## Remarks  
 If **I** is given, the comparison is case insensitive. See [IF](../vs140/IF--MASM-.md) for complete syntax.
