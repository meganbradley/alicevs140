# CMFCToolBar::SetHeight
Sets the height of the toolbar.  
  
## Syntax  
  
```  
void SetHeight(  
   int cyHeight   
);  
```  
  
#### Parameters  
 [in] `cyHeight`  
 The height of the toolbar, in pixels.  
  
## Remarks  
 This method redraws the toolbar after it sets the height.  
  
## Requirements  
 **Header:** afxtoolbar.h
