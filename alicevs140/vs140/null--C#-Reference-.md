# null (C# Reference)
The `null` keyword is a literal that represents a null reference, one that does not refer to any object. `null` is the default value of reference-type variables. Ordinary value types cannot be null. However, C# 2.0 introduced nullable value types. See [Nullable Types (C#)](../Topic/Nullable%20Types%20\(C%23%20Programming%20Guide\).md).
