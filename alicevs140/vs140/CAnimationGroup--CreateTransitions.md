# CAnimationGroup::CreateTransitions
A helper that creates COM transition objects.  
  
## Syntax  
  
```  
BOOL CreateTransitions();  
```  
  
## Return Value  
 TRUE is the method succeeds, otherwise FALSE.

 **Header:** afxanimationcontroller.h
