# CMap::GetCount
Retrieves the number of elements in the map.  
  
## Syntax  
  
```  
  
INT_PTR GetCount( ) const;  
```  
  
## Return Value  
 The number of elements.  
  
## Example  
 See the example for [CMap::Lookup](../vs140/CMap--Lookup.md).
