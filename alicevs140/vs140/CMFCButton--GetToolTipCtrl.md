# CMFCButton::GetToolTipCtrl
Returns a reference to the underlying tooltip control.  
  
## Syntax  
  
```  
CToolTipCtrl& GetToolTipCtrl();  
```  
  
## Return Value  
 A reference to the underlying tooltip control.
