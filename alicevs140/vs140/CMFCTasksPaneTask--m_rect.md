# CMFCTasksPaneTask::m_rect
Specifies the bounding rectangle of the task.  
  
## Syntax  
  
```  
CRect m_rect;  
```  
  
## Remarks  
 This value is calculated by the framework when the task is drawn.  
  
## Requirements  
 **Header:** afxTasksPane.h
