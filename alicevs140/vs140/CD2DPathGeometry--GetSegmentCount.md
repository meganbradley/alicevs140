# CD2DPathGeometry::GetSegmentCount
Retrieves the number of segments in the path geometry.  
  
## Syntax  
  
```  
int GetSegmentCount() const;  
```  
  
## Return Value  
 Returns the number of segments in the path geometry.  
  
## Requirements  
 **Header:** afxrendertarget.h
