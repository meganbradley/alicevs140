# CMFCPropertyGridProperty::IsModified
Indicates whether the current property is modified.  
  
## Syntax  
  
```  
BOOL IsModified() const;  
```  
  
## Return Value  
 `TRUE` if the property is modified; otherwise, `FALSE`.  
  
## Remarks  
  
## Requirements  
 **Header:** afxpropertygridctrl.h
