# thread::swap Method
Swaps the object state with that of a specified `thread` object.  
  
## Syntax  
  
```  
void swap(  
   thread& Other  
) _NOEXCEPT;  
```  
  
#### Parameters  
 `Other`  
 A `thread` object.  
  
## Requirements  
 **Header:** thread  
  
 **Namespace:** std
