# CWinApp::UnregisterShellFileTypes
Call this member function to unregister all of your application's document types with the Windows File Manager.  
  
## Syntax  
  
```  
  
void UnregisterShellFileTypes( );  
  
```  
  
## Requirements  
 **Header:** afxwin.h
