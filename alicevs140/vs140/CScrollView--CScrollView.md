# CScrollView::CScrollView
Constructs a `CScrollView` object.  
  
## Syntax  
  
```  
  
CScrollView( );  
```  
  
## Remarks  
 You must call either `SetScrollSizes` or `SetScaleToFitSize` before the scroll view is usable.  
  
## Requirements  
 **Header:** afxwin.h
