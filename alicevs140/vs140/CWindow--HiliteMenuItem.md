# CWindow::HiliteMenuItem
Highlights or removes the highlight from a top-level menu item.  
  
## Syntax  
  
```  
  
      BOOL HiliteMenuItem(  
   HMENU hMenu,  
   UINT uHiliteItem,  
   UINT uHilite   
) throw();  
```
