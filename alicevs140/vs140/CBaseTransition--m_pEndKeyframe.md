# CBaseTransition::m_pEndKeyframe
Stores a pointer to the keyframe that specifies the end of the transition.  
  
## Syntax  
  
```  
CBaseKeyFrame* m_pEndKeyframe;  
```  
  
## Requirements  
 **Header:** afxanimationcontroller.h
