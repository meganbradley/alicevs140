# CCtrlView::m_dwDefaultStyle
Contains the default style for the view class.  
  
## Syntax  
  
```  
  
DWORD m_dwDefaultStyle;  
  
```  
  
## Remarks  
 This style is applied when a window is created.  
  
## Requirements  
 **Header:** afxwin.h
