# overwrite_buffer::has_value Method
Checks whether this `overwrite_buffer` messaging block has a value yet.  
  
## Syntax  
  
```  
bool has_value() const;  
```  
  
## Return Value  
 `true` if the block has received a value, `false` otherwise.  
  
## Requirements  
 **Header:** agents.h  
  
 **Namespace:** concurrency
