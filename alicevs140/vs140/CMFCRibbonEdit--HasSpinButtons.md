# CMFCRibbonEdit::HasSpinButtons
Indicates whether the text box has a spin button.  
  
## Syntax  
  
```  
virtual BOOL HasSpinButtons() const;  
```  
  
## Return Value  
 `TRUE` if the text box has a spin button; otherwise `FALSE`.

## Requirements  
 **Header:** afxribbonedit.h
