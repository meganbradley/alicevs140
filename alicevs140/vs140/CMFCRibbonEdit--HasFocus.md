# CMFCRibbonEdit::HasFocus
Indicates whether the [CMFCRibbonEdit Class](../vs140/CMFCRibbonEdit-Class.md) control has the focus.  
  
## Syntax  
  
```  
virtual BOOL HasFocus() const;  
```  
  
## Return Value  
 `TRUE` if the [CMFCRibbonEdit Class](../vs140/CMFCRibbonEdit-Class.md) control has the focus; otherwise `FALSE`.

 **Header:** afxribbonedit.h
