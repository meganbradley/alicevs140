# CFrameWndEx::OnDestroy
Called by the framework when the frame is destroyed.  
  
## Syntax  
  
```  
afx_msg void OnDestroy();  
```  
  
## Remarks  
 The accelerator table and all windows are destroyed.  
  
## Requirements  
 **Header:** afxframewndex.h
