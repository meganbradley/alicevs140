# cos Function
Calculates the cosine of the argument  
  
## Syntax  
  
```  
inline float cos(  
   float _X  
) restrict(amp);  
inline double cos(  
   double _X  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
## Return Value  
 Returns the cosine value of the argument  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::precise_math
