# C6551
warning C6551: Invalid size specification: expression not parsable  
  
> [!NOTE]
>  This warning occurs only in code that is using a deprecated version of the source-code annotation language (SAL). We recommend that you port your code to use the latest version of SAL. For more information, see [Using SAL Annotations to Reduce C/C++ Code Defects](../vs140/Using-SAL-Annotations-to-Reduce-C-C---Code-Defects.md).
