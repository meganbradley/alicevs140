# CSpinButtonCtrl::GetBase
Retrieves the current base for a spin button control.  
  
## Syntax  
  
```  
  
UINT GetBase( ) const;  
```  
  
## Return Value  
 The current base value.  
  
## Requirements  
 **Header:** afxcmn.h
