# basic_streambuf::gptr
A protected function that returns a pointer to the next element of the input buffer.  
  
## Syntax  
  
```  
  
char  
_  
type *gptr( ) const;  
  
```  
  
## Return Value  
 A pointer to the next element of the input buffer.  
  
## Requirements  
 **Header:** <streambuf\>  
  
 **Namespace:** std
