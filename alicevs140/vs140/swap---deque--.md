# swap (&lt;deque&gt;)
Exchanges the elements of two deques.  
  
## Syntax  
  
```  
void swap(  
   deque<Type, Allocator>& _Left,  
   deque<Type, Allocator>& _Right,  
);  
```  
  
#### Parameters  
 `_Left`  
 An object of type `deque`.  
  
 `_Right`  
 An object of type `deque`.
