# rsqrtf Function (fast_math)
Returns the reciprocal of the square root of the argument  
  
## Syntax  
  
```  
inline float rsqrtf(  
   float _X  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
## Return Value  
 Returns the reciprocal of the square root of the argument  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::fast_math
