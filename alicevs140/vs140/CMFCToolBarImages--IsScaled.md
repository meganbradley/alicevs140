# CMFCToolBarImages::IsScaled
Tells whether the underlined images are scaled or not.  
  
## Syntax  
  
```  
BOOL IsScaled () const;  
```  
  
## Return Value  
 `TRUE` if underlined images are scaled; otherwise `FALSE`.

## Requirements  
 **Header:** afxtoolbarimages.h
