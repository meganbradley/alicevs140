# CDC::GetViewportExt
Retrieves the x- and y-extents of the device context's viewport.  
  
## Syntax  
  
```  
  
CSize GetViewportExt( ) const;  
```  
  
## Return Value  
 The x- and y-extents (in device units) as a `CSize` object.  
  
## Requirements  
 **Header:** afxwin.h
