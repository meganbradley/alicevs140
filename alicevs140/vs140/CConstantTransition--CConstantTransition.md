# CConstantTransition::CConstantTransition
Constructs a transition object and initializes its duration.  
  
## Syntax  
  
```  
CConstantTransition (  
   UI_ANIMATION_SECONDS duration  
);  
```  
  
#### Parameters  
 `duration`  
 The duration of the transition.  
  
## Requirements  
 **Header:** afxanimationcontroller.h
