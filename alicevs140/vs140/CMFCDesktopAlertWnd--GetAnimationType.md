# CMFCDesktopAlertWnd::GetAnimationType
Returns the animation type.  
  
## Syntax  
  
```  
CMFCPopupMenu::ANIMATION_TYPE GetAnimationType();  
```  
  
## Return Value  
 One of the following animation types:  
  
-   `NO_ANIMATION`  
  
-   `UNFOLD`  
  
-   `SLIDE`  
  
-   `FADE`  
  
-   `SYSTEM_DEFAULT_ANIMATION`  
  
## Requirements  
 **Header:** afxDesktopAlertWnd.h
