# CMFCHeaderCtrl::RemoveSortColumn
Removes the specified column from the list of sort columns.  
  
## Syntax  
  
```  
void RemoveSortColumn(  
   int iColumn   
);  
```  
  
#### Parameters  
 [in] `iColumn`  
 The zero-based index of the column to remove.  
  
## Requirements  
 **Header:** afxheaderctrl.h
