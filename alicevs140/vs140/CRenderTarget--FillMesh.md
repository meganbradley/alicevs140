# CRenderTarget::FillMesh
Paints the interior of the specified mesh.  
  
## Syntax  
  
```  
void FillMesh(  
   CD2DMesh* pMesh,  
   CD2DBrush* pBrush  
);  
```  
  
#### Parameters  
 `pMesh`  
 The mesh to paint.  
  
 `pBrush`  
 The brush used to paint the mesh.  
  
## Requirements  
 **Header:** afxrendertarget.h
