# CPictureHolder::SetPictureDispatch
Connects the `CPictureHolder` object to a `IPictureDisp` interface.  
  
## Syntax  
  
```  
  
      void SetPictureDispatch(  
   LPPICTUREDISP pDisp   
);  
```  
  
#### Parameters  
 `pDisp`  
 Pointer to the new `IPictureDisp` interface.  
  
## Requirements  
 **Header:** afxwin.h
