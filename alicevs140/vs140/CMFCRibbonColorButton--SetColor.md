# CMFCRibbonColorButton::SetColor
Selects a color from the regular color area.  
  
## Syntax  
  
```  
void SetColor(  
   COLORREF color   
);  
```  
  
#### Parameters  
 [in] `color`  
 A color to set.  
  
## Requirements  
 **Header:** afxribboncolorbutton.h
