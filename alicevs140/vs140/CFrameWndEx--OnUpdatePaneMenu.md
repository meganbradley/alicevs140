# CFrameWndEx::OnUpdatePaneMenu
Called by the framework to update the pane menu.  
  
## Syntax  
  
```  
afx_msg void OnUpdatePaneMenu(  
   CCmdUI* pCmdUI  
);  
```  
  
#### Parameters  
 [in] `pCmdUI`  
 Pointer to the pane user interface object.
