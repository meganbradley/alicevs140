# COlePropertyPage::SetPageName
Sets the property page's name, which the property frame will typically display on the page's tab.  
  
## Syntax  
  
```  
  
      void SetPageName(  
   LPCTSTR lpszPageName   
);  
```  
  
#### Parameters  
 *lpszPageName*  
 Pointer to a string containing the property page's name.  
  
## Requirements  
 **Header:** afxctl.h
