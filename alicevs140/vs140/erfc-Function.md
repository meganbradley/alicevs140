# erfc Function
Computes the complementary error function of _X  
  
## Syntax  
  
```  
inline float erfc(  
   float _X  
) restrict(amp);  
inline double erfc(  
   double _X  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
## Return Value  
 Returns the complementary error function of _X  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::precise_math
