# probitf Function
Returns the inverse cumulative distribution function of the argument  
  
## Syntax  
  
```  
inline float probitf(  
   float _X  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
## Return Value  
 Returns the inverse cumulative distribution function of the argument  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::precise_math
