# CSplitterWnd::IsTracking
Call this member function to determine if the splitter bar in the window is currently being moved.  
  
## Syntax  
  
```  
  
BOOL IsTracking( );  
  
```  
  
## Return Value  
 Nonzero if a splitter operation is in progress; otherwise 0.  
  
## Requirements  
 **Header:** afxext.h
