# CD2DMesh::Get
Returns ID2D1Mesh interface  
  
## Syntax  
  
```  
ID2D1Mesh* Get();  
```  
  
## Return Value  
 Pointer to an ID2D1Mesh interface or NULL if object is not initialized yet.  
  
## Requirements  
 **Header:** afxrendertarget.h
