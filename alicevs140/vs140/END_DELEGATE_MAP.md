# END_DELEGATE_MAP
Ends a delegate map.  
  
## Syntax  
  
```  
END_DELEGATE_MAP();  
```  
  
## Remarks  
 This macro marks the end of a list of delegate entries, which compose a delegate map. For an example of how this macro is used, see [EVENT_DELEGATE_ENTRY](../vs140/EVENT_DELEGATE_ENTRY.md).  
  
## Requirements  
 **Header:** msclr\event.h
