# CAcl::IsNull
Returns the status of the `CAcl` object.  
  
## Syntax  
  
```  
  
bool IsNull( ) const throw( );  
  
```  
  
## Return Value  
 Returns **true** if the `CAcl` object is NULL, **false** otherwise.  
  
## Requirements  
 **Header:** atlsecurity.h
