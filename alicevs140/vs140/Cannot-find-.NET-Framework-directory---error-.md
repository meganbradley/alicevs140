# Cannot find .NET Framework directory: &lt;error&gt;
The directory where the [!INCLUDE[dnprdnshort](../vs140/includes/dnprdnshort_md.md)] assemblies are stored could not be located.  
  
 **Error ID:** BC31508  
  
### To correct this error  
  
1.  Compile the program again to see if the error recurs.  
  
2.  If the error continues, save your work and restart [!INCLUDE[vsprvs](../vs140/includes/vsprvs_md.md)].  
  
3.  If the error recurs, reinstall [!INCLUDE[vbprvb](../vs140/includes/vbprvb_md.md)].  
  
4.  If the error persists after reinstallation, notify Microsoft Product Support Services.
