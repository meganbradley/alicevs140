# CDC::FlattenPath
Transforms any curves in the path selected into the current device context, and turns each curve into a sequence of lines.  
  
## Syntax  
  
```  
  
BOOL FlattenPath( );  
```  
  
## Return Value  
 Nonzero if the function is successful; otherwise 0.  
  
## Requirements  
 **Header:** afxwin.h
