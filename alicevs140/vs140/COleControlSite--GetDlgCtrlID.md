# COleControlSite::GetDlgCtrlID
Retrieves the identifier of the control.  
  
## Syntax  
  
```  
  
virtual int GetDlgCtrlID( ) const;  
  
```  
  
## Return Value  
 The dialog item identifier of the control.  
  
## Requirements  
 **Header:** afxocc.h
