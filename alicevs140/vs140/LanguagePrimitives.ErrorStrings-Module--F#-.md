# LanguagePrimitives.ErrorStrings Module (F#)
Contains strings that are used for certain error messages. For internal use only.  
  
 **Namespace/Module Path:** Microsoft.FSharp.Core.LanguagePrimitives  
  
 **Assembly:** FSharp.Core (in FSharp.Core.dll)  
  
## Syntax  
  
```  
module ErrorStrings  
```  
  
## Remarks  
  
## Values  
  
|Value|Description|  
|-----------|-----------------|  
|[AddressOpNotFirstClassString](../vs140/ErrorStrings.AddressOpNotFirstClassString-Function--F#-.md)|Gets a string describing that first class uses of the address-of operators are not permitted.|  
|[InputArrayEmptyString](../vs140/ErrorStrings.InputArrayEmptyString-Function--F#-.md)|Gets a string describing that the input array was empty.|  
|[InputMustBeNonNegativeString](../vs140/ErrorStrings.InputMustBeNonNegativeString-Function--F#-.md)|Gets a string describing that the input must be non-negative.|  
|[InputSequenceEmptyString](../vs140/ErrorStrings.InputSequenceEmptyString-Function--F#-.md)|Gets a string describing that the input sequence was empty.|  
|[NoNegateMinValueString](../vs140/ErrorStrings.NoNegateMinValueString-Function--F#-.md)|Gets a string describing that negating the minimum value of a twos complement number is invalid.|
