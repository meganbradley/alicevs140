# basic_string::back
Returns a reference to the last element in the string.  
  
## Syntax  
  
```  
const_reference back() const;  
reference back();  
```  
  
## Return Value  
 A reference to the last element of the string, which must be non-empty.  
  
## Remarks  
  
## Requirements  
 **Header:** <string\>  
  
 **Namespace:** std
