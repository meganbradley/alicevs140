# CMFCToolTipInfo::m_bDrawIcon
Specifies whether all tooltips display icons.  
  
## Syntax  
  
```  
BOOL m_bDrawIcon;  
```  
  
## Remarks  
 Set this member to `TRUE` to display an icon on each tooltip, or `FALSE` to display tooltips without icons.  
  
## Requirements  
 **Header:** afxToolTipCtrl.h
