# CListView::RemoveImageList
Removes the specified image list from the list view.  
  
## Syntax  
  
```  
  
      void RemoveImageList(   
   int nImageList   
);  
```  
  
#### Parameters  
 `nImageList`  
 The zero-based index of the image to remove.  
  
## Requirements  
 **Header:** afxcview.h
