# CMFCCaptionBar::SetButtonPressed
Specifies whether the button stays pressed.  
  
## Syntax  
  
```  
void SetButtonPressed(  
   BOOL bPresed=TRUE   
);  
```  
  
#### Parameters  
 `bPresed`  
 `TRUE` if the button keeps its pressed state, `FALSE` otherwise.  
  
## Requirements  
 **Header:** afxcaptionbar.h
