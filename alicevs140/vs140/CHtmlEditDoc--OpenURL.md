# CHtmlEditDoc::OpenURL
Opens a URL.  
  
## Syntax  
  
```  
  
      virtual BOOL OpenURL(  
   LPCTSTR lpszURL   
);  
```  
  
#### Parameters  
 `lpszURL`  
 The URL to open.  
  
## Return Value  
 Returns **TRUE** on success, **FALSE** on failure.  
  
## Requirements  
 **Header:** afxhtml.h
