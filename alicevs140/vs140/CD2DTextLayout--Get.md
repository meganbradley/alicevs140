# CD2DTextLayout::Get
Returns IDWriteTextLayout interface  
  
## Syntax  
  
```  
IDWriteTextLayout* Get();  
```  
  
## Return Value  
 Pointer to an IDWriteTextLayout interface or NULL if object is not initialized yet.  
  
## Requirements  
 **Header:** afxrendertarget.h
