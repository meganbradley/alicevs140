# CMFCPropertyGridToolTipCtrl::Create
Creates a window for the tooltip control.  
  
## Syntax  
  
```  
BOOL Create(  
   CWnd* pWndParent  
);  
```  
  
#### Parameters  
 [in] `pWndParent`  
 A pointer to the parent window.  
  
## Return Value  
 TRUE if the window was successfully created; otherwise, FALSE.  
  
## Requirements  
 **Header:** afxpropertygridtooltipctrl.h
