# CMFCShellListCtrl::EnableShellContextMenu
Enables the shortcut menu.  
  
## Syntax  
  
```  
void EnableShellContextMenu(  
   BOOL bEnable = TRUE  
);  
```  
  
#### Parameters  
 [in] `bEnable`  
 A Boolean that specifies whether the framework enables the shortcut menu.  
  
## Requirements  
 **Header:** afxshelllistctrl.h
