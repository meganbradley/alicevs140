# CUrl::GetUrlPathLength
Call this method to get the URL path length.  
  
## Syntax  
  
```  
  
inline DWORD GetUrlPathLength( ) const throw( );  
  
```  
  
## Return Value  
 Returns the URL path length.  
  
## Requirements  
 **Header:** atlutil.h
