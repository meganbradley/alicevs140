# CFileDialog::EndVisualGroup
Stops the addition of elements to a visual group in the dialog.  
  
## Syntax  
  
```  
HRESULT EndVisualGroup();  
```  
  
## Return Value  
 Returns S_OK if successful; an error value otherwise.
