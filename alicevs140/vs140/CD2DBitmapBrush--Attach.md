# CD2DBitmapBrush::Attach
Attaches existing resource interface to the object  
  
## Syntax  
  
```  
void Attach(  
   ID2D1BitmapBrush* pResource  
);  
```  
  
#### Parameters  
 `pResource`  
 Existing resource interface. Cannot be NULL  
  
## Requirements  
 **Header:** afxrendertarget.h
