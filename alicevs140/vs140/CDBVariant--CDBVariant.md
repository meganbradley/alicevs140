# CDBVariant::CDBVariant
Creates a NULL `CDBVariant` object.  
  
## Syntax  
  
```  
  
CDBVariant( );  
  
```  
  
## Remarks  
 Sets the [m_dwType](../vs140/CDBVariant--m_dwType.md) data member to **DBVT_NULL**.  
  
## Requirements  
 **Header:** afxdb.h
