# CSnapInPropertyPageImpl::CSnapInPropertyPageImpl
Constructs a `CSnapInPropertyPageImpl` object.  
  
## Syntax  
  
```  
  
      CSnapInPropertyPageImpl(  
   LPCTSTR lpszTitle = NULL   
);  
```  
  
#### Parameters  
 `lpszTitle`  
 [in] The title of the property page.
