# CommandName Element
The `CommandName` element specifies the text that appears in the keyboard category in the **Options** dialog box, and in the **Commands** list in the **Customize** dialog box.  
  
## Syntax  
  
```  
<CommandName>MyCommand</CommandName>  
```  
  
## Attributes and Elements  
 The following sections describe attributes, child elements, and parent elements.  
  
### Attributes  
 None.  
  
### Child Elements  
 None.  
  
### Parent Elements  
  
|Element|Description|  
|-------------|-----------------|  
|[Strings Element](../vs140/Strings-Element.md)|Groups text elements, such as `ButtonText` and `CommandName`.|
