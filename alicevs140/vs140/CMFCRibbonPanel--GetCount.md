# CMFCRibbonPanel::GetCount
Retrieves the number of ribbon elements that are contained in the ribbon panel.  
  
## Syntax  
  
```  
int GetCount() const;  
```  
  
## Return Value  
 The number of ribbon elements that are contained in the ribbon panel.  
  
## Requirements  
 **Header:** afxRibbonPanel.h
