# CMFCRibbonBaseElement::GetParentCategory
Retrieves the ribbon category for the ribbon element.  
  
## Syntax  
  
```  
CMFCRibbonCategory* GetParentCategory() const;  
```  
  
## Return Value  
 A pointer to the ribbon category.
