# CMFCPropertyGridCtrl::IsDescriptionArea
Indicates whether the description area of the property grid control is displayed.  
  
## Syntax  
  
```  
BOOL IsDescriptionArea() const;  
```  
  
## Return Value  
 `TRUE` if the description area is displayed; otherwise, `FALSE`.  
  
## Remarks  
 Use the [CMFCPropertyGridCtrl::EnableDescriptionArea](../vs140/CMFCPropertyGridCtrl--EnableDescriptionArea.md) method to hide or display the description area.  
  
## Requirements  
 **Header:** afxpropertygridctrl.h
