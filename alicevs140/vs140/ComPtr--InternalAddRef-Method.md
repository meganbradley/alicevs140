# ComPtr::InternalAddRef Method
Increments the reference count of the interface associated with this ComPtr.  
  
## Syntax  
  
```  
void InternalAddRef() const;  
```  
  
## Remarks  
 This method is protected.  
  
## Requirements  
 **Header:** client.h  
  
 **Namespace:** Microsoft::WRL
