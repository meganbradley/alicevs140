# IDispEventImpl::GetTypeInfoCount
Retrieves the number of type information interfaces that an object provides (either 0 or 1).  
  
## Syntax  
  
```  
  
      STDMETHOD(GetTypeInfoCount)(  
   UINT* pctinfo   
);  
```
