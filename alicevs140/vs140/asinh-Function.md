# asinh Function
Calculates the inverse hyperbolic sine of the argument  
  
## Syntax  
  
```  
inline float asinh(  
   float _X  
) restrict(amp);  
inline double asinh(  
   double _X  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
## Return Value  
 Returns the inverse hyperbolic sine value of the argument  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::precise_math
