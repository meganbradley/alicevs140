# CFrameWnd::GetTitle
Retrieves the title of the window object.  
  
## Syntax  
  
```  
  
CString GetTitle( ) const;  
  
```  
  
## Return Value  
 A [CString](../vs140/CStringT-Class.md) object containing the current title of the window object.  
  
## Requirements  
 **Header:** afxwin.h
