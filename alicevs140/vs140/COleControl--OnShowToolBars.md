# COleControl::OnShowToolBars
Called by the framework when the control has been UI activated.  
  
## Syntax  
  
```  
  
virtual void OnShowToolBars( );  
```  
  
## Remarks  
 The default implementation does nothing.  
  
## Requirements  
 **Header:** afxctl.h
