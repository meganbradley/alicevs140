# CMFCTasksPane::GetGroupCount
Returns the total number of groups.  
  
## Syntax  
  
```  
int GetGroupCount() const;  
```  
  
## Return Value  
 The total number of groups in the task pane.  
  
## Requirements  
 **Header:** afxTasksPane.h
