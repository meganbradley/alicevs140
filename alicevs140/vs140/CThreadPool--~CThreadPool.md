# CThreadPool::~CThreadPool
The destructor for the thread pool.  
  
## Syntax  
  
```  
  
~CThreadPool( ) throw( );  
  
```  
  
## Remarks  
 Calls [CThreadPool::Shutdown](../vs140/CThreadPool--Shutdown.md).  
  
## Requirements  
 **Header:** atlutil.h
