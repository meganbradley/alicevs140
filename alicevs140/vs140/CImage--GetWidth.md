# CImage::GetWidth
Retrieves the width, in pixels, of an image.  
  
## Syntax  
  
```  
  
int GetWidth( ) const throw( );  
  
```  
  
## Return Value  
 The width of the bitmap, in pixels.  
  
## Requirements  
 **Header:** atlimage.h
