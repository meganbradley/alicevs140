# CMFCColorPopupMenu::SetPropList
Sets the property grid control object of the embedded `CMFCColorBar` object.  
  
## Syntax  
  
```  
void SetPropList(  
   CMFCPropertyGridCtrl* pWndList  
);  
```  
  
#### Parameters  
 [in] `pWndList`  
 Pointer to a property grid control object.  
  
## Requirements  
 **Header:** afxcolorpopupmenu.h
