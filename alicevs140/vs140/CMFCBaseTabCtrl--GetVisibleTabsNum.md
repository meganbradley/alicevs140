# CMFCBaseTabCtrl::GetVisibleTabsNum
Retrieves the number of currently visible tabs.  
  
## Syntax  
  
```  
virtual int GetVisibleTabsNum() const;  
```  
  
## Return Value  
 The number of visible tabs.  
  
## Requirements  
 **Header:** afxbasetabctrl.h
