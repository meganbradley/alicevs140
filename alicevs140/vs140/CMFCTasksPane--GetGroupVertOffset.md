# CMFCTasksPane::GetGroupVertOffset
Returns the vertical offset of a group.  
  
## Syntax  
  
```  
int GetGroupVertOffset() const;  
```  
  
## Return Value  
 The vertical offset of a group, in pixels.  
  
## Requirements  
 **Header:** afxTasksPane.h
