# SERVICE_ENTRY
Indicates that the object supports the service id specified by *SID*.  
  
## Syntax  
  
```  
  
      SERVICE_ENTRY(   
   SID    
)  
```  
  
#### Parameters  
 *SID*  
 The service ID.

 See the example for [BEGIN_SERVICE_MAP](../vs140/BEGIN_SERVICE_MAP.md).
