# CRectTracker::m_nHandleSize
The size, in pixels, of the `CRectTracker` resize handles.  
  
## Syntax  
  
```  
  
int m_nHandleSize;  
  
```  
  
## Remarks  
 Initialized with the default system value.  
  
## Requirements  
 **Header:** afxext.h
