# accelerator::device_path Data Member
Gets the path of the accelerator. The path is unique on the system.  
  
## Syntax  
  
```  
__declspec(property(get=get_device_path)) std::wstring device_path;  
```  
  
## Requirements  
 **Header:** amprt.h  
  
 **Namespace:** Concurrency
