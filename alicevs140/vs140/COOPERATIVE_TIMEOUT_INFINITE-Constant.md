# COOPERATIVE_TIMEOUT_INFINITE Constant
Value indicating that a wait should never time out.  
  
## Syntax  
  
```  
const unsigned int COOPERATIVE_TIMEOUT_INFINITE = (unsigned int)-1;  
```  
  
## Requirements  
 **Header:** concrt.h  
  
 **Namespace:** concurrency
