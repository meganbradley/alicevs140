# CD2DSolidColorBrush::GetColor
Retrieves the color of the solid color brush  
  
## Syntax  
  
```  
D2D1_COLOR_F GetColor() const;  
```  
  
## Return Value  
 The color of this solid color brush  
  
## Requirements  
 **Header:** afxrendertarget.h
