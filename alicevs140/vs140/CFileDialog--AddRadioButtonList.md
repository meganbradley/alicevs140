# CFileDialog::AddRadioButtonList
Adds an option button (also known as radio button) group to the dialog.  
  
## Syntax  
  
```  
HRESULT AddRadioButtonList(  
   DWORD dwIDCtl  
);  
```  
  
#### Parameters  
 `dwIDCtl`  
 The ID of the option button group to add.
