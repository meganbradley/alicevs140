# tgamma Function
Computes the gamma function of _X  
  
## Syntax  
  
```  
inline float tgamma(  
   float _X  
) restrict(amp);  
inline double tgamma(  
   double _X  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
## Return Value  
 Returns the result of gamma function of _X  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::precise_math
