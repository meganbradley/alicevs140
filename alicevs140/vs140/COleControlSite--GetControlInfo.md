# COleControlSite::GetControlInfo
Retrieves information about a control's keyboard mnemonics and keyboard behavior.  
  
## Syntax  
  
```  
  
void GetControlInfo( );  
  
```  
  
## Remarks  
 The information is stored in [COleControlSite::m_ctlInfo](../vs140/COleControlSite--m_ctlInfo.md).  
  
## Requirements  
 **Header:** afxocc.h
