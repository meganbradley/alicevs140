# CAxWindow::GetWndClassName
Retrieves the name of the window class.  
  
## Syntax  
  
```  
  
static LPCTSTR GetWndClassName( );  
  
```  
  
## Return Value  
 A pointer to a string containing the name of the window class that can host nonlicensed ActiveX controls.  
  
## Requirements  
 **Header:** atlwin.h
