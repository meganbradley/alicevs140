# fmod Function (fast_math)
Calculates the floating-point remainder of _X/_Y  
  
## Syntax  
  
```  
inline float fmod(  
   float _X,  
   float _Y  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
 `_Y`  
 Floating-point value  
  
## Return Value  
 Returns the floating-point remainder of _X/_Y  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::fast_math
