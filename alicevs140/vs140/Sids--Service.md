# Sids::Service
Returns the SECURITY_SERVICE_RID SID.  
  
> [!IMPORTANT]
>  This function cannot be used in applications that execute in the [!INCLUDE[wrt](../vs140/includes/wrt_md.md)].  
  
## Syntax  
  
```  
  
CSid Service( ) throw(...);  
  
```

 **Header:** atlsecurity.h
