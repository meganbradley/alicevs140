# Scheduler::~Scheduler Destructor
An object of the `Scheduler` class is implicitly destroyed when all external references to it cease to exist.  
  
## Syntax  
  
```  
virtual ~Scheduler();  
```  
  
## Requirements  
 **Header:** concrt.h  
  
 **Namespace:** concurrency
