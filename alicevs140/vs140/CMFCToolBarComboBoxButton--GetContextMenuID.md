# CMFCToolBarComboBoxButton::GetContextMenuID
Gets the shortcut menu resource ID for the combo box button.  
  
## Syntax  
  
```  
UINT GetContextMenuID();  
```  
  
## Return Value  
 The shortcut menu resource ID.  
  
## Requirements  
 **Header:** afxtoolbarcomboboxbutton.h
