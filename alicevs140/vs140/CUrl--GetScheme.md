# CUrl::GetScheme
Call this method to get the URL scheme.  
  
## Syntax  
  
```  
  
inline ATL_URL_SCHEME GetScheme( ) const throw( );  
  
```  
  
## Return Value  
 Returns the [ATL_URL_SCHEME](../vs140/ATL_URL_SCHEME.md) value describing the scheme of the URL.  
  
## Requirements  
 **Header:** atlutil.h
