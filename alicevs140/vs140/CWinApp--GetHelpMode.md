# CWinApp::GetHelpMode
Retrieves the type of help used by the application.  
  
## Syntax  
  
```  
  
AFX_HELP_TYPE GetHelpMode( );  
  
```  
  
## Return Value  
 The help type used by the application. See [CWinApp::m_eHelpType](../vs140/CWinApp--m_eHelpType.md) for more information.  
  
## Requirements  
 **Header:** afxwin.h
