# CPaneFrameWnd::OnBeforeDock
Determines if docking is possible.  
  
## Syntax  
  
```  
virtual BOOL OnBeforeDock();  
```  
  
## Return Value  
 `TRUE` if docking is possible; otherwise, `FALSE`.  
  
## Requirements  
 **Header:** afxpaneframewnd.h
