# Compiler Warning (level 1) C4821
Unable to determine Unicode encoding type, please save the file with signature (BOM)  
  
 The compiler could not determine the encoding type for a file. To resolve this warning, save the file with a byte order marker. See [Managing Files with Encoding](assetId:///ab3a74da-5c8f-4680-8882-f1def4909e82) for more information.
