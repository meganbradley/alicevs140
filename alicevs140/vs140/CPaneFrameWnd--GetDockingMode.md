# CPaneFrameWnd::GetDockingMode
Returns the docking mode.  
  
## Syntax  
  
```  
virtual AFX_DOCK_TYPE GetDockingMode() const;  
```  
  
## Return Value  
 The docking mode. One of the following values:  
  
-   `DT_STANDARD`  
  
-   `DT_IMMEDIATE`  
  
-   `DT_SMART`  
  
## Requirements  
 **Header:** afxpaneframewnd.h
