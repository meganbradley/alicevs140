# CAcl::SetEmpty
Marks the `CAcl` object as empty.  
  
## Syntax  
  
```  
  
void SetEmpty( ) throw( );  
  
```  
  
## Remarks  
 The `CAcl` can be set to empty or to NULL: the two states are distinct.  
  
## Requirements  
 **Header:** atlsecurity.h
