# Linker Tools Warning LNK4220
invalid 'linker option' value 'value'; assumed default  
  
 An out-of-range value was specified with the [/TLBID](../Topic/-TLBID%20\(Specify%20Resource%20ID%20for%20TypeLib\).md) option. The default value for **/TLBID** is 1.
