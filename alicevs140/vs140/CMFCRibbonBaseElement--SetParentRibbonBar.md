# CMFCRibbonBaseElement::SetParentRibbonBar
Sets the parent ribbon bar for the ribbon element.  
  
## Syntax  
  
```  
virtual void SetParentRibbonBar(  
   CMFCRibbonBar* pRibbonBar  
);  
```  
  
#### Parameters  
 [in] `pRibbonBar`  
 Pointer to the parent ribbon bar.
