# CMDIChildWndEx::AdjustDockingLayout
Adjusts the docking layout.  
  
## Syntax  
  
```  
virtual void AdjustDockingLayout(  
   HDWP hdwp = NULL  
);  
```  
  
#### Parameters  
 [in] `hdwp`  
 Handle to a deferred window position structure.  
  
## Requirements  
 **Header:** afxmdichildwndex.h
