# CToolBarCtrl::SetIndent
Sets the indentation for the first button in a toolbar control.  
  
## Syntax  
  
```  
  
      BOOL SetIndent(  
   int iIndent   
);  
```  
  
#### Parameters  
 *iIndent*  
 The value specifying the indentation, in pixels.  
  
## Return Value  
 Nonzero if successful; otherwise zero.  
  
## Requirements  
 **Header:** afxcmn.h
