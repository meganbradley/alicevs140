# CDHtmlDialog::SetHostFlags
Sets the host UI flags.  
  
## Syntax  
  
```  
  
      void SetHostFlags(  
   DWORD dwFlags   
);  
```  
  
#### Parameters  
 `dwFlags`  
 For possible values, see [DOCHOSTUIFLAG](https://msdn.microsoft.com/en-us/library/aa753277.aspx) in the [!INCLUDE[winSDK](../vs140/includes/winSDK_md.md)].
