# probit Function
Returns the inverse cumulative distribution function of the argument  
  
## Syntax  
  
```  
inline float probit(  
   float _X  
) restrict(amp);  
inline double probit(  
   double _X  
) restrict(amp);  
```  
  
#### Parameters  
 `_X`  
 Floating-point value  
  
## Return Value  
 Returns the inverse cumulative distribution function of the argument  
  
## Requirements  
 **Header:** amp_math.h  
  
 **Namespace:** Concurrency::precise_math
