# CAtlModule::GetLockCount
Returns the lock count.  
  
## Syntax  
  
```  
  
virtual LONG GetLockCount( ) throw( );  
  
```  
  
## Return Value  
 Returns the lock count. This value may be useful for diagnostics and debugging.  
  
## Requirements  
 **Header:** atlbase.h
