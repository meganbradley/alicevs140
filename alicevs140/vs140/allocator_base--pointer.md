# allocator_base::pointer
A type that provides a pointer to the type of object managed by the allocator.  
  
## Syntax  
  
```  
typedef Type *pointer;  
```  
  
## Remarks  
 The pointer type implements  as a typedef for `Type``*`.  
  
## Requirements  
 **Header:** <allocators\>  
  
 **Namespace:** stdext
