# CDockingManager::GetMiniFrames
Gets a list of miniframes.  
  
## Syntax  
  
```  
const CObList& GetMiniFrames() const;  
```  
  
## Return Value  
 A list of miniframes that contain the control bars that belong to the docking manager.  
  
## Requirements  
 **Header:** afxdockingmanager.h
