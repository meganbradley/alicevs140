# CMFCTasksPane::AddPage
Adds a page to the task pane.  
  
## Syntax  
  
```  
int AddPage(  
    LPCTSTR lpszPageLabel  
);  
```  
  
#### Parameters  
 [in] `lpszPageLabel`  
 Specifies the label for the page.  
  
## Return Value  
 The zero-based index of the new page.  
  
## Requirements  
 **Header:** afxTasksPane.h
