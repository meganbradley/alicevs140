# CHtmlView::OnQuit
This member function is called by the framework to notify an application that the Internet Explorer application is ready to quit.  
  
## Syntax  
  
```  
  
virtual void OnQuit( );  
  
```  
  
## Requirements  
 **Header:** afxhtml.h
