# CPaintDC::m_hWnd
The `HWND` to which this `CPaintDC` object is attached.  
  
## Syntax  
  
```  
  
HWND m_hWnd;  
  
```  
  
## Remarks  
 `m_hWnd` is a protected variable of type `HWND`.
