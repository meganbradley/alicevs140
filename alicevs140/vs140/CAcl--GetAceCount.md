# CAcl::GetAceCount
Returns the number of access-control entry (ACE) objects.  
  
## Syntax  
  
```  
  
virtual UINT GetAceCount( ) const throw( ) = 0;  
  
```  
  
## Return Value  
 Returns the number of ACE entries in the `CAcl` object.  
  
## Requirements  
 **Header:** atlsecurity.h
