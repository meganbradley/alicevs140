# ios
Supports the ios class from the old iostream library.  
  
## Syntax  
  
```  
  
typedef basic  
_  
ios<char, char  
_  
traits<char> > ios;  
  
```  
  
## Remarks  
 The type is a synonym for template class [basic_ios](../vs140/basic_ios-Class.md), specialized for elements of type `char` with default character traits.  
  
## Requirements  
 **Header:** <ios\>  
  
 **Namespace:** std
