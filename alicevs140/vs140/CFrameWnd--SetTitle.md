# CFrameWnd::SetTitle
Sets the title of the window object.  
  
## Syntax  
  
```  
  
      void SetTitle(  
   LPCTSTR lpszTitle  
);  
```  
  
#### Parameters  
 `lpszTitle`  
 A pointer to a character string containing the title of the window object.  
  
## Requirements  
 **Header:** afxwin.h
