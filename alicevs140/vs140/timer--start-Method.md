# timer::start Method
Starts the `timer` messaging block. The specified number of milliseconds after this is called, the specified value will be propagated downstream as a `message`.  
  
## Syntax  
  
```  
void start();  
```  
  
## Requirements  
 **Header:** agents.h  
  
 **Namespace:** concurrency
