# CMFCRibbonFontComboBox::GetCharSet
Returns the specified character set.  
  
## Syntax  
  
```  
BYTE GetCharSet() const;  
```  
  
## Return Value  
 Character set (see LOGFONT in the Windows SDK documentation).
