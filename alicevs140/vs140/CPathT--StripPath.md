# CPathT::StripPath
Call this method to remove the path portion of a fully qualified path and file name.  
  
## Syntax  
  
```  
  
void StripPath( );  
  
```  
  
## Remarks  
 For more information, see [PathStripPath](http://msdn.microsoft.com/library/windows/desktop/bb773756).  
  
## Requirements  
 **Header:** atlpath.h
