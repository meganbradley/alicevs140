# error_condition::value_type
A type that represents the stored error code value.  
  
## Syntax  
  
```  
typedef int value_type;  
```  
  
## Remarks  
 The type definition is a synonym for `int`.  
  
## Requirements  
 **Header:** <system_error>  
  
 **Namespace:** std
