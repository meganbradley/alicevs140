# CMFCColorBar::GetColor
Retrieves the currently selected color.  
  
## Syntax  
  
```  
COLORREF GetColor() const;  
```  
  
## Return Value  
 The currently selected color.  
  
## Requirements  
 **Header:** afxcolorbar.h
