# Type characters are not allowed in label identifiers
A statement label contains at least one identifier type character.  
  
 A statement label must be a valid Visual Basic name. The allowed characters do not include any of the identifier type characters (`%`, `&`, `@`, `!`, `#`, and `$`). See [Declared Element Names](../vs140/Declared-Element-Names--Visual-Basic-.md).  
  
 **Error ID:** BC31395  
  
### To correct this error  
  
-   Remove the type identifier character or characters from the statement label.
